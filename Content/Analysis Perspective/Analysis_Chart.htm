Analysis Chart

  • There are a collection of built-in Charts (Cash Flow, NPV Sensitivity, Pareto) within the Analysis Perspective.
  • The Options view allows you to set the Source of the data (Analysis Case, Project Set, Jurisdiction, Project or Asset), as well as the individual elements of the Cashflow tab (Before or After Tax, CAPEX, Revenue, etc.) to be charted.
  • The Data Source view provides access to the data being charted in a Tabular format, which can be Exported if desired (to MS Excel, or copied to clipboard)

 

Chart Types

You are able to choose the follow chart types and styles, by clicking on one of the chart icons in the Chart area:

  • Cash Flow - bar, line or area chart
  • NPV Sensitivity - line or area chart
  • Pareto - bar, line or area chart
