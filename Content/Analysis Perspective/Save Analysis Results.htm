Save Analysis Results

Upon completion of an Analysis Case calculation, the full set of results (data as appear in all Analysis Views) can be saved by locating the button in the first five (Summary, Custom, Cashflow, Income, and Metrics) Analysis View tabs in the Analysis Perspective:

By clicking the button, an 'Analysis Results' dialog box will appear, prompting the User to specify the Description and Comment for the current set of Analysis Case results to be saved:

Click Next to continue to specify the Navigator folder to save into, and Finish to complete the save.

The new file (Training Default) will appear in the Navigator, under Analysis Results:

Next time when esi.manage is re-opened after closing, the same set of results can be retrieved and viewed/exported by opening the saved Analysis Results file from the Navigator. Note that the Analysis Perspective will only display a single set of Analysis Results at a time.
