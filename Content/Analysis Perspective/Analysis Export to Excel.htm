Analysis Export Options

After running an Analysis Case and results are generated in the Analysis Perspective, the results can be exported into either Excel (.xlsx) or CSV with a variety of options for formatting:

  • Excel
    • Export to Excel (comprehensive analysis results)
    • Export to Perspectives (Perspectives-compatible format)
    • Export to esi.portfolio (esi.portfolio-compatible format)
  • CSV
    • Export to CSV - Custom tab (formatting follows the active Custom tab layout)
    • Export to CSV - Cashflow tab (formatting follows the active Cashflow tab column selections)

The following sections describe each of the five options for exporting analysis results.

Export to Excel

This option is available in the Summary, Custom, Cashflow, Income, and Metrics tabs. The option can be found by locating the button:

The resulting output file has .xlsx as file extension, and has an individual tab for each of the five tabs. The file is a comprehensive set of analysis results from each of the five Analysis View tabs it was sourced from, with the formatting and data in each output file tab matches the Analysis View tab it is named after.

Note:Data in the Custom tab from Export to Excel are rolled-up totals for the analysis window (for each Analysis Case, Project Set, Project, Asset, or Fiscal Model Hierarchy node), and contains no dates or time-series detail.

For the InputData to be properly populated, the active layout should contain any of the time-series metrics calculated from the Analysis Case that needs to be exported to Perspectives, as well as a single non-time-series metric (such as NPV):

For the Attributes to be properly populated, the active layout should contain any of the hierarchy attributes or custom attributes that needs to be exported to Perspectives:

For the SelectionConstraints to be properly populated, the Selection Constraints table in the Project Perspective must be populated (see Selection Constraints for more details).

Export to esi.portfolio

This option is available in the Custom tab only. The option can be found by clicking the button next to the button, and selecting the option:

The metric and attribute requirements for the active layout in the Custom tab for the Export to esi.portfolio option is identical to those for the Export to Perspectives.

Export to CSV - Custom tab

This option is available to the Custom tab only. The option can be found by clicking the button next to the button, and selecting the option:

The active layout of the Custom tab determines the columns (metrics and attributes) in the CSV file, as well as the grouping level (Analysis Case, Project Set, Project, Asset, or Fiscal Model Hierarchy node) of the data. For more details on configuring layouts, see Custom Analysis .

Note:Unlike Export to Excel, data in the Export to CSV from the Custom tab has dates and time series information.

An Excel report file can be uploaded to esi.manage as a template for esi.manage to populate Analysis Case results data into the file. In the Cashflow tab, the User can right-click on any of the nodes at any roll-up level, and select the template to generate a custom economic report:
