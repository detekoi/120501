Income Statement

  1. To run the Analysis Case, click the "Analyze the Analysis Case" button
  2. Once the calculations are completed, the Income Statement table will be populated with the results.

 

Note: The time horizon can be set in the Analysis Settings between Monthly and Yearly.  You can also set the time horizon to be Monthly for X Periods (X being a multiple of 12) and then Yearly from then on. The Income Statement can now be exported by pressing the export to excel button in the top right corner.

 

Display Options

Click on the Options tab on the upper right side of the Income Statement to expand the display options:

In the Statement Variant dropdown, there are four built-in layouts of the Income Statement that the User can choose from. Each layout variant (Default, Canada 1, Canada 2, US) has minor layout differences in the Income Statement, as well as different definitions for some of the Income Statement metrics.
