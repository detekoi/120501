Cash Flow Analysis

  1. To run the Analysis Case, click the "Analyze the Analysis Case" button on the Project List, inside a Project, or from the context menu after selecting Project Set(s).
  2. Once the calculations are completed, the Cash Flow Analysis table will be populated with the results.
    • If the Analysis case is being run for the first time, the table will be collapsed.
    • Clicking the "twisty" next to a Node will expand the table showing the sub-nodes below it.
    • At each node, the summary roll-up is shown which captures all data under the node.
    • The Details (monthly or yearly periodic data, as a configured in the Analysis Case) can be seen by clicking on the Details button to the left of the node.  Viewing the details for a node will hide all sub-nodes below it. Clicking the Details button again will hide the details, and the sub-nodes will become visible. See images below for before and after.

 

Cash Flow table Navigation

As there are a variety of approaches to running Analysis Cases, there can be a significant degree of nesting in the table. While you can successively navigate through nodes (selecting the Expand option) there are convenient features to streamline this.

If you right click on a row in the table, the context menu will be displayed.

 

By adding or removing focus you can be more specific with what projects or options you would like to see.

 

Applying Focus

Since there is often a large amount of data in an Analysis Case, it is not possible to visualize all the data at once in the Analysis Charts. It is possible to "Add Focus" to certain information in the Cashflow table, so that the Analysis Chart only reflects the data of interest.

  1. Expand the Cashflow table to the data to be used in the Analysis Chart.
  2. Right-click on the node that contains the data to be displayed in the Analysis Chart and select "Add Focus". Focus is indicated by a magnifying glass next to the node. Focus can be added to multiple nodes, independently, and all will be displayed in the Analysis.
  3. Switch to the Analysis Chart and configure to display the data as needed.

 

Nominal and Real Values

The cashflow view supports both nominal and real values. To toggle between the two, select the desired option or simply click the toggle button in the toolbar.

Real values are based on the uninflated currency at the Real Base Date defined in the Analysis Case:

Nominal values are based on currency with inflation applied (see Currency Inflation for details on setting up currency inflation schedule).

 

Cash Flow Column Headings

The column headings presented in the Analysis Cashflow can be customized.Click the 'Change Columns' icon to be presented with a set of available and chosen columns. The same functionality can be found in the right-click context menu.

 

You will be presented with the Change Columns dialogue window upon clicking the icon above. Here you can add/remove columns from/to your display. To add a column, highlight it under the Available section, and press the right arrow button to move it to the Selected section.

The up and down buttons allow the you to re-order columns to suit your preference.

 

Select column descriptions:

  • Effective Price = Revenue / Total Production
  • Net Margin = Revenue - OPEX - Current Tax

 

Extracting Data from Analysis Results

Data can be extracted from Analysis Results in several ways. Data can now directly copied out of the Cashflow and Income tables, by selecting and copying cells to the clipboard, which can then be pasted into a spreadsheet. The Cashflow page can be directly exported to a .csv format by selecting the “Export to csv” button at the top right of the window. The entire result of the Analysis Case can also be exported directly to Excel. See the separate section on Analysis Export Options for more information.
