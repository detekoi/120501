Analysis Perspective

The Analysis Perspective provides an interactive environment designed to help you better understand your Project Cash Flows, Burdens and Economic Metrics, ultimately giving you a better understanding of your Projects and how they compare against one another.

It allows you to compare the same projects against Multiple Price Decks and/or Properties.

You can perform Economic Look Backs against previously Archived projects using historical price files.

The Start Date (as displayed on the Summary tab) is taken as the earliest start date from among the projects within the analysis case.

The End Date now is sensitive to the Analysis Start and Years Span properties:

  • Full Cycle: the End Date is the latest project start date + the number of years set in the "Years Span" box in the Analysis Case
  • Point Forward: the End Date is the specified Point Forward date + the number of years set in the "Years Span" box in the Analysis Case
