Chart Options

There are many elements of data that comprise a Cash Flow Analysis:

  • Individual entries such as Drilling Cost or Gas Revenue
  • Aggregates (CAPEX, Revenue)
  • Cash Flow Tags (BTAX, ATAX)
  • Local or Jurisdictional Burdens
  • Cumulative, Present Value

The Options view within the Analysis Chart allows you to select what Data you want to Chart. As you turn on Cash Flow elements (only After Tax is shown below), the additional entries will be charted automatically. There is no requirement to re-run the Analysis Case to chart data in a different manner.

Note: the Options view will change accordingly based on the Analysis Chart you are viewing.
