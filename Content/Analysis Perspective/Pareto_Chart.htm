Pareto Chart

The Pareto Chart is used to "Rank" on a selected Economic Metric.

  • Select the Metric that you want to Rank on.
  • Select the Source (typically Project or Asset, but you could select Project Set or Jurisdiction).
  • Specify the number of entries that you want to show.
  • Specify the Order
    • Based on the interpretation of the Metric, the Order will either show the best or worst from the Source (i.e., Best Projects or Poorest Projects)

See Chart Options for a description of how to Apply Focus.
