NPV Sensitivity Chart

The NPV Sensitivity Chart will recalculate the NPV of each Source at different Discount Rates:

  • The Discount Rates that will be used are based on the Max%, Min % and number of Trials (in this screen shot, there will be NPV's calculated at 2% increments starting at 5%).
  • If you want to use this Chart to find the IRR (the Discount Rate where NPV is 0), you can adjust the Min and Max accordingly to find the region where the Axis is crossed.
  • You can also modify some of the Economic Parameters (Discount Date, Mid Year toggle, etc.) and have these parameters used when calculating the NPV.
  • The Data Source page will present the chart data in a Tabular form, which can be Exported to Excel if required.

See Analysis Chart Options for a description of how to apply Focus to the Source.
