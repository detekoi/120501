Fiscal Year

Company's can select the Month which starts their Fiscal Period in the Configuration perspective.  This allows Reports and Exports, the Data Matrix and Template Scheduler to use Fiscal Year labels. Note that changes to this setting require that esi.manage be restarted for the changes to be in effect.

 

Using a CAPEX Report as an example, with a Fiscal Start Month of July.

  1. A CAPEX Report on a Quarterly basis from Jan 1, 2010 to Dec 31, 2012 will have column labels of: Q1 10, Q2 10, Q3 10, etc.
  2. If the Use Fiscal Calendar option is selected on the Reports Date page (with the same reporting period), the column labels will be Q3 YE10, Q4 YE10, Q1 YE11, Q2 YE11, etc.

The data is not changed with this setting, rather the reporting labels will be modified accordingly.
