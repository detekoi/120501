Royalty Fields

The fields available for entry in the Royalty Fields table are as follows:

  1. Asset: The Asset that the Royalty applies to.
  2. Product: The product the Royalty applies to - Gas, Oil, etc.
  3. Pin Asset: Pin the royalty so it will not shift when an asset or product shifts.
  4. Date: The month and year the royalty applies to.
  5. Royalty Type: The royalty types defined in the Royalty Configuration perspective. It is possible to store up to three different Royalty Types (scenarios) that represent different prices (eg. P10, P50, P90).
