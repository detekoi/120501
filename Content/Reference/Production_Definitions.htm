Production Definitions

The Production Definitions table allows you to show, hide, and change the names of the definitions found in the 'Production Definition' drop-down of the Reports perspective.

  1. The "Visible" check determines if the definition will appear in the drop down list.
  2. The Description can be edited by double-clicking the name field and entering a new name, then clicking outside the line to save changes.
  3. The calculation checks are read only and cannot be changed at this time.
