Project Gantt

You can view the scheduled activities for your projects using a powerful Gantt Chart.

 

The list of the filtered Projects in the Project Browser is used to limit and specify the information displayed in the Gantt.

The Gantt chart shows the duration of specific measures of a project, from the Start Date to the end of the final measure (i.e. the start date of the earliest measure to the end date of the latest measure). The Gantt chart supports expenditures, and fixed OPEX not tied to a product.

You can use the Gantt to change the start and end dates for these expenditures, if there are no Actuals associated with the Project.  This tool can be used then to arrange project expenditures to control for overlap of expenditures.
