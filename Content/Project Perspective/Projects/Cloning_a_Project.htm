Copy a Project

You can Copy a Project from one budget to another.

 

To Copy a Project:

  1. Find the project in the Projects list.
  2. Right click the project and select "Copy Project."
  3. The "Copy Project" window pops up:

 

 

  1. Select a Target Budget (the source information will be populated based on the selected Project).
  2. Give the new project a name in the "Project" field of the Target column. (The default is Copy of [Original])
  3. Select the Project Manager.
  4. If the new Project starts later than the original, you can select a new Earliest Project Date using the calendar drop down.
  5. When the details have been entered, click "Copy".

A new project will be created. The new project will retain some data from the previous project. Actual information will not be copied to the new project - only forecast data.  Document attachments will also not be copied into new project.
