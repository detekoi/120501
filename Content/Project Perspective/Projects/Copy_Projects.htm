Copy Projects

You can create a copy of a project either by copying the forecast of a single project or by copying multiple projects at once.

  • Copy a Project -  Single project is copied to another budget following same rules that are used when we archive a budget.  No actuals are copied.
  • Copy Projects to a Selected Budget  – Multiple projects can be copied with or without preserving the original project and asset codes, mapped AFE's and actuals.

To copy a project, within the projects list, select a single project or hold down control and select multiple projects. You can also bulk select multiple projects by selecting the first project towards to the top of the projects list and hold down shift and then with your mouse, select the last project you would like in the list. This action selects all the projects from the first to the last project selected.

Once you have the projects highlighted that you would like to copy, right click and select ‘Copy project to budget’. This action is also available using the button circled below.

 

A copy selected projects screen will then open.

 

You will need to select a budget for the new projects to be copied to. If you select the same budget that the original projects live in, the screen will ask you to rename the duplicate projects using a prefix.  This will not copy the non-forecast data of the projects.  If you select a different budget, you can choose to keep existing project and asset codes, non-forecast data, and any attachments.
