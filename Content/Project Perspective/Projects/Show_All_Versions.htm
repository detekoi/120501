Show All Versions

By default, projects are only shown from a single budget. Project Sets can only be raised against a single budget so you can only ever define projects from one at a time.

However, if you want to look at multiple versions of projects across archives, you can use the "Show All Project Versions" option. This will change it so that*for the projects selected you can see all of the versions from all "Visible" budgets in the system.
