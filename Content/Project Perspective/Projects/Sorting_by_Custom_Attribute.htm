Sorting by Custom Attribute

Custom attributes can be added as columns to the Project List.  This adds a new layer of sorting of projects to the project list.

Custom attributes must be configured properly before they will be visible as columns in the Project List.  To see how to configure the custom attributes please see Configuring Custom Attributes.

Note: esi.manage needs to be restarted after Display in Tables has been selected in configuration for settings to be applied.
