Expenditure Analysis View

The Expenditure Analysis View has been designed to streamline Project Tracking. From this one view you can:

  1. Examine all Data Types (Forecast, Field Estimates and Actuals), as well as the calculated Blended values.
  2. Filter the data by Expenditure, Expenditure Type, or AFE.
  3. View the data using any of the Capital Definitions: Unrisked Gross, Unrisked Net, Risked Gross, Risked Net.
  4. Select an Sensitivity File to apply to the calculations.
  5. Choose whether to display Nominal (default) or Real amounts, and the associated base currency date
  6. Select a Escalation, Exchange and Scale File to apply across to the calculations.
  7. Choose to display the data Daily, Weekly, Monthly, Quarterly or on an Annual Basis, for up to 100 Periods.
  8. Choose the Dates to use for Start Date, and Last Actuals Date for Blending calculations.
  9. Modify the Chart view to display select data (Forecast and Blended for example), use different displays (Bar, Line or Area), or display Cumulative or Periodic values.

 

The Asset, Attributes, Expenditure Type and AFE drop downs utilize a multi-select Tree View control. The 'contents' of these list will be dependent on where the Analysis view was launched from:

  • From a Project - AFEs and Expenditures will be based on the data contained within the Project.
  • Global - AFEs and Expenditures will be the union of the Project('s) content.

 

Note: these tree controls do not interact, i.e.; Filtering the AFE list does not reduce the Expenditures list to those Expenditures mapped to the selected AFEs.

When the data is generated, it will be based on those Expenditures which satisfy ALL of the filtering.

 

Note: that the Analysis View can be Detached and moved to a second monitor.
