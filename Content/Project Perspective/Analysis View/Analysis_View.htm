Analysis View

The Analysis View is an ad-hoc Analytical Tool for Expenditure, OPEX and Production Data. There are three routes to accessing the Analysis View:

 

  1. From a Project - on the Expenditure, AFE, Production and OPEX pages of a Project - select the Analysis View button . The Analysis View will be initialized to the Project it was invoked from.
  2. Globally against the Active Selection - from the main Button Bar, the Analysis Button provides access to each of the Analysis Views. The Date Source will be initialized to all Projects displayed in the Projects List - either individually or as an aggregated value.
  3. Globally against a Project Set(s) - from the Navigator, you can open the Analysis View(s) against one or more Project Sets. The Data Source will be initialized to the combined list of Projects - either individually or as an aggregated value.

 

Tool Bar

The Analysis View toolbar is shared amongst the Expenditure, OPEX and Production Analysis.

The refresh button recalculates the analysis. When a new source is selected or a different set of parameters is specified the icon will show a warning sign indicating that a refresh is needed

Auto refresh refreshes the analysis every time you switch between different sources (e.g. Analysis cases, Budgets).

Show definition toggles the input parameters section of the analysis.
