Bulk Edit Field

To edit multiple projects at the same time, simply CTRL + Click on the projects you want to edit, and select one of the two ways to get to the Bulk Edit.

 

One the Bulk Edit option is selected, you will be prompted to select the appropriate fields within the Bulk Edit fields you wish to change.

 

Prepare bulk edits by selecting a bulk edit field in the left hand pane (click on the name of the field you want to change AND click the check box next to it). In the example below, Downtime is selected and a table appeared in the right hand pane.

 

Fill in appropriate information and continue to add edits or press OK to process the changes.

Refer to the dialogue header pane for status and error messages as you build your Bulk Edit Field actions.

The OK will not be made available after adding an edit until all necessary edit criteria are made.
