Hierarchy Attributes

There are several fields that can be edited within this field.

 The following steps outlining how to add new actions to the Hierarchy Attribute field:

1. Select Hierarchy Attributes by first clicking the to create new bulk edit action lines. Then choose a hierarchy name from the drop down list in the Name field.

 

2. Select a value from the drop down list in the Value field.

 

3. Repeat the above steps to create more hierarchy attribute bulk edit actions.

4. When all fields are populated, select the Hierarchy Attribute Bulk Edit field to put in the pending Bulk Edit Action list.
