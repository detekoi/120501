Downtime

Change or add new downtime entries to a group of projects. Changes are applied to all assets in a project.

 

To create a Downtime Bulk Edit action:

1. Select Downtime. Choose Update or Add from the drop down menu. Update deletes all downtime entries and then changes to the bulk edit entries. Add does the same thing as Update, expect in the case when there is no entry for downtime. In that case, Add will create a new entry or entries.

 

2. Click the to create new bulk edit actions.

3. Enter the downtime information into the new bulk edit action.

 

4. Repeat the above steps to create more Downtime bulk edit actions.

5. Select the Downtime Bulk Edit field to put it in the Pending Bulk Edit Actions list.
