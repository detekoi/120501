Fixed Attributes

This field allows you to change the fixed attributes of the projects selected. Within this field, you can change the following:

  • Project Status
  • Project Location
  • Project Manager
  • Project Comment

To change this field, select "Fixed Attributes", press the to allow for more fields that allow many more changes.

You will need to fill in the appropriate fields you wish to edit.

1. Choose an attribute name from the drop down list in the Name field

 

2. Select the Clear check box to clear the attribute value. Fields that are mandatory cannot be cleared. Or,

 

3. Select a value from the drop down list in the Value field or enter text in the text field. The type of data entry depends on how the attribute is configured in esi.manage.

 

4. Repeat the above steps to create more Fixed Attribute bulk edit actions.

5. Select the Fixed Attribute Bulk Edit field to put it in the Pending Bulk Edit Actions list.

 

Note: When the OK button has been pressed, all open Project Detail editors are closed without saving. This is to remove the possibility of stale data being saved.
