Tabular Scaling

This will allow you to scale tabular production forecasts by a specified factor. Changes are applied to all assets in a project with tabular production data.

To create a Tabular Scaling Bulk Edit action:

1. Select the Tabular Scaling option. Choose the Product and Set the Scale Factor.

 

2. Select the Tabular Scaling Bulk Edit field to put it in the Pending Bulk Edit Actions list.
