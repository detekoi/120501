Derived Yield Production Volumes

The derived product stream must already exist on the Production tab or no edits will be made. Changes are made to all assets in a project.

There are several different fields that can be edited within this tab.

The following are the steps outline the process of adding Derived Yield Production Volumes.

 

1. Select Volume (Derived Yield).

 

2. Select the Primary Product and the Derived Product.

 

3. Set the Sample on-stream date to provide a different "Starts on" date. This date is solely for your information. It has no effect on the project fields.

 

4. Set the units for the products and whether it is a By-Product or Associated Product.

5. Choose whether the downtime should be set relative to the onstream date, or to an absolute calendar day. If set to calendar day, set the appropriate start date.

6. Choose Update or Add from the drop down menu. Update deletes all derived production entries and then adds in the bulk edit entries. Add does the same thing as Update, expect in the case when there is no entry for derived production. In that case, Add will create a new entry or entries.

 

7. Enter the Offset, Time, Ratio, and Interpolation fields. See derived products for more information.

8. Select the Volume (Derived Yield) Bulk Edit field to put it in the Pending Bulk Edit Actions list.
