Date Shifting

Shift project dates to start on a specific date or shift the project dates by a specified number of days.

 

To create a Date Shifting bulk edit option:

  1. Select the between Shift by Start Date or Shift by Days.
  2. Enter the New Earliest Date or days to shift by.
  3. Once the new dates are set, select the Date Shifting Bulk Edit field to put it in the Pending Bulk Edit Actions list.
