Shrinkage

Change project production shrinkage factors. Changes are applied to all assets in a project.

To create a Shrinkage Bulk Edit Action:

1. Select Shrinkage. Choose Update or Add from the drop down menu. Update deletes all shrinkage entries and then changes to the bulk edit entries. Add does the same thing as Update, expect in the case when there is no entry for shrinkage. In that case, Add will create a new entry or entries.

Shrinkage rates are affected on a per-product basis, and can be selected with Select Products.

 

2. Edit the table with product shrinkage percentages as you would in the Shrinkage Tab in the Production Tab of the Project Editor.

 

3. You can also change the columns by pressing the .

4. Select the Shrinkage Bulk Edit field to put it in the Pending Bulk Edit Actions list.
