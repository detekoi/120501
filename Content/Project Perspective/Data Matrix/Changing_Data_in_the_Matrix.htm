Changing Data in the Matrix

There are several routes to changing Project Data from within the Data Matrix.

Note: that whenever a value has been changed from what was obtained from the Project, the value will show in Blue.

In order to Update an entry - the Data Matrix Display Unit (bottom right corner of data matrix) must be the same as the line's Source Unit. See Columns to display the Currency unit.

  • Direct Entry of Values - simply type in a value within a Cell.
  • Copy/Cut and Paste - select any number of cells:, multi-selection can be by dragging around a region, selecting one corner of the region and then selecting the opposite corner while holding down the Shift Key, or successively selecting cells while holding down the Ctrl key.  Right click within the selection and choose Copy or Cut. Choose the location to be pasted into, and the same 'pattern' of selection will be pasted into.
  • Bulk Update - allows you to apply either a % or new Total Amount to a selection of cells. If a Total amount is provided, this is converted to a % and each cell is updated appropriately.
  • Update Forecast From - allows you to Update the Forecast based on CAPEX Actuals (or Field Estimates), or OPEX Forecast based on Actuals or Field Estimates. This allows you to 'Shift' Forecast amounts into Periods beyond the current date.

Note: that in the following descriptions, none of the Data within the Matrix will be saved until the main Save button (upper left corner of esi.manage) is pressed.

 

The Impact of Periodicity

The Periodicity controls how the data is displayed in the Matrix.

  1. If you have originally entered data on an Annual Basis and request Monthly representation in the Data Matrix, your Annual data will be apportioned over the Period.
  2. If you Edit this data and then commit it back to the Project - what is displayed in the Matrix will determine the corresponding 'Schedule' in the Project.
  3. For this reason if you have created a Monthly Expenditure schedule in a Project(s), you should Edit in the Data Matrix on a Monthly basis.
  4. If you are Editing on an Annual basis in the Matrix, and then Update values back into the Project, your Monthly schedule will be replaced with an Annual one.

Copy/Cut and Paste

To Copy a selection of Values from one 'location' to another:

  1. Highlight the 'source' cells, right click and select Copy.

 

  1. Move to the upper left corner of the 'target' cells, right click and select Paste.

 

  1. The 'schedule' and amounts of the 'source' cells have overwritten the 'target' (in this case the 2,100,000 value shown in the first image is where the Paste was applied).
  2. The Pasted values are shown in Blue, and the Grand Total updated based on the Pasted values.
  3. To move the values shown in the first screen earlier in the year (same Project), highlight the cells as shown in the first image. Right Click and select 'Cut'.
  4. Move to the Upper Left corner of the Target region and select Paste.

 

  1. This has moved the original Expenditures two months earlier.
  2. Note that all of the values are shown in Blue - none of the changes have been updated to the Project.
  3. You can Undo all changes (one at a time) made prior to committing, or select Refresh to re-obtain the current data from the Projects.
  4. You must select Save from the main button ribbon to commit the changes back to the Projects.

 

Bulk Update

The Bulk Update button allows you to update a group of cells.

  1. Select the cells to be updated.
  2. Select the Bulk Update button -

 

  1. The Total for the selected cells is displayed.
  2. You can either enter an Update Percentage, or a new Update Total (which will be converted to a %).
  3. This percentage is then applied to each cell in the range.
