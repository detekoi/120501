Data Matrix

The Data Matrix Editor allows you to Create or Update Expenditures and OPEX across multiple projects. The interface is designed to mimic a familiar spreadsheet workflow (copy and paste, undo, etc.), and incorporates a set of Filtering Capabilities.

  • The Data Matrix allows you to view all Data Types (Forecast, Field Estimates and Actuals).
    • For Expenditures with AFEs, the Field Estimates or Actuals from the AFE(s) will be displayed in the Data Matrix.
  • Both OPEX and Expenditure information will display the Source Currency Unit for each entry.
    • All values will be converted into the selected Display Currency, which can be toggled within the Matrix.
    • Only entries whose Source Currency is the same as the Display Currency can be updated.
  • The Production Editor allows you to choose units for Gas and Liquids independently.
  • The Data Matrix will present the data in different Period Granularity (Daily, Monthly, Quarterly, Annual), over a Horizon that you define.
  • Forecast's can be Updated from Actuals to aid in re-forecasting.

 

Multiple Data Matrix’s can be Opened - the 'source' will be indicated in the Page Tab for each Matrix instance.
