Data Matrix Dates

The Data Matrix Date Controls function on two base options: Periodicity and Duration.

  • Change Focus Period - changes the time span shown on screen. This can go from 3 months to 50 years.
  • Change Periodicity - changes the interval shown on screen. This can go from Monthly to Annual.

 

 

 

 

 

 

 

 

WARNING: Any Data that you commit back to a Project from the Data Matrix will be based on the Periodicity. While this is valuable at the start of a Budget Period (converting a Yearly Expenditure or OPEX schedule to a monthly basis), if you have already constructed a Monthly schedule within a Project and then Edit in the Matrix on an Annual Basis - any commits will Replace the Project's schedule.

In this situation, it is extremely important to edit at the same granularity as your expenditure schedule. Changes made in the editor override the schedule set out in the expenditure screen.

Example: If you have a 6 month expenditure starting in January and ending in June, and update the expenditure at an annual level, the expenditure will now change so that it runs from January 1st to December 31st.

 

The arrows (either end) of the  Spark Line Window (at the bottom of the Matrix) will scroll the display period earlier or later.
