Data Matrix Filtering and Display Options

There is a wide variety of Filtering features with the Data Matrix.

  • Show and Hide Columns
  • Sort a Column ascending / descending - successive sorting can be applied (Column 1 then Column 2)
  • Filter based on a Column entry or apply a Custom Filter
  • Re-order columns in the Display

 

This is best illustrated by an example - using the above screen shot as the start point:

  1. Show and Hide Columns - Right Click on the Column Headers.
    • Individual Columns can be added or removed, or select More to get a Modal dialogue to make multiple changes at one time.
  2. In this example Line Number will be removed, and Project Code, Onstream Date and Grand Total will be added.
    • The left view shows the Starting Point - the right view shows the changes being applied in the More view.

 

 

  1. By clicking the on the column header, each column can be Filtered based on one of the Entries (i.e., Drill), or a Custom Filter can be constructed - in this case for Expenditure Types which contain Drill.

 

  1. The Column order can be changed by selecting a Column and dragging it to a new location - in this case Asset will be brought to the left most position.
  2. Sorting can be applied across multiple Columns.
    1. Select the First column and Sort as desired (ascending or descending). Successively selecting the Column Label cycles through the sorting.
    2. Hold down the Ctrl Key, and then apply the sorting (select the Column Label) to the Second (Third) columns to be sorted.

 

Each of the Columns will indicate what has been applied:

  1. Project Code has been moved to the first column, and is the 2nd level of sorting - descending.
  2. Onstream is the 1st level of sorting - ascending.
  3. Type has been Filtered.
  4. Focus Total has been added, and Line number has been removed.

The Remove All filters button will clear any filtering that has been applied.
