Data Matrix Controls

 

The choices are:

  • Add Line - provides the ability to add an Expenditure line to the Highlighted project.
  • Delete Line - deleted the selected line from the Project (the cursor must be within the data portion of the matrix for this to be enabled).
  • Open Project - will open the Project Editor for a highlighted Project.
  • Clear Filters - will remove all of the Applied Filters.
  • Update Forecast - Updates the Forecast for selected items from either Actuals or Field Estimates.
  • Bulk Update - allows a selection of cells to be Inflated (deflated) by a Percentage, or to a new Total amount.
  • Change Focus Period - adjust the Horizon of the data.
  • Change Periodicity - adjust the Granularity of the data.
  • Change Type - change the Type of data that is displayed (Field Estimates, Actuals or Forecast).
  • Export to Excel - Export the current Data Set to Excel.
  • Undo - undo the last action.
  • Refresh - will obtain the most recent set of data, particularly relevant if the Data Matrix has been opened from the Projects List which is being Filtered via the Browser.
  • The Production Editor has additional buttons to allow Units to be set and to display production as a daily rate.

 

See Changing Data in the Matrix for a description of the editing features.
