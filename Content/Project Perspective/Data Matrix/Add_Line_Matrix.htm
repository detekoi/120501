Adding Entries within the Matrix

New Project entries (Expenditures or OPEX) can be added directly from the Data Matrix. When a particular line has been selected, choosing the Create Expenditure button will bring up the following screen.

 

The key points are:

  1. The Category, Type and Asset will be initialized at the same 'values' as the selected line.
    • You can change any of these (the drop down options are the same as if the Expenditure was created within the Project Editor).
  2. The Start Date will be initialized to the Starting date for the Data Matrix.
    • The End Date will be based on the Data Matrix Periodicity (i.e., if the data is being viewed with a Quarterly granularity, the Expenditure will be initialized to 3 months duration.
  3. Only a Gross Amount can be supplied, the Working Interest will draw from the Ownership table.

 

Additional Expenditure information can be accessed by selecting Next.

 

Adding OPEX Entries within the Data Matrix

Fixed OPEX Entries can also be added in a similar manner (note that you can define the Period for the new OPEX item):

Select the "Fixed OPEX" tab at bottom of view, and select to add new Fixed OPEX.
