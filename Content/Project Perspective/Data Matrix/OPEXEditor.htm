OPEX Editor

The OPEX editor shows all fixed OPEX line items for the projects in the selection the matrix was opened against.

By changing the Type you can edit the Forecast, Field Estimate, Financial Actual and Actual amounts.

 

See Changing Data in the Matrix for a description of Updating amounts, or Update From for a description of updating the Forecast from Actuals (or Field Estimates).

Note the difference between the Grand Total column, the Focus Total and the information displayed at the bottom of the window.

  1. The Grand Total is for the life of the OPEX entry - OPEX is often Forecast for a number of years, on a Monthly Basis.
  2. The Focus Total is the Total for the current period - i.e., 3800 / month for 1 year is 45,600.
  3. The bottom of the screen displays some statistical information based on  the highlighting.  Try selecting a group of cells at once, and notice the changing values at the bottom of the window.
  4. The 'Spark Line' is a graphical representation of the data that exists for the selected Projects. The grey region indicates the 'shape' of the selected data type (in this case Forecast).
  5. The Yellow box shows the current Horizon.

 

Changes are not passed through to the OPEX screen and committed to the database until the save button is pressed.
