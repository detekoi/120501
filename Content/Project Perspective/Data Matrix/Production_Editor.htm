Data Matrix - Production Editor

The Production page tab of the Data Matrix presents Production data in the same manner as the Expenditure and OPEX editors, however, you cannot update forecast production data in the data matrix, although changes to production forecast data can be made through the bulk editor.

The Data Matrix controls are the same as for the other Editors, with two additions.

  • Daily Rate Option - displays the data as the Average Daily Rate for the period.
  • Convert to Tabular Segment - forecast formulaic production can be converted to tabular, thus becoming editable in the matrix.

 

Please note, Derived products are not editable through the data matrix. There is a column that displays which products are derived, though only Actuals and Field Estimate values are shown for derived products.

The classification column is populated when forecast type is selected.

 

See Changing Data in the Matrix and Update Forecast From for a description of the various Workflows.
