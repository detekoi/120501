Data Matrix - Update From

In addition to Changing data within the Matrix directly, you have the option of Updating the Forecast from either Field Estimates or Financial Actuals,  including AFE Actuals and Field Estimates.

The Update Forecast button will open the following dialogue:

 

  1. If you have selected a number of Edited cells, this will be indicated in the Dialogue.
  2. The Update Source determines what will be the source for the Forecast Update (Field Estimates or Actuals).
  3. You can toggle the Update All option, in which case all cells with data will be changed.
    • Note that if Expenditures (or OPEX Items) were originally scheduled on a Yearly Basis, and you are working with the Matrix on a Monthly Basis, the Update function will change the schedule based on what is displayed in the Matrix, even if a particular value hasn't been edited.
    • In addition, the Update All option will apply to all Data within the Matrix - i.e.; if Filtering has been applied it has not removed the data from the Matrix, only the display.

 

You also have the option of applying logic to Maintain Totals up to a certain point in time.

  1. Maintain Total will determine the delta between the current Forecast Total (for an Expenditure), and what the new Total will be after Updating the Forecast from the Actuals.
    • This delta is then applied to the Period beyond the Update window (periods that there are Actuals for), such that the Total Forecast Amount doesn't change.
  2. Selecting Until End Of (in conjunction with Maintain Total) will apply this logic, but only up to the selected date (appropriate for multi-year Expenditures where you are wanting to control the adjustments to within the current year).
    • For example, if there is a 3 year Expenditure at 10,000,000 per year, and the Actuals for the first six months of the year are 4,000,000, selecting Maintain Total to Dec 2010 will apportion the Delta (1,000,000) over the remainder of this year.

The option Source Date Until specifies to which date the source's data will be copied over the forecast data. It works in conjunction with the Maintain Totals option. The drop down menu housing the dates is inclusive.

  • This proves useful, for example, if 8 months of source data are available but only the first 6 months are to be updated.

 

There has been another option added within the Update Actual with Field Estimates, this is the Convert Nominal to Real option, this will covert the Nominal values within the Data Matrix into Real Values.
