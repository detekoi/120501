Deleting Type Curves

To delete an existing Type Curve, return to the 'Type Curves' tab within the Project perspective. Select the Type Curve to be deleted:

 

Select the 'Delete Type Curve' button to delete the selected Type Curve. The selected Type Curve will be removed from the list.

 

Note: if a Type Curve is currently in use (in the Project Editor), and an attempt to delete said Type Curve is made, an error message will appear. To correct the issue, any Patterns and Recurrences within the Budget that use the Type Curve to be deleted will need to be modified or removed in order for the deleting to proceed.
