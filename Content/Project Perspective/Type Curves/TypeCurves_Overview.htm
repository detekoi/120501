Type Curves Overview

Type Curves are template patterns for Production, Expenditures, and Royalties that can be created for a budget and applied in many combinations and repetitions to different assets within each project in the budget.

 

As shown in the screen shot, Type Curves are found under the 'Type Curves' tab in the Project Perspective.

This section pertains to the functionality of the Type Curves tab found in the Project Perspective, which focuses in the creating of Type Curves. For the functionality of the Type Curves tab in the Project Editor, which focuses on the using of Type Curves, refer to Type Curves under the Project Editor section in the table of contents of this help document.
