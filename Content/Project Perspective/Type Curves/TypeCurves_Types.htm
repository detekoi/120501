Types of Type Curves

Three types of Type Curves can be created in the Project Perspective:

 

Production Type Curves

Creating a Production Type Curve is similar to creating a Production in the Project Editor. Unlike creating a Production, different Production Type Curves appear in separate tabs and must be created separately. In addition, Price Profiles do not need to be specified when creating a Production Type Curve. Both Compound Production Type Curves and Derived Production Type Curves can be created.

For details on the creation of Production curves, refer to Production within the Project Editor section in the table of contents of this help document.

 

Compound Production Type Curve

When the 'Add Compound Production Type Curve' button is selected, the following screen appears in a new tab:

 

These options can be configured for the Compound Production Type Curve added:

  • Name: name of the compound production type curve
  • Product: specify the type of product for the type curve (oil, gas, water, etc.)
  • Production Rate: specify units of the production rate
  • Sample Onstream Date: start date of the production curve

Derived Production Type Curve

Derived Productions are dependent on existing Compound Productions. They can be created, but can only be used to forecast a secondary Production (as a By-Product or an Associated Product). When the 'Add Derived Production Type Curve' button is selected, the following screen appears in a new tab:

 

These options can be configured for the Derived Production Type Curve added:

  • Name: name of the derived production type curve
  • Derived Product: specify the type of derived product (oil, gas, water, etc.)
  • Primary Product:
    • Specify the type of primary product the derived product is dependent upon
    • Specify the units of the derived product and primary product
    • Specify the derived product as a By-Product or an Associated Product
  • Production Rate: specify units of the production rate of the derived product
  • Sample Onstream Date: start date of the production curve
  • Start Event:whether the derived production rates are relative to the primary production onstream date, or to a calendar start date

Expenditure Type Curves

Creating an Expenditure Type Curve is similar to creating an Expenditure in the Project Editor. Unlike creating a Production, different Production Type Curves appear in separate tabs and must be created separately. In addition, the options for creating an Expenditure Type Curve is slightly simplified, as the Expenditure Category is not required prior to selecting the Expenditure Type, and details such as Working Interest are not considered.

For details on the creation of Expenditures, refer to Expenditures within the Project Editor section in the table of contents of this help document.

When the 'Add Expenditure Type Curve' button is selected, the following screen appears in a new tab:

 

These options can be configured for the Expenditure Type Curve added:

  • Name: name of the expenditure type curve
  • Currency: specify the currency of the expenditure to be reported in
  • Expenditure Type: specify the type of expenditure (drill, completion, workover, etc.)
  • Sample Start Date: start date of the expenditure

Royalty Type Curves

Creating a Royalty Type Curve is similar to creating a Royalty Burden in the Project Editor. Unlike creating a Production, different Production Type Curves appear in separate tabs and must be created separately.

For details on the creation of Royalty Burdens, refer to Burdens within the Project Editor section in the table of contents of this help document.

These options can be configured for the Royalty Type Curve added:

  • Name: name of the royalty type curve
  • Product: specify the product type that the royalty is applicable to (oil, gas, etc.)
  • Start Date: start date of the royalty
