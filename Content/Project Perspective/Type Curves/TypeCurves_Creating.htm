Creating Type Curves

The creation of Type Curves occurs in the Project perspective, where the 'Type Curves' tab resides. First step of the process of creating a Type Curve is to select the Project perspective button placed at the top of the esi.manage screen:

 

Within the Project perspective, select the 'Type Curves' tab. The following screen appears:

 

To create Type Curves, expand the drop down menu at the upper right side of the window, and select one of the following choices:

  • Add Compound Production Type Curve
  • Add Derived Production Type Curve
  • Add Expenditure Type Curve
  • Add Royalty Type Curve
