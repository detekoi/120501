Selection Constraints

The Selection Constraints tab on the Projects Perspective is for use with the esi.portfolio integration. If you have esi.portfolio, populating this table will populate the export file from esi.manage Analysis results with selection constraints. Changes made in this table will have no effect on data or calculated results inside the esi.manage application.

This table has the same rules applied to it as the selection constraints table in esi.portfolio (and Perspectives), however there are a few options to help you quickly populate it.

  • Copy and paste works for patterns in this table, such as copying the first 12 periods, highlighting the next 10 years, and pasting in a repeating pattern.
  • Clicking the "Set Max = 1" button will set the max for the total and all periods to 1, and leave the minimum at 0.
  • "Clear Rows" button will delete all entries in a given row
  • The arrows will move your focus to the beginning or end of data entered in the row you have highlighted.
