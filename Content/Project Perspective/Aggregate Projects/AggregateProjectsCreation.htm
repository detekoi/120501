Aggregate Projects Creation

To aggregate projects, select the desired projects in the Project Browser and right click to select Aggregate Projects, or alternatively click the summation icon found in groups icons top right.

 

Project Sets can also be aggregated. If a set has been created, right click the set in the Navigator and select Aggregate Projects in 'X'.

 

The Aggregate Projects window will open with the options to select Target Budget as well as Project Manager for Target Projects. The Selected Project window confirms the projects being created.
