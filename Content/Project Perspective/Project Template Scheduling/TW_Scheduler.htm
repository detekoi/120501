Template Well Scheduler

The Project Template Scheduler is accessed from the Project's Perspective, and is only Active if the Project Browser has been populated with Projects from a Template Well Source Budget. (see Active Selection)

To open the Scheduler, select this button from the Project Browser button ribbon.

 

The Distribution Granularity can be either:

  • Monthly - the Monthly option has been provided to 'force' Projects to start on the first day of the month that they will be scheduled in.
  • Daily - the Projects will be scheduled based on the number / period - resulting in them being evenly distributed within the Scheduler display period.

The Schedule Editor will open with whatever Projects were displayed in the Project Browser. The Basic information about each of the Template Projects will be displayed. Information about each of the buttons and their purpose can be obtained by clicking them in the image below.

 

You can change the Periodicity and Focus Period to suit your current needs.  If the Distribution Granularity is finer than the Periodicity, then the scheduled template wells will be distributed evenly throughout the scheduled period (eg. If Daily Distribution Granularity, and Monthly Periodicity, then the Template Scheduler will schedule wells evenly throughout the month if more than one.  If Monthly Granularity and Monthly Periodicity, all wells will be scheduled on the first of the month).

The number of instances of the particular Source Project to be created in a given period is determined by entering the value. You can Copy and Paste values as needed.

 

The Grand Total sums each row - all values are Blue until they have been Saved (also indicated by the * in the Title Tab). To create the Projects with the specified frequency, Save the Scheduler. A progress bar will be displayed indicating the status.

 

The Values will now show in Black, indicating that the Virtual Projects have been created. The next time you enter the Scheduler, all Virtual Projects will be indicated against their Source Template. The counts will be based on the selected Periodicity.

See Promoting a Virtual Project for further information.
