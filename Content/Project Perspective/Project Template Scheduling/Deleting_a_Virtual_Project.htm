Deleting a Virtual Project

Virtual projects that are created via the Template Scheduler can be deleted in the same way as Updating a Virtual Project.   Virtual Projects are non-editable, so they cannot be edited from within the Project List, including deleting.  When a Virtual Project is highlighted within the Project List, the Delete Project button will be inactive.

  1. To Delete an existing Virtual Project, enter new values for the Template Project in the Time Period you want to change (delete the cell contents if you wish to have no Virtual Projects during that time period). As you type in values, the Scheduler will show that there are Changes to be saved - the new values will be Blue (if not all Virtual Projects have been deleted) and the Scheduler shows the *.
  2. When you save the Schedule, only the instances in Blue will result in a change to the Virtual Projects. Within any period that is changed - existing Projects will be deleted and new instances created.
