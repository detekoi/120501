Set PEEP Product Price Scenarios

The interface with PEEP allows the mapping of esi.manage price scenarios with the price scenarios in PEEP. This allows the user to see the impact of high or low price scenarios compared with the base price scenario for example.

The PEEP product price scenarios can be set directly from esi.manage by clicking the PEEP interface drop down from any of the project editor screens and selecting 'Set PEEP Product Price Scenarios'.

 

A screen appears that then allows the selection of the relevant price scenario.

 

Note: the price scenarios have to be configured in esi.manage, in PEEP and also mapped in the mapping file.
