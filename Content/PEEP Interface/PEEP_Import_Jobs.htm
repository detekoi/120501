PEEP Import Jobs

The PEEP interface facilitates the bulk import of cases from PEEP into esi.manage using the PEEP Import Jobs in the Navigator.

 

 

Selecting -  New >PEEP Import Job from the Navigator will open the PEEP Import Job Wizard.

 

From the PEEP Import Job Wizard, select the target budget and the PEEP Model Type to import from. The import job will display the list of cases in PEEP available to import.

 

The "Showing/Total" row at the bottom of the import job displays details about the case list. The first item shows the total number of cases read from the PEEP Case Search vs. the number of cases read from the PEEP Case Filter. The second item shows the number of selected cases vs. the total number of selected cases. The third item shows the number of cases already in esi.manage from the PEEP Case Search vs. the number of cases already in esi.manage from the PEEP Case Filter.

Once PEEP Cases have been selected, they can be imported by pressing the import icon: . The Cases will be imported into the projects list of the selected Budget.
