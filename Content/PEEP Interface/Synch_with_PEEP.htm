Synch with PEEP

Projects can be Synchronized with PEEP. esi.manage measures ( production, OPEX and CAPEX are exported to PEEP. The Calculation in PEEP is run and esi.manage then imports the Ownership, Tax, Royalties and economic Metrics from PEEP, while leaving Capital and Operating Expenditures, and Production values in esi.manage intact.  In this way, esi.manage can be used to track forecast and actuals data, while PEEP is used as the economics engine.

To synch with PEEP

  1. Select the Project(s) to be synched with PEEP.
  2. Initiate the synch action by either invoking the context menu (select projects and right click) and choosing the Synch with PEEP action or by clicking the Synch with PEEP button within each project.
  3. A dialogue box will open with a list of projects to be synchronized, and a drop down box to select a number of Parameters. (Just like Exporting to Peep)
  4. It is very important to ensure that the esi.manage economic group (found in details section of the esi.manage project is an exact match to the model name that is linked to the PEEP case for the interface with PEEP to run.
