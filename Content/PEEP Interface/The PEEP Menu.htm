The PEEP Menu

 

The PEEP Menu can be accessed one of four different ways:

  1. The Project List Button Ribbon
  2. The Project List Right Click (Context) Menu
  3. The Project Set Right Click (Context) Menu
  4. The general Button Ribbon within a project.

Note: The PEEP Menu icon will change to match the last action run through the PEEP Interface to facilitate the easy repetition of work flows.
