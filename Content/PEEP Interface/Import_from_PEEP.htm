Import from PEEP

Once Projects have been imported from PEEP Cases, the data contained in these projects can be refreshed from the PEEP database by selecting the project(s) and pressing the Import From PEEP button located on the Project View button ribbon or the context menu.

Refreshing from PEEP can be done at a project level by pressing the Import From PEEP button from within the Project Details View.

Entire Project Sets can be refreshed by populating a Project Set and right-clicking to Import From PEEP within the Navigator View.

When refreshing projects from PEEP, a dialogue box will appear confirming the refresh action, press OK to continue with the refresh, or Cancel to abort.

 

Note: it is important that the Economic group in the esi.manage project is an exact match to the model name in the PEEP case for the interface to run.
