Update PEEP Case List Cache

The PEEP Case list is stored in a Cache to improve performance.

 

The Cache refresh option is found in the PEEP Interface drop down menu on the Projects Perspective Button Ribbon.

When the refresh is selected, a progress bar is displayed that can cancel the cache refresh should it be needed.

This will allow the refresh of the cache without having to open a PEEP Import Job to ensure that the PEEP link has an up to date case list from PEEP.
