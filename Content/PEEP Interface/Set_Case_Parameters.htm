Set Case Parameters

Within esi.manage there is a function to set a number of Case Parameters in PEEP directly from esi.manage.

Initiate the Set Case Parameters action by either invoking the context menu and choosing the Set PEEP Case Parameters action or by clicking the Set Case Parameters button.

 

The PEEP parameters that can be set from this function are PEEP Price database, PEEP Currency database, Discount start date and Escalation start date.
