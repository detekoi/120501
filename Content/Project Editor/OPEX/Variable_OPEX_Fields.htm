Variable OPEX Fields

Description of Type Options

  • User Supplied – input by the user
  • Selection – a drop down selection, based on choices configured elsewhere. Selections may be filtered based on previously specified or selected data
  • Default – will be defaulted to a value, but may be overwritten
  • Initialized – will be defaulted to a calculated, or propagated value, but may be overwritten
  • Calculated – will be calculated from other data, and cannot be changed
  • Display – information

Comments

For information purposes

User supplied

 

 

  1. Asset
    • Selection of an Asset establishes a relationship between the Variable Opex and the corresponding Source Production.
  2. Product
    • Selection of a Product establishes a relationship between the Variable Opex and the corresponding Source Production, unless the volume unit is an equivalence unit.  If the Product is omitted, then the variable OPEN rate will be applied to all products.
    • The POS%, WI%, and Shrinkage of the Source Production segments are used for Net, Net Risked and Net Risked Sales calculations.
    • Product is used for Grouping and Filtering in Reports.
    • It is also used in Dollar per Volume calculations (Lifting Cost) to associate the Operating Expense with the corresponding Production.
  3. Category and Type
    1. If Category is the initial selection, the Type list will be filtered accordingly.
    2. If Type  is initially selected, Category will be populated with the corresponding Category.
    3. Category and Type are Filter/Grouping options for Reports.
  4. Status
    1. Status is set by User, can be used for filtering in Opex reports.
    2. Status options can be defined in the configuration perspective for improved filtering.
  5. Volume Unit
    • The Units for the Variable Opex
  6. Use Sales Volume
    • Toggle to choose whether or not to calculate variable Opex against the Raw "unshrunk" production, or the "shrunk" Sales production.
    • Using this toggle can allow you to choose not to calculate variable Opex against derived products whose primary product segment has shrinkage.
