OPEX

The Opex screen contains fields to enter the data on the Operational Expenses of the project. Unlike the expenditures editor, the Opex editor focuses specifically on the cost of maintenance, upkeep, and employee salary. Expenditures tend to be one-time costs, and feature all-encompassing budgets. The Opex specifically shows how much it costs to keep the existing well producing.

There are three types of OPEX in esi.manage:

  • Fixed OPEX applies to cost per unit time, and is independent of the volume of production.
  • Variable OPEX is a cost per unit production.
  • Price-Based Opex has a price profile assigned to an Opex type and production stream

Each Opex type has an associated schedule to apply the costs to the project.

Opex defaults can be set up at a hierarchy level. See Opex.
