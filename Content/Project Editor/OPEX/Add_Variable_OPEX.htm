Add Variable OPEX

Variable OPEX is associated with a Production segment, and is defined as a ratio to the volumetric production.

 

Adding Variable OPEX

  1. Start from the Opex Editor in the Project view.
  2. When creating a new project, any Opex defaults based on hierarchy will be present. These defaults can be overwritten by duplicating the default Opex
  3. Click the Green Plus Icon to add a new line of Variable Opex.
  4. Fill out the necessary fields in the newly created line.
  5. Move to the Schedule view with the newly created header highlighted and enter a Start Date.
    • There are two main ways to add to the schedule, using the Green Plus Icon to add a single line and define Start and End Date/Duration, or use the Append Tabular Entries Icon. Append Tabular Entries will ask for periodicity and how many periods to add, and will add that many lines.
    • When using a single line in the schedule with no Periodicity selected, the Gross Amount will be allocated between the Start and End Dates.
    • Monthly, Quarterly and Annual Time Periods (Daily and Weekly are allowed but are not commonly used) will be interpreted based on the configuration of the requested calculations.

    • Using the End Date in conjunction with the Start Date will auto populate the Duration (days) column, and the Duration (days) column used in conjunction with the Start Date will auto populate the End Date.

  6. Save the Project

Delete Variable Opex

  1. Left click to select the OPEX line you wish to delete.
  2. Click the red "X" to remove the line completely.
  3. 3. If a default has been overwritten, deleting the overwritten Opex will restore the default.

 

Duplicating a Header

Using the Duplicate Selected Rows button, header(s) can be quickly multiplied. When duplicating headers, the schedule associated with the original is also duplicated.
