Attachments

Projects can have attachments associated with them, through the attachments tab.

 

Uploading an Attachment

To attach a file to a Project, start by pressing the Upload button (shown above) and then navigate through the pop-up to the file you want to attach.  Select the file and press the open button.

 

Downloading an Attachment

To download an Attachment onto your local computer, press the Download button (shown above).

When the file transfer has completed, the following dialogue box will appear:

 

You can open the file directly, or open up the default download folder and move the file to a different location.

 

Deleting an Attachment

To remove an attachment from a Project, press the Delete button.  You will have to confirm the deletion in the pop-up dialogue before the file will be deleted.

Warning:This will delete the attachment for everyone forever! (Unless someone re-uploads it)

Notes:

  1. The default download location can be set by navigating to Window > Preferences > Attachments.
  2. If you try to download an attachment to a folder you don't have the correct permissions to, you will receive an error.
  3. There is a size limit to the space available for attachments in esi.manage. This can be edited in the Configuration>System>Miscellaneous>Document Attachments, given you have the appropriate permissions to edit the Configuration.
