Asset Fields

Description of Type Options

  • User Supplied – input by the user
  • Selection – a drop down selection, based on choices configured elsewhere. Selections may be filtered based on previously specified or selected data.
  • Default – will be defaulted to a value, but may be overwritten
  • Initialized – will be defaulted to a calculated, or propagated value, but may be overwritten
  • Calculated – will be calculated from other data, and cannot be changed
  • Display – information

Asset Operators

Cost Only An asset marked as cost-only is not subject to truncation during an economic analysis run when the truncation type is CALCULATED. “cost-only” has no effect if the truncation type is NONE. Cost-only is also ignored when the truncation type is ASSET_ECONOMIC_LIMIT_DATE. Boolean  

 

Relationships and Impacts

  1. Asset Name and UWI
    1. The Asset Name is what will appear in the Asset Drop downs for the other Measures.
    2. If UWI has been supplied, this will be appended to the Asset Name for those drop downs.
  2. Wellcount POS%
    1. This is the Forecast Probability of Success for the Asset.
    2. This is only used for reporting purposes. If you wish to set Risk or Probability of Success for individual measures, see the Dependencies page.
