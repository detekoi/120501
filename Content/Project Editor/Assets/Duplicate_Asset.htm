Duplicate an Asset

Duplicating is similar to cloning with the major difference that it occurs within the same project.

 

To Duplicate an Asset:

  1. Left click to select the Asset you want to duplicate from the list.
  2. Right click and select "Duplicate Asset" or use the Duplicate Asset button.
  3. When prompted, identify how many duplicates you wish to make and the number of dates to stagger them by.
  4.  The Duplication process can take several seconds to a minute depending on the number of entries being duplicated and the amount of data in the base asset.
  5. Once the new assets have been created, change the names to reflect their accurate titles, and any other necessary fields to individualize them.
  6. Pay specific attention to the "Dependency" fields, as any copied dependencies will be identical to the asset they were duplicated from.

 

Note: This is very useful for multi-well projects. Fully enter all information on the first Asset, and then use the duplicate function to create 50, 100, or however many new Assets are necessary, and then only change the handful of fields that need to be customized (rather than the entire asset).
