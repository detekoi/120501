Add an Asset

Any asset can be tracked in this screen, including Wells and a Group of Wells (Producing Pools).

 

To add a new asset

  1. Open a Project and click on the"Asset" tab at the bottom of the Project Editor view.
  2. Right click anywhere in the table and select "Add Asset" from the menu. There is also a button in the top right of the view to do this.
  3. This will create a blank row in the table which you can enter. Fill in as many of the fields as possible by double-clicking them, and either typing the information in or selecting options from a drop-down list.
  4. To save the changes, click anywhere outside of the newly created row and the changes will be committed to the database.
  5. Repeat as necessary for all assets associated with the project.

 

Delete an Asset

To delete an asset,

  1. First select the asset you wish to delete.
  2. Next, right click to open the menu, and select "Delete Asset", or you can use the delete button as shown above.

After this, you will see one of two dialogue boxes:

  1. A confirmation asking if you are sure you want to delete the asset. Clicking yes will delete it.
  2. A message informing you that you do not have permission to delete the asset. You must be the original or current Project Manager or an Administrator to delete it
