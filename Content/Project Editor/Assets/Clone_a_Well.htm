Clone an Asset

This function allows you to copy an Asset (Forecast data only) from another Project, into the current Project.

 

To Clone an Asset:

  1. Start by accessing the Project and selecting the Asset tab.
  2. Click on the "Clone Asset" button.
  3. A new window will open.

 

 

  1. Select the Source Budget and Source Project you wish the Cloned asset to be brought in from.
  2. Select a Source Asset from the desired Budget/Project pair using the drop-down list.
  3. The earliest date of the selected asset is automatically entered once you select it.
  4. You can Shift the Asset's dates by setting a new earliest date if desired.
  5. Imported assets can be renamed once imported into new project.
