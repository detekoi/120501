Shift Asset Dates

It is possible to shift the dates of an Asset. Shifting an Asset will shift related milestones as well as dates for Expenditures that the asset has associated Well Count Dependencies on.

 

To Shift Asset Dates:

  1. Open a Project and access the Asset editor.
  2. Right click the Assets table and select "Shift Dates", or press the Shift Dates button in the context menu.
  3. The dialogue shown above will open.
  4. Select the Assets option.
  5. A list if Assets appears in the Group list.
  6. Select the Assets you want to shift. Single-click to select, or hold down Ctrl and click to select multiple Assets.
  7. Enter a CAPEX and/or OPEX Inflation %, if applicable.
  8. Select to shift by a set number of days (By Delta), or By Start Date, in which case esi.manageTM will calculate the number of days for you.
  9. Click "Shift Dates".

 

For other shifting options, please refer to "Shift Date Options"
