Asset Attributes

Custom attributes are a mechanism for appending additional data. Asset attributes, once configured will appear on the right hand side of the assets screen.

The asset attribute information can then be captured for each asset in the project.

The asset attribute can alternatively be appended to the asset table in the top of the screen. This is done by checking the 'Display in tables' flag when configuring the asset attribute.

Editing a Vector or Time Series attribute

There are 3 non single value types of asset custom attributes.

  • Vectors: - Allows you to store distributions as custom attributes as a single entity.
  • Time Series - A relative attribute to the Asset or Project that will shift in time appropriately.
  • Absolute Time Series - A fixed in time attribute that does not shift.

Editing a Vectors or Time Series:

  1. Double click on the Value column for the corresponding row.
  2. Click on the button with the ellipses to open the Custom Vector Editor or the Custom Time Series Editor window.

  3. Fill in the values in the value column and hit OK
