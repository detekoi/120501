Add a Resource

Resources can be allocated to the table using the "Add Resource" button in the toolbar above the table.

  1. Select the appropriate Capacity File. This is used to determine Allocations across all Projects consuming Resources from this Capacity File.
  2. Select the Resource Type.
  3. Select the Expenditure (Allocated To) the resource is associated with.
  4. The Start and End Dates will initialize to the Expenditure's dates.
    • You can modify these dates - for example to indicate a set up or travel time for the Rig.
  5. Enter an Allocation of how many resources to commit.
    • The Unallocated field will calculate across all Projects using this Capacity File and Resource, in relation to the Available Resources in the Capacity File.
    • The allocation dates are considered in this calculation.
    • A negative Unallocated amount indicates over-allocated.

The Total fields at the top of the page give a summary of the project resources.

 

  • Maximum Allocated Resource: How many total Resources you have committed to the Project. This includes Resources of all types. (The sum of all "Allocated" fields).
  • Gross Cost: The sum of all of the allocated Resources multiplied by their respective Unit Costs.
  • Maximum Risked Resource: The sum of all allocated Resources after being multiplied by the POS% of their dependent activities.
  • Risked Cost: The sum of all allocated resources multiplied by both their POS% and Unit Cost.

 

Delete a Resource

You can remove Resources from your project if necessary.

 

To Delete a Single Resource

  1. Single-click the resource you want to delete.
  2. Right click the table and select "Delete Resource" from the menu.
  3. When prompted, click "OK".

 

To Delete All Resources

  1. Click the "Delete All" button from the menu at the top right of the table.
  2. Select "OK".
