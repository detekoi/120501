Map AFEs

AFEs are assigned in the Map AFEs screen. You access this screen by clicking the Assign AFE button.

To Map an AFE to an Expenditure:

**This step assumes that you have already created the AFEs on the AFE screen**

  1. Click the yellow bucket "Assign AFE" button above the expenditure line item table.
  2. The screen shown above appears.
  3. The left-hand table shows the Expenditures in the current project, and the right hand shows a list of AFEs based on the option selected in the Show AFEs section above the AFE table.
    • Available to Project shows AFEs added to that specific project on the AFE screen.
    • Assigned to Project shows all AFEs currently assigned to Expenditures in the current project.
    • Unassigned by Project shows all AFEs assigned to the project but not already mapped to an Expenditure
    • Unassigned by Budget shows all AFEs attached to the current budget but not already mapped to an Expenditure.
  4. Select the Expenditure you want to map an AFE to. It will be displayed in the Expenditure Details pane.
  5. Select the desired AFE and press the "down" arrow to map it to the Expenditure.
