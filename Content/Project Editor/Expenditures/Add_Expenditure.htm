Add an Expenditure

Expenditures are tracked using a 3-stage process.

  1. Enter the Expenditure Line item
  2. Enter Forecast, Field Estimates, and Actuals as they become available.
  3.  Assign an AFE (or AFEs) to the expenditure.

 

To add an Expenditure Line Item:

  1. First access the Expenditure screen in the Project Editor.
  2. Click the "Add New Expenditure" button as shown above, or begin entering data on the empty line (line 140).

  1. This will create a blank new row in the Expenditures table. Double click on a field to edit it.
  2. Fill out the required (bolded) fields and any others that are applicable to your workflow.
  3. Repeat as necessary for all expenditures that need to be added.

To add a Forecast, Field Estimate, or Actual

  1. Select an Expenditure Line item from the Expenditure table for which you want to enter forecast, field estimate or actual data.
  2. In the Expenditure Details table, select the tab that corresponds with the numbers you are entering (ie - Field Estimates go in the Field Estimate table) - click the "Add New Field Estimate"
  3. A new line appears for you to enter information into.
  4. Enter the bolded information - Start Date, End Date, and Gross Amount.
  5. Working Interest is inherited from the Ownership Table, and "Net Amount" is calculated automatically. (Gross Amount * WI%).
  6. Click on another line in the table to commit the change.
  7. Notice that now the total for that amount has been updated in the base line item.

Note: Some fields may be read-only depending on the Budget status the Project is assigned to. In other words, you cannot add "Actuals" while in Budget mode.

To Expand CAPEX Detail

From the Expenditure Detail - Forecast tab you can expand the detail to a finer date granularity.  

  1. Select a  forecast line.
  2. Right-click toExpand to Annually, Quarterly, Monthly or Daily.

See Map AFEs for assigning AFEs to the expenditure.

 

Delete an Expenditure

You can delete Expenditures during the stage they were added only. (You can delete an Expenditure added during the Budget phase only while it is still a budget. Once the budget becomes a forecast, you cannot delete it, however youcan delete items added during the forecast phase.)

Note: this process is the same for Expenditure Line Items, Forecast Amounts, Field Estimates, and Actuals - the only difference is the table you are deleting from.

  1. First select the expenditure you wish to delete.
  2. Next, right click to open the right click menu.
  3. From the right click menu, select "Delete Expenditure". Instead of using the menu, you may choose to use the Delete Expenditure button as shown above.

 

You will be prompted by one of two dialogues:

  1. Either you will be prompted to confirm you want to delete the item, select yes to do so.
  2. Or you will be informed you do not have access to delete an expense in this project. Only users with write access to the project can do this.
