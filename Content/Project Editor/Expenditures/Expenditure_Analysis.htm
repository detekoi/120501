Expenditure Analysis

  1. From the Expenditure editor, click the Expenditure Analysis button (chart icon) to access the analysis screen. This provides you with a graphical view of your expenditure schedule, including breakdowns of the forecast, field estimates, and actuals.
  2. You will have to define the criteria you want to see by selecting the AFE(s) and/or Expenditure(s) from within the current project that you want to filter on. Define filters, definitions, blending logic and chart type then a time frame to see table and chart breakdowns of the capital spending over that period.
  3. Click the Auto Update icon  to turn it on -  the table and chart will automatically refresh when you open a new project to represent new data. If auto update is NOT selected, you will have to manually click the "refresh" button for new information to display, even if you open a new project.
  4. The Table Data can be exported to Excel.

 

Chart Options

There are a number of options to modify how the Chart is displayed. These are accessed by right clicking within the chart area:

  • Scale - linear or logarithmic
  • Series Type - bar, line or area
  • Data Mode
    • Period Total - display each month's periodic expenditure amount
    • Cumulative from Inception - cumulative total by period, including Expenditure amounts which occurred prior to the Starting Date.
    • Cumulative from first visible Period - cumulative total by period from the Starting Date.
  • Change Series - select which of the available data columns (Forecast, Field Estimates, Actuals and Blended) should be charted.
  • Save Chart Image - access a file dialogue to save the chart as an image file, which can be used within other documents.
    • The image can also be utilized within the windows Clipboard, as described below.
