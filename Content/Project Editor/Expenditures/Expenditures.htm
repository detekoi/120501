Expenditures

The Expenditures page tab is where Capital Expenditures are Forecast and Tracked for a particular activity.

 

In addition to the required Expenditure data (Type, Asset, Amount, Currency, Start and End Date), there are a number of additional data elements:

  • Started and Ended - flags which aid in Project Tracking workflow.
  • Actuals and Field Estimates - display aggregate actuals data entered against the Expenditure.  These are entered into the Expenditure Details tab.
  • Event Link and Event Offset - These can link a given line item to the economic limit of the asset (eg. floating abandonment)

 

Each Expenditure can be Scheduled to provide a more precise spending profile.
