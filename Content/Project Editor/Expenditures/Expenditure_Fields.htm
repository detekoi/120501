Expenditure Fields

Please note that the fields listed here may be different (and/or in different order) than what you see in esi.manage.  This is due to the configuration of columns, and can be edited by clicking the Change Columns button

Description of Type Options

  • User Supplied – input by the user
  • Selection – a drop down selection, based on choices configured elsewhere. Selections may be filtered based on previously specified or selected data.
  • Default – will be defaulted to a value, but may be overwritten
  • Initialized – will be defaulted to a calculated, or propagated value, but may be overwritten
  • Calculated – will be calculated from other data, and cannot be changed
  • Display – information

Relationships and Impacts

  1. Category and Type
    • If Category is the initial selection, the Type list will be filtered accordingly. If Type is initially selected, Category will be populated with the corresponding Category.
    • Category and Type are Filter/Grouping options for Reports and Expenditure Analsysis.
  2. Asset
    • Selection of an Asset establishes a relationship between the Expenditure and all other Measures (CAPEX, Production, Reserves and OPEX).
    • WI% synchronization
    • Risk (POS%) propagation
  3. Status
    • Part of CAPEX Report(s) output
  4. Gross Amount
    • A contributor to the Estimated Gross Estimate at Completion
    • Initializes the Expenditure Schedule allocation
    • Will be updated based on changes to the Expenditure Schedule
    • Will be updated based on committing the Gross Estimate at Completion
  5. Start Date / End Date
    • If an Expenditure Dependency chain has been established, the Start Date for an Expenditure must be later than the End Date of the Expenditure it is Dependant On.
    • End Date must be equal to or later than the Start Date
    • Resource assignment will be dependant on the Start Date.
  6. Event Link / Event Offset
    • Setting an Event link will start the expenditure at the economic limit date of the asset, with the designated offset in months.
  7. Started
    • The Start Date can no longer be changed.
    • The Probability of Doing this Expenditure must now be 100%.
    • The information for the Expenditure now shows up the Actual row in the Gantts.
    • Milestone Report
  8. Ended
    • The End Date can no longer be changed.
