Project Header

The project Header fields appear on each screen within the project for easy identification.

  1. Budget: The Budget that contains the Project.
  2. Manager: The assigned Project Manager for the project. This can be changed by selecting a new manager from the "Project Manager" pick list in the Overview section of the Details page. The Project Manager is the main security level, and largely determines the visibility and ability of users to read and/or write to the project. See the User/Project Manager Access page for more details on using Project Managers to control security, or the User Roles section for more details on access rights.
  3. Status: Reflects the stage the project is at. Default options include Approved, Budgeted, Completed, In Progress, Inventory, and Proposed.
  4. Currency: The currency set for the project.
  5. [Read Only]: If the budget is set to read only, the [read only] flag is displayed in the Project Header.
