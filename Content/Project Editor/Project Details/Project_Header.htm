Project Details

The Project Details screen is where key information about a project is entered. From this screen you can edit the Project Hierarchy, Project Name, and other details you initially entered when creating the project. This is typically the data that is used for filtering and finding Projects later, so it is important that this information be kept up to date.

If you make an incorrect change on the Project Details screen, you can undo the change by clicking the Undo button on the Project Details toolbar.
