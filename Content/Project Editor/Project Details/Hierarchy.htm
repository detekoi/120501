Hierarchy

The Hierarchy represents your company's business/organizational structure. This information organizes and categorizes a Project, and can be used for Filtering and Grouping by the Project Browser and Reports.

The Hierarchy, and the available options are defined within the Configuration; double-click an "Instance" column to open the selections available for that hierarchy type and select the applicable option.

All fields in the Project Hierarchy are required.  To prevent the manual entry of every field, defaults can be set in the Configuration (eg. ~No Access Restrictions is a default entry) and default entries are often denoted by a tilde ("~").
