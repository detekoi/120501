Archived Budgets using Custom Attributes

There some exceptions to using Custom Attributes within an Archived Budget that you should be aware of:

 

Custom Attribute in Archive Budget

Archived Budget contains values for an Attribute Type: The attribute may not be deleted, renamed or altered with the exception of:

  • Category name may be changed.
  • Enabled flag may be changed
  • Sort Order may be changed
  • Default value may be changed
  • Is Lower Better
  • Is Rankable
  • Read Only
  • Show Comment

 

Custom Attribute in Archive Budget with Enable Flag set to False

Archive Budget contains values for an Attribute Type with Enabled Flag set to False. When viewing the attribute value screen for the archive that has a value, the value will be shown.

When viewing the Attribute value screen for a non-archive that has a value, the value will not be shown. The value will not be available for ranking or custom charting.

Custom Picklist selection is in use by the Archive. The Picklist selection and Picklist may not be edited or removed.

You may however add new selections to the Picklist.
