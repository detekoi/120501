Project Custom Attributes

Project Custom Attributes, found in the Project Details tab, are a way to annotate a Project with information beyond what is represented by the Hierarchy. The Groups (Cycle Time, Geological, etc.) and the corresponding Attributes are defined within the Configuration.

  • Values may be entered into these attributes to further define the project.
  • To add comments, click on the Comment bubble .
  • If a comment exists the comment bubble will have a check mark on it . If you hover over the comment bubble the comment will be visible.

You can collapse or expand the Custom Attributes by either clicking on the arrow at the attribute Grouping or by using the expand all/collapse all buttons.

 

Editing a Vector custom attribute

Custom attributes can be a vector and as such values may be entered for all members of the configured vector. Vectors will appear as a regular row of the Project Custom Attributes window.

Editing a Vector:

  1. Double click on the Value column for the corresponding row.
  2. Click on the button with the ellipses to open the Custom Vector Editor window.
  3. Fill in the values in the value column and hit OK
