Project Attributes

Custom Attributes are a mechanism to annotate a Budget, Project set, Project or Asset with additional information.  This data can be used to track Regulatory requirements, Geological Information, Partner Information, Economic or Financial Metrics, or Activity data such as Drilling Rig days.

Project attributes can also be used to calculate project metrics, which in turn can be used for ranking purposes. See bottom of page for more details.

The following image shows how the Attributes will appear within a Project, which will make the explanation of how to Configure the various Attributes clearer.

 

See Custom Attributes Configuration

One of the main benefits of project attributes is once they are completed, projects can be filtered by them in the project browser.

 

In addition, project attributes can be used to calculate Project Metrics and then used in Ranking. Please note for use in project metrics and ranking, the attribute has to have the rankable flag checked. See below.
