Dependencies Options

You can interact with the Dependency page tab either using buttons on the right side of the page or through the Context (Right Mouse Click) menu. The options that are accessible are:

  • Create, Delete, Cut, or Paste - create a New Dependency (Expenditure, Production, Fixed OPEX, Resource allocation) or Cut the existing Dependency, or Paste the previously cut Dependency or Delete the item.
  • Edit an existing Dependency (Expenditure, Production, Fixed OPEX, Resource allocation) in a pop-up box.
  • Unlink - Unlink the selected item.
  • Match Actual Offset - the process of setting the Desired Offset to be the same as the Actual Offset.
  • Match Desired Offset - the process of setting the Actual Offset the same as the Desired Offset.
  • Synchronize Dependency Offsets - the process of setting the Actual Offset Date to match the Desired Offset Date for all Dependencies.
  • Auto-resize Visible Columns - columns in the Dependencies table are expanded or contracted based on the length of the content.
  • Change Columns - enables the addition or removal of variables (columns) in the Dependencies table.
  • Refresh - updates the Dependencies table with any changes made in other tabs of the Project Editor (Expenditures, Production, OPEX, Resources).
