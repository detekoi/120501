Dependency Offsets

Offsets are the difference between the Start date of one measure, and the Start/End date of the measure it is Dependent on.  Whether or not the Linkage is Start -> Start or End -> Start determines if the Offset is between Start Dates, or Start and End Date.

 

Match Offsets

The Desired Offset can be edited within the Dependency screen, and can be used to reflect travel time of a Rig, delay in starting Production, or any other gap in time between measures.  Any Desired Offset that does not match the Actual Offset will be highlighted in yellow.

Synchronize Dependency Offsets

Synchronizing Dependency Offsets is the same as Matching Desired Offset, except that all measures will have the Actual Offset changed to match the Desired Offset.
