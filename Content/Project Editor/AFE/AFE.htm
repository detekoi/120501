AFE Page

 

 

When AFEs are being used, they will contain the Field Estimates and/or Actuals that relate to an Expenditure Forecast.

This page utilizes three related tables.

  • AFEs - this list the AFE Header information (AFE Name, Description, Gross Amount, etc.). The AFE selected in this table will determine what information is populated below.
  • AFE Details - provides access to the Field Estimates and Actuals which are associated with the selected AFE.
  • Expenditure Line Item - displays some basic information about the Expenditures which are mapped against the AFE.
