Production tab

The Production tab will allow the user to enter Production actuals.

Editing a measure (Production)

  1. Select an Asset from the drop-down menu
  2. Select a Product from the drop-down menu.
  3. Select the Type from the drop-down menu
  4. Select the Volume Unit associated to the Product that was selected previously.
  5. Select the Period by entering a year in the `Show year`
  6. Enter values into the `Production`table.
