History

The History tab will allow you to edit non-forecast values of ‘Production’, ‘Revenue’ and ‘OPEX’.

 

History fields

The following fields must be defined in order to enter values for a given year:

  • Asset: Selection
  • Product: Selection
  • Type: Selection
  • Volume Unit: Selection
  • Year: Selection

Description of Input Options

  • User Supplied - input by the user
  • Selection - a drop down selection, based on choices configured elsewhere. Selections may be filtered based on previous specified or selected data
  • Default - will be defaulted to a value, but may be overwritten
  • Initialized - will be defaulted to a calculated, or propagated value, but may be overwritten
