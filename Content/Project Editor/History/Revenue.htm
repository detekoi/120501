Revenue

The Revenue tab will allow the user to enter Revenue actuals.

Editing a measure (Revenue)

Revenue actuals are entered using a 6-step process.

  1. Select an Asset from the drop-down menu.
  2. Select a Product from the drop-down menu.
  3. Select the Type from the drop-down menu.
  4. Select the Currenct Unit from the drop-down menu.
  5. Select the Period by entering a year in the `Show Year`.
  6. Enter values into the revenue table.
