OPEX

The OPEX tab will allow the user to enter OPEX actuals.

Editing a measure (OPEX)

Production actuals are entered using a 7-step process.

  1. Select an Asset form the drop-down menu.
  2. Select a Product from the drop-down menu.
  3. Select the Opex Type from the drop-down menu.
  4. Select the Currency Unit from the drop-down menu.
  5. Select Type from the drop-down menu.
  6. Select the Period by entering a year in the `Show Year`.
  7. Values just be entered in the OPEX table.

Refreshing data

For refreshing the data within the tables press the ‘Refresh’ button
