Add Production volumes

  • To enter production volumes, a header row must be created in the upper table of the production editor screen by clicking the add production button with the volume tab selected.
  • Create a header row for each separate product with a decline type of either compound (includes tabular as well as formulaic) or derived (associated or by product). Once the symbol indicating missing fields is gone , click away from the line and then click back into it to access the detail in the lower left table.
  • Other buttons on the same ribbon will delete a header row, open the production analysis screen, duplicate a row, edit production actual, hide and change the orders of the columns in the table (configure columns) and refresh the screen.
