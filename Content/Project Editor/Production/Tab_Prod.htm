Tabular Production

Tabular production is entered by creating a product header row (Oil) with a compound decline type. Once the header row is created, in the compound curve editor screen, select 'tabular' as curve type at the top of the data entry column. To edit values in the production curve editor, you must change the edit mode to Tabular (top right of editor, discussed below). If you want to create multiple entries at once, or have a little more control over the dates, you can click on the Customize tabular production period durations button (below).

 

As the compound decline curve editor supports entry of both compound and formula based curves, you need to change the view to ensure you are looking at the tabular entry screen.  

There are a number of tools along the top of the curve editor pane provided to make entering tabular data simple.

 

Customize tabular production period durations.  

By clicking this icon, a new screen appears. This allows you to append multiple monthly entries either using calendar months or ideal months (365.25/12 = 30.4375 days per ideal month).

When changing the End Date or Duration, the system retains the rate value by default and calculates the total. This can be changed by selecting Total instead of Rate in the drop down. The total will then be retained when changing the End Date or Duration, and the rate will be recalculated. This change is not saved in the metadata and will revert once the window is closed.

Note: You can select this button in Closed Budgets, as a means to view tabular production.  Since it is a Closed Budget, the data will be read only.

 

Change how tabular values are changed and edited  

This button allows the user to view/enter the data as a daily rate or as a total volume for the duration without going into the “customize tabular production period durations” screen.

 

Buttons to change tabular entries duration and periodicity  

These buttons change the periodicity from monthly to yearly data and also extend the durations from one to fifty years.

 

Change columns  

This button allows the user to change the order of the reserve classification columns.

By right clicking on each of the columns the application also supports cut, copy and paste functionality allowing the users to copy and paste tabular data from a spreadsheet.
