Production Analysis

The Production Analysis screen provides you with a powerful tool for tracking your production, and the impact of downtime, curtailment, and other factors in real time. This screen provides both a tabular and graphical breakdown of your production profile. The chart can be displayed as a line, bar or area chart.

  1. From the Production editor, click the Production Analysis button (chart icon) to access the Analysis Screen.
  2. You will have to define what you criteria you want to see by selecting which Asset(s) and Product(s) from within the current project to filter on. Define filters, definitions, Units, Royalty and chart type then a time frame to see table and chart breakdowns of the volume production over that period. Note that the Asset and Product list only include those available to the current project.
  3. Click the Auto Update icon  to turn it on - the table and chart will automatically refresh when you open a new project to represent new data. If auto update is NOT selected, you will have to manually click the "refresh" button for new information to display, even if you open a new project.
  4. The Table Data can be exported to Excel.

 

Chart Options

There are a number of options to modify how the Chart is displayed. These are accessed by right clicking within the chart area:

  • Chart Scale - linear or logarithmic
  • Chart Series Type - bar, line or area
  • Data Mode
    • Period Total - display each month's periodic Production
    • Daily Rate - display a Period's production as the average Daily Rate. This updates the Table Data as well.
    • Cumulative from Inception - cumulative total by period, including Production amounts which occurred prior to the Starting Date.
    • Cumulative from first visible Period - cumulative total by period from the Starting Date.
  • Change Series - This feature is now in the Data Set section of the analysis parameters. Use this to add/remove series’ from the chart and legend.
  • Export Chart to File - access a file dialogue to save the chart as an image file, which can be used within other documents.

  • Maximize Chart – Clicking this will expand the chart display to cover the production analysis table of data above. Clicking it again will return the display to default.
