Production

The production editor is where you model production forecast and track production actuals information.

The screen is split into three areas.

  1. Upper table with five separate tabs showing production volumes (containing product header rows as compound or derived decline types), downtime (per asset), shrinkage (per asset or product), Reserve (showing economic limit dates as well as technical and economic gross reserves), and Quality (can set time variant quality for each asset and/or product)
  2. Bottom left table which represents the detail behind the compound or derived header rows. The table has separate tabs for Reserve categories (eg. Proven, Probable, Possible, Contingent, etc) and within each category, separate columns for Reserve classifications (eg. P1D, P1U etc), with separate columns inside each classification for each segment of the decline curve (Curve 1, Curve 2...).  In the example below, there is a 2 segment curve entered as the 1P classification, and because the 1P/2P/3P classifications are cumulative and nothing is entered for 2P or 3P, they inherit the values of the lower classification.  See Reserve Classifications for more information on possible ways to configure Reserves classifications.
  3. The chart in the bottom right corner relates to the detail in the header row and detail in the bottom left table.
