Compound Production

When Compound is selected as the Decline type in the header row in the upper table, the bottom left pane (curve editor pane) is initialized.  Selecting Compound production will allow you to enter either formulaic production curves (covered on this page), or tabular production.

 

Curve editor Pane showing Formula view

 The curve editor pane has separate tabs for each reserve category (reserves, contingent resources, etc). Within each reserves category are columns for the reserve classifications (Proved, probable, possible etc). Within the reserve classifications are the curve segments. Each segment is labelled as Curve 1, Curve 2....(n) etc so that each reserve classification can contain one or more curve segments.

  • add an additional segment curve, click in the reserve category and classification and the additional column is created by clicking the icon at the top of the curve editor pane. By clicking on an existing curve, a new column will be created to the right of the existing selected column. Note that it will automatically set the start date to follow the previous end date. To delete a segment curve, click on the icon next to the green plus icon.
  • An overall decline curve can be created from a mix of different formulaic and tabular segment curves.
  • As the curve editor screen can support edits to tabular and formulaic curves, the view of the pane can be changed from formula to tabular by clicking on the ‘change edit mode’ icon, see below for more information.
  • As you enter data in the reserve classification (eg 1P) column it will automatically populate the 2P and 3P with the same number.  This is intended to make data input simple if you only wish to use a single curve
  • If you wish to add specific data to the 2P or 3P columns click on the Curve Type and select Tabular again to make the column editable (letters go black)

 

Production Curve (Formula view)

  • You can select a curve type and begin filling in the details of the production curve.  When esi.manage has enough information it will fill in the rest of the cells, based on the information you have provided.  User provided cells are shown in white and calculated cells are shown in yellow.
  • As you enter production into the curves editor pane, the chart in the bottom right pane will automatically draw the completed curves.  This curve does not take any capacity constraints, shrinkage or downtime into effect when it is drawn, but instead gives you an instant visualization of the curve you are creating.

 

Fields:

  • Curve Type - Select from Linear, Hyperbolic, Exponential, Harmonic, or Tabular
  • Start Date - initialized either from the start date provided in the production segment line, or from the previous curve's end date
  • End Date - calculated based on the parameters provided
  • b-exp - required for the Hyperbolic decline curve type.  This must be provided and cannot be a calculated value.
  • Qi - this is the initial production rate
  • Qf - this is the final production rate for the curve
  • di % - this is the annual initial decline rate (expressed as a percent)*
  • df % - this is the final decline rate (expressed as a percent)
  • Duration (mo) - the duration of the decline curve in months
  • Production - the total cumulative production from the curve segment

 

*esi.manage requires a yearly effective decline value for di%. This is the % reduction in production at the end of the 1st year. For example,  if your initial production is 100 bbl/d, and you expect a 90 bbl/d production at the end of the 1st year, this represents an effective decline of 10% and di% = 10%.

Note: that the effective decline (d) is different from a nominal decline (a) that you would use straight into an exponential function, like the ones you would find in Excel, for example. There: y=exp(-a*x). The two, expressed as % or fractions of 1, are related by the following equations: a = -ln(1-d); d = 1-exp(-a).
