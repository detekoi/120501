Production Import

You can directly bring in production numbers from an external file, provided the file is in a compatible import format.

 

Begin by selecting File>Import from the top toolbar.  

 

  1. A dialogue box will appear, and you navigate to the Production option
  2. The editor shown at the top will appear.
  3. Browse to the desired file to import
  4. If the first row is a header, check that toggle.
  5. Check the toggles for the types of production you would like to import.
  6. (Optional) Scan File to verify the import file and correct any errors that pop up. Any records with errors will simply not be included in the import.
  7. Click "Finish".
  8. Close the import window once the import has completed.

 

See also Importing & Exporting>CSV Import Process and the topics contained within it for more information on the importing of production data.
