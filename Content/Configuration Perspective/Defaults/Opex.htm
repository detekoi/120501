Opex

 

The Opex defaults page is where defaults for Opex can be configured by hierarchy. Once configured, Opex will be applied across all projects that have the associated project hierarchy attributes.

Defaults can be created for Fixed, Variable, and Price-Based Opex.

 

By setting up a high level of hierarchy, the hierarchy can be drilled into set up Opex at different levels. For example, Opex has been set up each at the Cecilia, Alberta, Canada, and Canada level. Creating a project in Cecilia will generate Opex at the Cecilia level, as well as assume the Opex from the previous nodes as seen by the greyed out lines of Opex.

 

For information on creating Opex, see Opex.
