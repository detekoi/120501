Length Units

Depth is an optional Field for an Asset; it is not used in any calculations and is provided solely for informational purposes.

 

 

Table Options

 

To Add A Length Unit

  1. Scroll to the last line in the table (it will be empty).
  2. Enter the Abbreviation (up to 8 characters), this must be unique within the table.
  3. Enter the Description (up to 50 characters).
  4. Click outside the line to save the change.

 

Changing A Length Unit

Once an Length Unit has been used within a Project, there is only one option for changes:

  • Rename the Abbreviation or Description

Deleting a Length Unit that is in use is not allowed.
