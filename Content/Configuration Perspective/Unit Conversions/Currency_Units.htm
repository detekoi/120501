Currency Units

Expenditures and OPEX entries, as well as Revenue for Production Actuals (if supplied) require a Amount and Currency Unit. The desired Currency Units and the Conversions between them are defined within the Configuration. These entries will also be available as Units of Measure options for Reports, Analysis Cases and Ranking.

 

Table Options

 

To Add A Currency Unit

  1. Scroll to the last line in the table (it will be empty).
  2. Enter the Currency Unit Abbreviation (up to 8 characters).
  3. Enter the Currency Unit Description (up to 50 characters)
  4. Click outside the line to save the change.

 

Changing A Currency Unit

Once a Currency Unit has been used within a Project, a Currency Conversion entry, or a Report, there is only one option for changes:

  • Rename the Abbreviation of Description

Deleting an Currency Unit  that is in use is not allowed.
