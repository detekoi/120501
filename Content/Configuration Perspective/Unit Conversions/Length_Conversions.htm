Length Conversions

The program will automatically convert lengths when you change from one unit of measurement to another. This editor allows you to define the conversion rates for those calculations. *The purpose is primarily to allow you to define conversion rates for units that you add after-the-fact rather than edit the ones already there.

 

 

Table Options

 

To Add An Length

  1. Scroll to the last line in the table (it will be empty).
  2. Enter a "From Quantity"
  3. Select the "From Length" unit.
  4. Enter a "To Quantity"
  5. Select the "To Length" unit.
  6. Click outside the line to save the change.
