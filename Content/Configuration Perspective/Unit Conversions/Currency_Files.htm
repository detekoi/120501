Currency Files

esi.manage allows for multiple time variant Currency Exchange Rates which are handled through Currency Files.

 

Currency File Parameters

  1. Abbreviation - The unique abbreviation for the currency file
  2. Description - The Full Description/Name for the Currency Exchange File
  3. Base Currency - The Base Currency that all the currencies in the Currency Conversion table are converted to.
  4. Active - Sets which Currency File is used for Revenue calculations
