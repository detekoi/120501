Currency Conversions

Monetary Amounts (Expenditures, OPEX, Revenue) are entered with a Currency Unit as appropriate, i.e., Capital can be entered in thousands of dollars, and OPEX in dollars. Reports, Analysis Cases and Ranking can display the results in any of the configured Currency Units. This requires Conversions to be supplied between the various Currency Units.

 

To Add A Currency Conversion

  1. Once a Currency Unit has been configured it will be added as a column in the Currency conversion table.
  2. The time variant table is all based off a single base currency as defined in the Currency File.
  3. Select the cell that you want to edit and enter the appropriate number.
