Alberta Royalties Custom Asset Attributes.

Specific Custom Asset Attributes need to be configured to provide the Calculator with data not native to esi.manage. The calculator configures asset attributes based on the Attribute Type which has to be set specifically for each field. The names of the fields do not need to be specific, however recommended names are below:

  1. In configurations, expand Budget/Project>Custom Attributes
  2. Select "Configure Custom Attributes"
  3. Create the following attributes:
