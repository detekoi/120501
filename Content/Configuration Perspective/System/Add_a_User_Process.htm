Process for Adding a new User

Adding Users is an administrative process. There are three steps to this, each step is found on a different page in the Configuration:

  1. Ensure that a Role exists that has the correct level of permissions you want the new user to have.
    • It's a good idea to establish a number of roles with varying permissions when the program is first installed.
  2. Add the User.
    • Enter their login ID and optionally their Full Name.
    • Assign the appropriate Role to the user.
  3. Set Project Manager Access.
    • Either set which Project Managers the new user will have access to (Read, Write or None).
    • Alternatively, if the new user is a project manager, set which existing users have access to their project(s).
