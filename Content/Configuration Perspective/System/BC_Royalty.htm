British Columbia Royalty Framework

The British Columbia Royalty Framework allows users to calculate the royalty rates for wells that fall within the Fiscal Model of British Columbia.  The Royalty Framework consists of several configuration pages, allowing defaults to be set at the Global , Fiscal Model  or Asset level . Note, that there are unique fields on all levels and inputs may be required for all pages to see the desired result.  The calculation will inherit the values from the lowest level they are available: if Asset-level inputs have been created, it will over-ride the inputs at the Fiscal Model or Global level.

To view the calculated royalty payable for each product of a project or group of projects, run an Analysis Case .
