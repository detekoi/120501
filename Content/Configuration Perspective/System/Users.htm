Users

A User is identified first by their login credentials (if Windows Credentials are being used within SQL server to enable access to the esi.manage database). A Full Name can be provided to better identify a particular User. In addition to the Permissions (Role) that each user will be associated with, the User Name is used to identify ownership of Reports, Project Set, Export Jobs and Analysis Cases (see Navigator Folders & Workflow).

New users are added through this table.

 

To Add a User

  1. Go to the last row in the table (it will be empty).
  2. Type in a name in the User Name column - this must match their windows login.
  3. Select the appropriate Role for the user.
  4. Click outside the row to save the change.

 

To Delete a User

If a User has Projects, Reports, Project Sets etc., re-assign them to another user prior to deleting the User.

  1. Select the user you want to delete so it becomes highlighted.
  2. Right click the user and select "Delete User".
  3. When prompted, click "OK" to confirm.
