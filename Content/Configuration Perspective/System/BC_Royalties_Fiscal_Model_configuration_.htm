BC Royalties Fiscal Model configuration

To adjust the defaults of the British Columbia Royalties Framework they can be configured in the Fiscal Model section of Fiscal Model. Navigate to the British Columbia Fiscal Model and to the Royalties tab:

 

Once the British Columbia Royalty Framework configuration page has assigned the Fiscal Model hierarchy, the Royalty Provider pick-list will have the option of British Columbia:

 

Defaults can be selected from the third sub-tab of the Royalties page labelled BC Royalties:

 

This tab controls the fiscal model defaults on how the royalties will be applied.  If there are no asset-level defaults applied to a project in the British Columbia fiscal model, the settings on this page will be applied. The options are below

PCOS Facility – pick-list created from the PCOS list on the Global configuration page

Conservation Gas – apply, do not apply, or null

Gas Cost Allowance – CDN $ / unit

Deep Gas Royalty Credit – the default area for the gas credit program

Deep Gas Re-entry Royalty Credit - the default area for the deep gas re-entry credit program

Heavy Oil – apply, do not apply, or null

Posted Minimum Price – Default price forecast for posted minimum

Oil Programs / Deep Royalty Programs – Apply, do not apply or null

Discovery Oil Program, Deep Discovery Royalty Program, Deep Royalty Program, Deep Re-entry Royalty Program – use this program if applicable
