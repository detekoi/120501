Default (Corporate) Economic Metrics

Metrics defined in the Configuration Perspective are termed Corporate Metrics. They are the metrics that will be part of any Analysis Cases that is created, and cannot be edited within the Projects Perspective, i.e.; if a Default DROI has a Discount Rate of 15%, it will always be calculated using 15% regardless of the discount rate in the Analysis Case.

 

New Metrics are added via the button, which will provide a list of available metrics. The Configuration of each Metric varies.

The preconfigured metrics are IRR, Economics Limit and Payout.

Any other metric can be defined as a custom metric. For explanations of the calculation of each of the Metric types (eg. DROI, PIR, etc...) see Economics Calculations.

For metrics calculated in the Spreadsheet Economics, they can be selected as a Custom metric in the formula creator. The Metric Custom Name must match the name defined in the Spreadsheet Economic calculation Workbook.

Production Tax and Local and Federal Tax Rates are Configured by Jurisdiction.

Depreciation is configured with Expenditure Types.
