Capacities

Resource calculations (allocation, over-allocation) are based on Capacity Files.

  • Capacity Files are associated with a Budget - typically the Current Forecast.
  • Individual Capacity Files can be created to represent different business areas.
  • Within a Capacity File, specific Resource Types are selected and then Configured.

 

 

To create a Capacity file:

  1. Begin by selecting a Budget.
  2. Enter a Description for the Capacity file.
  3. Click outside of the row to save the changes.

 

To Configure a Capacity item:

  1. Select the Capacity File you want to attach the Type to.
  2. Go to the last line in the Capacity by Resource Type table.
  3. Select a Resource Type from the drop-down.
  4. Enter a Start and End dates to set the available/valid time period.
  5. Enter the Capacity of the selected resource for the time period.

    6. Note: some resources, such as Completion Crews are not identified by name, and as such the Capacity will represent how many crews are available.

  6. Enter the Unit Cost and select a Currency and periodicity.
    • If you are tracking a contractor as a resource for example, enter their daily cost and select the currency and period as daily; or a salaried employee, their annual salary - desired currency - and period as Annually.
  7. The Daily rate is calculated automatically.

 

The Resource Cost information is used in specific Reports. Resources are also an element that can be configured as part of a Currency Escalation file.
