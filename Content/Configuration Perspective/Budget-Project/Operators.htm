Asset Operators

The Asset Operator is an optional Field for both Assets and Expenditures. In addition to capturing information about a particular Asset, the corresponding colour coding affects how the Gantt displays the "planned" rollup bar.

Each Operator entry is made up of 4 data elements:

  • Name - up to 50 characters; these are the entries that appear in the Operators drop down list.
  • Default - which of the Operators will be the Default for any new Asset or Expenditure created.
  • Background -  within the Gantt, the cells behind any Asset operated by the given operator will be the selected colour. as will the corresponding Planned Bar.
  • Foreground - within the Gantt, text overtop the "Planned" bar will change.

 

 

Table Options

 

To Add An Operator

  1. Scroll to the last line in the table (it will be empty).
  2. Enter the Operator Name (up to 50 characters).
  3. If appropriate, choose a Foreground and Background colour.

 

Changing an Operator

Once an Operator has been used within a Project, you can:

  • Change the Name that will appear
  • Change the Background or Foreground colours.

Deleting an Operator that is in use is not allowed.
