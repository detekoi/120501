Project Status

Project Status is found in the Overview portion of the Project Details page. This is an Optional entry, however, it can be a powerful field to manage Projects within an Evergreen Forecast. The Project Browser can Filter based on Project Status, allowing easy filtering of Projects which are part of the current year's Plan, but have not yet begun. In addition, Reports can Group By Project Status, separating activities that are In Progress from those that are Completed or Not Started.

The Entries in this table should be configured to match your company's business process.

To Add A Project Status

  1. Scroll to the last line in the table (it will be empty).
  2. Enter the Project Status Description (up to 50 characters).
  3. Click outside the line to save the change.

 

Changing A Project Status

Once a Project Status has been used within a Project, there is only one option for changes:

  • Rename the status Description

 

Deleting a Project Status that is in use is not allowed.
