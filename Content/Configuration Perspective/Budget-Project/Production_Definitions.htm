Production Definitions

Production Calculations can be performed using any of the Production Definitions shown here.

  • The Production Definitions table identifies how the various Modifiers are applied for each of the Production Definitions. These settings cannot be modified.
  • The Descriptions, which are displayed as choices in Reports and Analysis views, can be renamed to match your company's terminology.
  • The "Visible" check determines if the definition will appear in the drop down list.

You can change the Description of a Production Definition to match your company's terminology (or remove it from the various calculation screens by unchecking the Visible flag), but you cannot add or remove items from the table, or change the settings on a Description.

Note: Changes to the "visible" flag requires that esi.manage be restarted to take effect.
