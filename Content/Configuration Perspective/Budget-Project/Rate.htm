Rates

Production Rate Units are used as part of Forecasting a Production segment, the Abbreviations will appear in the drop down list to specify the Rate that corresponds to the IP.

The Description is the label that will appear when reporting Production as a Daily Rate.

 

Table Options

 

To Add A Production Rate

  1. Move to the last line in the table (it will be blank).
  2. Enter the Abbreviation (up to 8 characters).
  3. Enter the Description (up to 50 characters)
  4. Select a Volume Unit for the Rate (this will be populated with the list of Production Units).
  5. Select the corresponding Time Period.
  6. Click outside the line to save the change.
