Expenditure Groups

Expenditure Groups factor into Depreciation, allowing for multiple Expenditure Types to be depreciated by the same formula.  

 

Table Options

 

To Add An Expenditure Group

  1. Scroll to the last line in the table (it will be empty).
  2. Enter the Expenditure Group Abbreviation (up to 8 characters).
  3. Enter the Expenditure Group Name (up to 50 characters).
  4. Click outside the line to save the change.

 

To Add An Expenditure Type to An Expenditure Group

  1. Scroll to the last line in the table (it will be empty).
  2. Select the Expenditure Type from the drop-down box.
  3. Click outside the line to save the change.

Note: An Expenditure Type can belong to multiple Expenditure Groups but can only exist once per Expenditure Group.

 

Changing An Expenditure Group

Once an Expenditure Group has been selected for an Expenditure Type, there is only one option for changes:

  • Rename the Abbreviation or Name

Deleting an Expenditure Group that is in use is not allowed.
