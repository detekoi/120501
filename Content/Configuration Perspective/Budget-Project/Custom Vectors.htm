Custom Vectors

A custom vector is an attribute instance which is represented as an association list of key/value pairs.They describe fixed labels (keys) of a specifive Custom Vector attribute.

  • Type - describes the fixed lables (keys) of a specific Custom Vector attribute.
  • Label - the name of a single key in a Custom Vector Type.
  • Attribute - is a custom attribute, which uses a set of labels (keys) described by its Custom Vector Type.
  • Attribute Instance - synonymous with Custom Vector.
