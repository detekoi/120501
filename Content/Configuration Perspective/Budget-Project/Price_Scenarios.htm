Price Scenarios

Price Scenarios make it very easy to manage price uncertainty.

 

  • Each price forecast and price profile will use the scenarios configured in this screen to allow the user to upload a price scenario, for example Brent may be the base forecast but the user may decide to create a Low, medium or high scenario for each of their price forecasts and profiles so that the economic effect of different price scenarios can be analyzed.
  •  Royalties are also captured, if required against each price scenario.

The Hidden flag is used to filter which Scenario(s) will be displayed in the Price Details. This is provided to simplify the creation and updating of the Price Forecasts. This flag does not remove the Price Scenario from the drop down lists.
