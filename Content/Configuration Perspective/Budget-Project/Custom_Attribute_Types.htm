Custom Attribute Types

There are three types of custom attributes that can be configured for four separate areas of the application.

  1. Budgets / Project Sets
  2. Project
  3. Asset

 

  • To Create a new Custom Attribute, click onto one of the custom attribute folders (Budget / Project Set, Project or Asset)
  • Click on the Add - Custom Attribute button on the middle of the screen.

The Entity Type: (Budget, Project or Asset) will be pre-populated, and you will be prompted to select an Attribute Type (Currency, Number, Percent, Date, Boolean, Text, Hyperlink, URL or Picklist) and enter a Name.

Click OK to create the Custom Attribute.

 

Attribute Types

  • Currency - a money field, with a maximum of 4 decimal places, positive and negative values allowed.
  • Number - a numeric field, with a maximum of 30 decimal places, positive and negative values allowed.
  • Percent - a numeric field entered as  a number (.15 = 15%) or a percent (20% = 20%)
  • Date - date entry will accept any entry that is allowed within the calendar control or select from the drop down calendar.
  • Boolean - a checkbox field that stores true/false information only.
  • Text - a free text field  with unlimited space.
  • Picklist - a drop down list created by the administrator in the picklist screen.
  • Hyperlink - a URL to a location on the file system or a web based URL. Can be used to attach files to a project, or call a web service interface to another system from within esi.manage.A hyperlink is made up of two parts, the URL (web or file system address) and the label. Hyperlinks have 3 display modes: Display Link , Edit Hyperlink URL and Edit Hyperlink Label. It’s possible to switch between the display modes by clicking the icon next to the hyperlink.

    When a hyperlink custom attribute is displayed within the projects table in the Project List view or the Asset Table in the Project Details view you can click on the link while holding the Alt key to follow it. Note that for the hyperlink to show up in the corresponding table the Display in Tables option must be selected.
    Hyperlink in Excel will show custom attribute label, tooltip with message and hyperlink. If label is not given, it will show URL itself.
  • Multi-Select Picklist - Allows you to pick multiple options from a picklist.
  • Custom Vectors - Allows you to store distributions as custom attributes as a single entity.
  • Time Series - A relative attribute to the Asset or Project that will shift in time appropriately.
  • Absolute Time Series - A fixed in time attribute that does not shift.

 

Configure Attributes Properties

Certain attribute properties (such as Rankable) will only appear for specific Attribute Types

  • Read Only - a flag to make the instances of the Attribute within the Project non-modifiable.
  • Enabled - a mechanism to hide an existing attribute from new Projects.
  • Category - a grouping mechanism which is used the Custom Attributes page of the Project.
    • The Category Name must be entered for each Attribute that belongs to a particular Group.
  • Show Comment - instructs a comment dialogue to be provided with the Attribute
  • Rankable - indicates that the particular Attribute value should be available as a Ranking Metric. This is typically used when Economic Metrics (NPV, DROI, etc.) are brought into an esi.manage Project or Asset from an external Economics package, and you wish to use those values as part of Project Ranking.
  • Display in Tables - allows the specified custom attribute to be included as a sortable column in the Project List
  • Filterable / Groupable - allows the specified custom attribute to be included as a filterable and groupable item within the Reports perspective.
  • Decimals - how many decimals to display for numeric Attributes.
  • Default Value - if you wish to provide a default value for all new instances of this attribute.
  • Date is Relative - A flag for both Project and Asset attributes, if checked the date will be persisted as relative to the project. If custom date is relative, its value cannot precede project/asset start date. If the relative date is changed, all values for the attribute will be converted from relative to absolute value and vice versa.
  • Series Values Aggregation Method : For Time Series and Time Series Relative To Project/Asset Start Date:
    • Non Summable - Values in series cannot be summed.
    • Sum - Values in series are added together for the period.
    • Average - Values in series are added together and the divided by the number of entries in the period.
