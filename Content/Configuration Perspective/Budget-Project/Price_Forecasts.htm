Adding a Price Forecast

To add a new Price Forecast, simply press one of the "Add a forecast" buttons.

esi.manage will display the following dialogue:

There are three different fields that are required to create a new Price Forecast. (With one additional optional field)

  1. Forecast Name - The Forecast name has specific rules for uniqueness.  If a name exists then it cannot be used at any higher or lower level (i.e. if at a jurisdictional level a price forecast exists, then that name cannot be used at the global or project level).  If a profile name is used at the jurisdictional or project level, it cannot be used within the same jurisdiction or project again, but it can be used in a different jurisdiction or project.
  2. Production Unit - You must specify the Product that the price profile deals with.  This will filter out any forecasts which have been created for a different product to avoid confusion.
  3. Currency - The profile also specifies the currency that is applied to the price forecasts.  This will filter out any forecasts which have been created for a different currency to avoid confusion.

You can also add a comment to the profile to add more details to the profile.
