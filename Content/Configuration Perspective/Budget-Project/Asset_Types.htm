Asset Types

The Asset Type is an optional entry for an Asset, allowing classification of the particular entry. The entries in this table should reflect your company's business process.

Table Options

 

To Add An Asset Type

  1. Scroll to the last line in the table (it will be empty).
  2. Enter the Asset Type Abbreviation (up to 8 characters), this must be unique within the table.
  3. Enter the Description (up to 50 characters).
  4. Click outside the line to save the change.

 

Changing An Asset Type

Once an Asset Type has been used within a Project, there is only one option for changes:

  • Rename the Description

Deleting an Asset Type that is in use is not allowed.
