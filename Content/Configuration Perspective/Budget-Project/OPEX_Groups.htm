OPEX Groups

OPEX Groups are used to facilitate Net Profit Royalty Tax Modifiers. OPEX Types can be assigned to multiple OPEX Groups if applicable, but can only exist once per OPEX Group.

 

 

Table Options

 

To Add An OPEX Group

  1. Scroll to the last line in the table (it will be empty).
  2. Enter the OPEX Group Abbreviation (up to 8 characters).
  3. Enter the OPEX Group Name (up to 50 characters).
  4. Click outside the line to save the change.
