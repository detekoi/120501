Expenditure Families

The Expenditure Families page is where Expenditure Families are set up for depreciation and DD&A purposes.  By default there is only a Tax Family, but more families can be added as needed.  DD&A is a line item in the  Income Statement and depreciation is a part of the cashflow statement.

esi.manage currently only uses the Tax family for Depreciation and DD&A calculations.

Other Expenditure Families can be set up for Concession Groups, or any other Depreciation Families the User wants configured.
