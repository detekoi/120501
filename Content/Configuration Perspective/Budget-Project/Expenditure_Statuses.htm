Expenditure Statuses

Expenditure Status is an Optional Field for an Expenditure Line, serving two purposes:

  1. Filtering - Reports can be Filtered to specific Expenditure Status'. Expenditure Status is also a Filter option within the Project Browser.
  2. Tracking - as an Activity moves through it's various phases (from Proposed to Completed), updating the Expenditure Status allows the Expenditure to be appropriately reported on (if Filtering is being used in a Report).

The entries in this table should reflect your company's business process. As the Drop Down in the Expenditure table sorts the entries alphabetically, consider using a numerical Prefix to indicate the appropriate progression of an Expenditure.

To Add An Expenditure Status

  1. Scroll to the last line in the table (it will be empty).
  2. Enter the Expenditure Status Description (up to 50 characters).
  3. Click outside the line to save the change.

 

Changing An Expenditure Status

Once an Expenditure Status has been used within a Project, there is only one option for changes:

  • Rename the status Description

Deleting an Expenditure Status that is in use is not allowed.
