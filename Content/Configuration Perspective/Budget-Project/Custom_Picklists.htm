Custom Picklists

To Create a new Custom Picklist, click on the Add - Custom Picklist button on the middle of the screen.

  1. You will be prompted to enter a Name. The PickList name is what will appear when configuring a Custom Attribute that is a PickList.
  2. Click OK to create the picklist.
  3. Supply the necessary information for the Pick List entries:
    • Sort Order controls how the entries will appear in the list (when be selected within a Project). This will bring the most common selections earlier in the list.
    • The Enabled flag allows pick list entries to be 'retired' - they are still in the list (to support older projects where the entry was selected), but prevents the items selection in the future.

 

The Custom Picklist(s) you create will be available as a selection from the Custom Attribute screen.

  1. Create a new Custom Attribute with a Picklist attribute type.
  2. Choose the Picklist you require from the Pick list properties. You can also choose a default selection from the list.

 

The Custom Attribute will appear on the screen you select, in this case it will appear on the Project Attributes screen inside the project. If the Entity Type was Asset, it would appear on the bottom of the Asset page of a project when an Asset is selected. If Budget was chosen, it would appear on the Budget page in Configuration, when a Budget is selected.
