AFE Statuses

AFE Status allows you to represent the progression of a particular AFE from proposal through to completion. This is a Configurable table; you can enter whatever Status' are appropriate to match your company's process. While AFE Status is an optional field for any AFE, it does facilitate the Filtered to, or Grouped By specific AFE Status within Reports.

Table Options

 

To Add an AFE Status

  1. Scroll to the last line in the table (it will be empty).
  2. Enter an Abbreviation (up to 8 characters) - it must be unique amongst all AFE Status Abbreviations.
  3. Enter the Description (up to 50 characters).
  4. Click outside the line to save the change.

 

Changing an AFE Status

Once an AFE Status has been used within an AFE, there is only one option for changes:

  • Rename the Abbreviation or Description.

Deleting an AFE Status that is in use is not allowed.
