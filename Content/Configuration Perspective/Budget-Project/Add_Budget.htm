Create new Budget

The first step to entering data about a given well or project is to assign it to a budget. A budget is a large-scale overview of the key information regarding all of the projects contained within it.

 

To create a new budget:

  1. Open the Configuration Perspective by selecting the button in the top right.
  2. A new hierarchy will appear on the left - expand the top option (Budget/Projects) by pressing the [+] beside it.
  3. Expand the Budget option below that the same way [+]
  4. Click on the "Budget" option below that - this time a single click on the text will work, no [+] sign to worry about.
  5. A new view, "Budget" will appear in the right hand pane.

 

 

You will see a screen similar to the following:

  1. Use the empty row at the bottom of the Budgets table.
  2. Enter a name for the budget. Since the budget is never accessed by users, it is common practice to give it a name that consists only of punctuation such as “\\\”. This makes it easy to spot in the budget table and emphasizes that this budget should not be used by users.
  3. Fill in the other necessary fields. Since the users will not be accessing this budget, uncheck the Visible box. Do not check the Template Projects Only check box.
  4. Save your work.
  5. The same can be achieved by pressing this button in the toolbar.


     
  6. A warning will appear if you fail to enter all the necessary fields, and list the ones that need to be filled in. If this happens, choose "Correct" to make the change, or "Discard" to delete the whole row and start over.
