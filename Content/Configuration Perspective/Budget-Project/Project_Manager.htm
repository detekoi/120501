Project Manager

A Project Manager is required selection for every Project. Project Manager can be used for Filtering and Grouping - the Project Browser and Reports can be Filtered to, or Grouped By a specific Project Manager.

  • In addition, Project Managers are mapped to User ID's, which can be used to control Access to Project information.
  • All Project Managers can be AFE Managers, however you can set a Project Manager to be AFE only.
    • In this case, the Name will appear in the drop down list for AFE Managers, but not in the drop down list for Project Managers.
  • The Active Flag is a mechanism for removing a Project Manager from the drop down lists, but not from any historical Projects.

 

Table Options

 

To Add A Project Manager

  1. Go to the last line in the table (it will be empty).
  2. Add a Name for the Project Manager - this is what will appear in any list.
  3. Make the Manager Active.
  4. If they are to be AFE Managers only, check this box.
  5. Select the User to map the Project Manager to the user ID.
  6. Click outside the line to save the change.

 

To Remove An Item from the List:

  1. Single click an item so that it becomes highlighted.
  2. Right click to bring up the menu.
  3. Select "Delete Project Manager" from the menu.
  4. When prompted, click "OK" to confirm the deletion.
  5. You can also use the "delete button" on the context menu.
