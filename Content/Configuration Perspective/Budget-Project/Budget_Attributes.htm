Budget Attributes

Budget attributes are a type of custom attribute. Once configured, budget attributes will appear in the lower half of the budget configuration screen for non-read only Budgets.

Budget attributes are also the same as Project Set attributes so the attributes will also show up in a project set.

The screen shot below shows where the budget attributes will appear in the budgets configuration page. Note the budget attribute has been configured to allow the user to enter a comment. This can be switched on or off for each attribute during configuration.
