Expenditure Categories

Every Expenditure Line in a Project must have an Expenditure Category. This association enables Filtering and Grouping - Reports can be Filtered to, or Grouped By specific Expenditure Groups.

The appropriate 'number' of Expenditure Categories is dependent on the granularity that your company Reports at or Tracks. Keep in mind that Project Hierarchy (Exploration or Development) can provide a differentiation between Exploration Drilling versus Development Drilling by using the Hierarchy element for Grouping or Filtering.

Each Expenditure Category entry is made up of 2 data elements:

  • Abbreviation - up to 8 characters.
  • Name - up to 50 characters. This is how the Category is represented in the Project, as well as within Reports.

 

Table Options

 

To Add An Expenditure Category

  1. Scroll to the last line in the table (it will be empty).
  2. Enter an Abbreviation (up to 8 characters) - it must be unique amongst all Abbreviations.
  3. Enter the Expenditure Category Name (up to 50 characters).
  4. Click outside the line to save the change.

 

Changing an Expenditure Category

Once an Expenditure Category has been used within a Project, there is only one option for changes:

Rename the Abbreviation or Category Name

Deleting an Expenditure Category that is in use is not allowed.
