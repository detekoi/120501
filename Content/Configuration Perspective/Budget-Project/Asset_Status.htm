Asset Status

Asset Status is an Optional Field for an Asset, serving two purposes:

  •  Filtering - Asset Status is a Filter option within the Project Browser.
  • Tracking - as an Activity moves through it's various phases (from Proposed to Completed), updating the Asset Status works in conjunction with the Filtering described above.

The entries in this table should reflect your company's business process.

 

Table Options

 

To Add An Asset Status

  1. Scroll to the last line in the table (it will be empty).
  2. Enter the Asset Status Abbreviation (up to 8 characters), this must be unique within the table.
  3. Enter the Description (up to 50 characters).
  4. Click outside the line to save the change.

 

Changing An Asset Status

Once an Asset Status has been used within a Project, there is only one option for changes:

  • Rename the status Description

Deleting an Asset Status that is in use is not allowed.
