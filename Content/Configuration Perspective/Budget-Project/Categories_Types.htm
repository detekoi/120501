Categories/Types

Resources can be configured to aid in determining a feasible schedule (based on Resource utilization). Specific Expenditures can be associated with a particular Resource; esi.manage will aggregate across all Projects using a specific Resource within a given period to report on allocation and over-allocation.

Resources are represented by:

  • Categories - this provides organization to the configuration of particular Resources.
  • Types - instances within a particular Resource Category.
    • These are entries that will have their Capacity configured, as well as be associated with a particular Expenditure within a Project.

Reports and the Project Browser can be Filtered based on Resource Categories and Types.

See Resource Capacities

 

Table Options

 

To Create a Resource Category:

  1. Scroll to the last line in the table.
  2. Enter an Abbreviation (up to 8 characters).
  3. Enter a Description for the Category (up to 50 characters).
  4. Click outside the new line to save the changes.

 

To Create a Resource Type:

  1. Every Type must be associated with a Category; begin by single-clicking the Category from the Category table you will be adding the Type to.
  2. The Types table will now become available.
  3. Scroll to the last line in the table.
  4. Enter an Abbreviation (up to 8 characters).
  5. Enter a Description for the Category (up to 50 characters).
  6. Click outside the new line to save the changes.

 

 

Efficiency Modifier:

The Efficiency Modifier is set up through the Configuration Perspective and is applied to a singular resource, this is defaulted at 1.0, this will set the efficiency in which a resource operates. This is an important feature because when used it changes the how fast/efficient the Resource can complete the activity assigned to it.
