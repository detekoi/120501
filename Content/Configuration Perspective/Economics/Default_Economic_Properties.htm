Default Economic Properties

The default Analysis Case Properties define the settings within the Analysis screens. The definitions on these fields can be found in Analysis Case Fields - Properties.

The Properties page is made up of 4 Sections:

  1. Custom Economic Calculations - specify the custom calculation parameter file to be used for the Analysis.
  2. Sensitivities - specify the scenarios for Pricing, Escalation, Currency Exchange, Inflation and Measure Scaling to be used for the Analysis.
  3. Data Truncation - defines the Span (number of years) for the generated Cash Flow, and the Start Date.
  4. Discounting Options - when the Discounting begins, the Discount Rate to apply, whether it is to be Mid Year (only valid with Yearly discounting), and whether to Retroactively Apply Interest prior to the Discount Date.
  5. Data Definition - how esi.manage is to generate the Cash Flow; i.e., Forecast or Blended, Gross, Net, Net Risked, what Price File to use for Revenue Calculations, and whether to apply Escalation.
  6. Display Settings - defines the Granularity (monthly or yearly) of the Cash Flow, and, with Monthly Data, whether the Discounting is to be applied Yearly or Monthly, as well as the Volume unit to be used for reporting. The monthly or yearly discount option also applies not only to the discounting in the analysis case but also updates the monthly or yearly discount setting for metrics and ranking.

For further details, please see the analysis properties page.

 

Locking fields

Each option on the Default Economic Properties page has the option to be "Locked" which will prevent users from changing these fields from within the Analysis Case. The lock can be set selectively - i.e.; only locking Discount Rate and As Of Date.
