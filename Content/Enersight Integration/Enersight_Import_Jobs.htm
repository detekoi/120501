Create a New Enersight Import Job

The Enersight Import Jobs are created through the Navigator, as shown below.

A dialog box will appear that prompts you for a name for the Import Job. As with all other files that are created in the Navigator, you can click Finish to create the Enersight Import Job, or click Next to choose a different folder besides your private folder to save the job.

Once the Enersight Import Job is created, you will be prompted for your Enersight username and password, with the option to save them.

After clicking OK, a new view will appear in esi.manage where you can choose the esi.manage Budget you wish to import into, as well as the Plan Data Set and version you created in Enersight. For more information see also Configuring Enersight Mapping File or Configuring Enersight Import Jobs
