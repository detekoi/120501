Configure and Run Enersight Import Jobs

In an Enersight Import Job, you must set the following three values:

  1. Target Budget: the esi.manage budget that you want to import data into
  2. Plan Data Set: the name of the Plan Data Set you wish to import from
  3. PDS Version: the version of the Plan Data Set you wish to import

Once these three values are populated, all opportunities that were saved in the PDS will appear in the Opportunity Selection window.

 

Selecting Opportunities

To choose which opportunities to import, simply check the box next to the opportunity name. In addition to the select/deselect all buttons, there is the option to multi-select. This is done by multi-selecting the desired opportunities (ctrl+click or shift+click), and pressing the space bar on your keyboard. This can be useful when wanting to only import projects which do not already exist in esi.manage.

There is also the option to alphabetize either column in the list by clicking on the columnheader (ie. "Opportunity" or "Project Exists in esi.manage"), or searching for opportunities which contain a specific string by typing into the "Enersight Opportunity Search" field.

 

Run Enersight Import Job

Once the Enersight Import Job is configured and at least one opportunity has been selected, click the "Run Enersight Import Job" button to import the opportunities into esi.manage.
