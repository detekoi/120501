Enersight Integration

The Enersight Integration allows for users to import saved PDS results from Enersight into esi.manage by using saved Enersight Import Jobs, found in the

Navigator

.

Also required to import from Enersight into esi.manage is the configuration of a mapping file, and Enersight connection information set in the esi.manage preferences. You can get the correct Enersight Service URL from your esi.manage adminstrator, or 3esi-Enersight support.
