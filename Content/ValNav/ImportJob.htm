Import Job

The Import Job can be accessed through the Navigator. Similar to creating reports and analysis cases, Right-Click and select New>ValNav Import Job. To open the Import Job requires the configuration to be correctly setup.

An Import Job Wizard will prompt for a name, target budget, and ValNav plan to be used for the job. These can be changed from within the Import Job post creation.

Projects can then be selected for import, then imported into esi.manage using the button.

The Showing/Total section gives information about what entities are visible and how many are not, based off of the ValNav Entity Search. The first value is the number of visible entities over the total number of entities. The second is the number of visible selected entities over the total number of selected entities. The last is the number of visible entities that exist in esi.manage over the total number of entities that exist in esi.manage.
