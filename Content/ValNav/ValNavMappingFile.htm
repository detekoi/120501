ValNav Mapping File

A sample mapping file can be found in the esi.manage installation files.

3esi\3.23 Package\Sample_Files\ValNav\Mapping.csv

The mapping file is a csv file with 3 separate columns unless indicated otherwise.

  • Column A - Mapping type.
  • Column B - Name of the type in ValNav.
  • Column C - Name of the esi.manage type. Note the type abbreviations set in the configurations have to be used in this column.

Economic metrics and be imported into esi.manage using the EntityResultMapping. In some cases this requires using calculations in JavaScript and should be set up by a 3esi consultant.
