Export.exe - Project Export

esi.manage is delivered with an external "export.exe" program, which is found under the esi.manage.export folder in your installation. Within this folder there are 2 files which you can edit:

 

  • export.ini - which establishes the memory allocated to your export job

 

Export.exe can be invoked from the command line, or can be called using a shell command from another application. The syntax is:

 

>> export.exe -P [PROPERTIES_FILE] {required} -L [PERFORMANCE_LOGGING_FILE] {optional}
