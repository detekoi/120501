XML Export Overview

The esi.manage XML Schema (see Web Service Test Page) is the basis for the Project Export (the Export Job) as well as the various interfaces implemented for the Integration Service.

The Export Job is configured inside of esi.manage. Within the Export Job, you define what Project Data (all 'user supplied' data can be included in the Export), whatever Calculated results (Measure Type, Data Type and Basis, Periodicity and Duration), and the Project Set(s) that should be exported.

The Export Job can be invoked by a user from inside of esi.manage (Run Export Job), externally via the Integration Service, or using the Export.exe application that is part of your esi.manage installation.
