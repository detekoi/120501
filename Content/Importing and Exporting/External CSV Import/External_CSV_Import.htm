External CSV Import

Import.exe (found in the esi.manage.imports.cmd folder of the esi.manage installation) allows OPEX data and Prices data (currently) to be imported against a selected esi.manage database.

The creation of this new application allows the imports to be invoked by an external application, as part of a pre-defined workflow. All of the steps that the internal CSV Import process follows can be invoked as part of the calling parameters.

 

This application is documented - entering:

> import --help

 

will bring up a description of the parameters that can be included, and their options.

A text file can be created which contains the appropriate switch values, and used as part of the calling syntax.
