External Currency Escalation Import File

There are seven different line types for a Currency Exchange File Import.

  1. File - identifies the currency escalation file to be defined
  2. Hierarchy code - identifies the Hierarchy element for following details.  Use the code, not the description.
  3. CAPEX - specifies that following details apply to CAPEX.  Use the CAPEX type code, not description.
  4. OPEX - specifies that following details apply to OPEX
  5. Resource - specifies that following details apply to resource costs
  6. Price - specifies that following details apply to prices
  7. Detail - defines the currency escalation rate for a particular date

Each line type has a different layout.
