External Currency Exchange Import File

There are three different line types for a Currency Exchange File Import.

  1. File - identifies the currency file to be defined
  2. Currency - identifies the currency being defined by following details
  3. Detail - defines the currency conversion rate for a particular date

 

Each line type has a different layout.
