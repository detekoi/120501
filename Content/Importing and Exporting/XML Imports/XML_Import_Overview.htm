XML Import Overview

esi.manage contains a set of Schemas which represent the data that is part of an esi.manage Project. This schema is used for both Exporting all data about a Project(s), as well as for importing specific Project Measure data. In addition, esi.manage contains an Integration Service (developed using Web Service protocols) which enables Project Export and Import to be invoked by an external application, allowing esi.manage data to be part of a defined process.

  • Schema Documentation can be accessed via the Web Service Test Page.
  • An Export Job is configured (and saved) inside of esi.manage. An Export Job can be Run (see Running an Export Job) from inside esi.manage, or invoked externally via either the Web Service interface or the Export application.
  • Project data (Hierarchy or Measures) can be created or updated via the Hierarchy, accessed from inside of esi.manage.
  • Specific Data (CAPEX, AFEs, Production, External Identifiers, Hierarchy and Project Export) can be invoked through the Web Service, either to obtain, create or update a Project's data.

 

XML Special Characters:

There are 5 special characters that are markup sensitive and have special treatment to escape the markup language.  The following is a list of treatments and the five characters that they represent:

  • &amp; → & (ampersand)
  • &lt; → < (less-than sign)
  • &gt; → > (greater-than sign)
  • &quot; → " (quotation mark)
  • &apos; → ' (apostrophe)
