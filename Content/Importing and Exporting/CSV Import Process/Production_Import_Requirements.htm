Note that:

  1. System and Identifier are interpreted together, as are Asset, UWI and Project. Enough information must be supplied in order to uniquely determine a Project / Asset / Product combination to associate the import against.
  2. If there more than 1 Production Segments within a Project that satisfy the described matching, the imports will be associated with the Production Segment that was created first (not the lowest Line #).
  3. If there is no Production Segment which satisfies the matching, then a segment with a 0 Forecast will be automatically created to associate the imports against.
  4. If a Product Quality value is supplied, a Product Quality Unit must be supplied as well.
