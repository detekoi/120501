Finish / Results

When you select "Finish" from the Import Data phase:

  1. A message box open that indicates that esi.manage will need to restart.
  2. Next the import wizard will close, and an internet window will open with "Import Results" (shown below).
  3. Once the Import results screen is open, esi.manage will prompt for a restart. Note that this is necessary to ensure that user who is performing the import has the correct data in all of the opened views. esi.manage will restart automatically.

 

This will list how many records succeeded, how many were skipped, and list the errors.
