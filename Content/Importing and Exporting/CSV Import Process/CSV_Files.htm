CSV Files

The required import format is a comma delimited file, or CSV file. This is an exportable format from Microsoft Excel.

To save a Microsoft Excel Document as a .csv, choose "Save As" and change the file type to "comma delimited" before saving.

 

Import File Guidelines:

Each Import Type (i.e, CAPEX Detail Import) is listed in the Help Files as to its specific Import Requirements.

In the Notepad view, commas are used to determine where the next field begins, if a name or value includes a comma, you must enclose it in quotations. Ex:

Wrong: x, 10,000, y  

Right: x, "10,000", y

 

A header may be included on the first line of the file. This can be used to list which fields are needed.

When including a header, make sure the"First Line is Header" option is checked at the Validate/Import stage. Leave this unchecked if not.
