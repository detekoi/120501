Import Data

This screen performs the import process from the records selected in the Choose Import File stage.

Selecting the "Proceed with valid records only" check box will continue the import process should any errors in the import file be encountered. Leaving this blank will display records with errors and halt with the import.

Select the "Import" button.

Any errors will appear in the "Errors" window, with a description of what the problem is and the offending record.

Once an import file has been validated, the import can be completed by selecting the "Finish" button.
