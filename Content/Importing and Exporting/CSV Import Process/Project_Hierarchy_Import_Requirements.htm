Note: If importing a child to a parent, the child Hierarchy Type must already be a child of the parent Hierarchy Type.  This is done by importing all the Hierarchy Types without any parents.  In esi.manage, create the Parent/Child Relationships between Hierarchy Types as desired.  Import the .csv again, with the Parent code entered.  After import, it is required to designate default Hierarchy elements for each parent, in the esi.manage UI.
