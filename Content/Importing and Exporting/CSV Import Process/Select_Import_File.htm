Select Import File

The import file can be a .txt or .csv extension, although the .txt file must follow the comma-separated-value format.

  1. Begin by clicking the "Browse" button.
  2. A search window will open. Select a valid file with a .txt or .csv file extension.
  3. When the first line in the import file is a header - a list of fields or title for example - check the "First Row is a Header" box.
  4. Click "Validate Import File" to ensure that all data is present, in the correct order, and properly formatted.
  5. {Optional} A list of errors or warnings may appear if there are problems with the import file. An error is a problem which will prevent a file from importing as it either breaks the required format or lacks necessary information. A warning is a problem with the import data - such as a duplicate record existing - which may be problematic but will not stop the import. These will be listed in the "Validation Errors" window, with an explanation of the error and which record caused it.
  6. {Optional} If there are errors during the import, the records can be corrected or the import can bypass the problem records by choosing to "Proceed with valid records only" - by checking this box the import will continue and ignore any records with errors and will still import those with warnings.
  7. Select "Next" to proceed to the next step of the import process.

 

(The Next button will be greyed out until the file has at least one valid record to import.)
