Note That:

  1. Line # is optional, however to use Dependency the Source Expenditure must have a specified Line #.
  2. Type must be Expenditure Type Abbreviation.
  3. POD% will default to 100%.
  4. If Currency is not supplied, the corresponding amounts will be interpreted in Budget Currency Units.
  5. Started and Ended flags cannot be set in the future.
  6. When updating an Existing Header, Asset is not required (provided that the Line Number is provided), however, for a New Header Asset must be supplied.
