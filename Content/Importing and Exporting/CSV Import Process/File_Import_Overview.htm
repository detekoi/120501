File Import Overview

There are 14 different types of file imports that esi.manage supports, via either the internal or external import feature.

  1. External Identifiers Import
  2. Project Hierarchy Import
  3. AFE Header Import
  4. AFE Details Import
  5. CAPEX Header Import
  6. CAPEX Details Import
  7. OPEX Import
  8. Production Import
  9. Price Import
  10. Variable OPEX Import
  11. Currency Exchange Import
  12. Currency Escalation Import
  13. Asset dependency Import
  14. Excel Template Import

All internal imports are accessed at the top of each perspective. Each import has its own specific csv import format or template.

They can be accessed via File>Import.
