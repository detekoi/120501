Bolded fields are mandatory, the import will fail for that record if a mandatory field is missing.

Note that:

  1. The entries for Line Number, Type, Asset, Start Date and End Date are used to first attempt to match an Import record to an existing Expenditure Line Item.
  2. If a match is found, then the existing information in the expenditure will be updated.
  3. If no match is found, then a new Line Item is created.

 

Data Relationship Rules

Rule 1: Started and Ended Flags

  1. The Started and Ended flags cannot be set to a date in the future.
  2. The Start and End Dates, and started and ended flags cannot all be updated in a single import record.

 

Rule 2: Dependant Expenditures

  1. If an expenditure is being made dependant on another expenditure, the source expenditure must have a specified Line Number.
  2. The source expenditure must be earlier in the import file (if both are being created from one import).
  3. The Start Date for the expenditure being made dependant must be later than the End Date for the source expenditure.
