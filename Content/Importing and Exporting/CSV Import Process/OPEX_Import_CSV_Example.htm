Note That:

  1. Unit of Measure (Currency) is Mandatory.
  2. Either Expenditure Type or Expenditure Type Abbreviation is acceptable for the Type.
  3. Product is not mandatory, however you will need to put a 'space' character in the corresponding cell in Excel.
  4. OPEX imports will aggregate - Lines 5 and 8 will add together to create a single entry.
