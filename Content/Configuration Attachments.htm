Configuration Attachments

Configuration attachments allow the user to track multiple Economic Reports, Custom Economic Models and Custom Economic Parameter files.

This section of the defaults is used as a repository that other areas within the application can point at to draw data from.

 

There are three types of documents that can be attached.

1. Custom Economic Parameters

2. Custom Economic Model

3. Economic Report

The attachments are uploaded by the user through the buttons located on the top right of the page.

The user has the ability to Upload, Download, Delete and Refresh.

 

Once downloaded into the esi.manage, users can then make use of these attachments through several areas within the application.

 

Economic Reports

Once an economic report has been attached, the user can move to the Excel Report Templates area within Configuration to reference this attachment. Learn more about the specifics of this area within the Excel Report Templates page in the Help File.

Custom Economic Model & Parameters

The custom economic model and parameters attachments can be found in the fiscal model area of the defaults. Within the Custom tab, the user has the ability to reference the uploaded attachments to allow for custom calculations while using the analysis case functionality within esi.manage.
