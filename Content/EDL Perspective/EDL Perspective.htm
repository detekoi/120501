EDL Perspective

The EDL Perspective houses the Excel Data Loader.  The Excel Data Loader “(EDL)” is a tool used by esi.manage to extract data from MS Excel documents by way of EDL “templates”. A template is a set of mapping rules which tells esi.manage how to map the spreadsheet into a particular format fit for esi.manage.  Once a template has been defined and tested, it can be stored, shared, and used by authorized persons to import multiple Excel files of the appropriate format into esi.manage. The EDL is an efficient way to streamline the data load process. To get started, the EDL perspective needs to be enabled.
