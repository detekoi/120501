User Parameters Tab

The User Parameters Tab allows users to define special variables that are unknown at template creation or are user specific in such a way that the parameters are prompted on import to be entered.  A parameter can be created by clicking the Green Plus icon. The Name is the variable named that is defined. The Description defines what the parameter is.  The Type can be selected as a String, Numeric, Date, or a Boolean value. The EDL will ensure that the value entered obeys this condition. A Default can be defined to populate the text box automatically when prompted. It can be overwritten at import. If Mandatory is selected, an error will prompt if the parameter is left blank. The prompt can be viewed and values entered within the template by pressing the human icon in the top right of the EDL template.

When a parameter is defined it can be used in the EDL in conjunction with the Mappings tab. Once defined, the parameter can be used with the appropriate attribute by selecting the Scalar Type Formula and using =ESI.USERPARAM("User Parameter")for the formula, where User Parameter is the parameter defined.
