Translations Tab

The Translations tab allows the user to define Excel synonyms for esi.manage attributes. For example, if a spreadsheet defines currency thousands of US dollars with the acronym MUSD, the EDL can translate MUSD to the esi.manage default kUSD, which describes thousands of US dollars as well. The attributes that are available in the Translations tab are the attributes that are mapped on the Mappings tab and have a defined list of values in esi.manage configuration. Initially there is only one option per attribute, but as an attribute is filled out, an additional row will appear for that attribute.

The Target column refers to the esi.manage value and the Source column refers to the Excel value.
