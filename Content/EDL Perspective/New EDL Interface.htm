EDL Interface

To start a new template, simply Right-Click in the Navigator pane and select New>EDL Template.

Name the template to be saved in the Private folder in the navigator.

By default, all newly created EDL Templates are stored in the user's private folder.

 

The screen that opens when a template is created is the Mappings tab. There are 4 tabs within an EDL template.

  • Overview: Template details and description
  • Mappings: Where data mappings from Excel to esi.manage live
  • Translations: Translate nomenclature from Excel into esi.manage defined attributes
  • User Parameters: Create parameters to be populated by a user upon import

Along the top right of the EDL there are 5 buttons.

  • Open Workbook: Opens Excel Workbook or brings it to front of applications if already open. This is a copy of the Excel workbook used to create the template – it is not the actual file. The Excel file attached to the template can be altered and saved but will not affect the original document, and vice versa.
  • Upload Workbook: Upload an Excel Workbook to be used as the format template for the EDL template.
  • Import Workbook: Imports current Excel file into esi.manage after mappings are created.
  • Undo a change
  • Redo a change
