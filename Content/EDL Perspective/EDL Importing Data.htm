Importing Data Using a Template

When a template has been created, saved, and successfully imported, it can be used to import other Excel documents that have the same format. The cells must be identical from workbook to workbook for the mapping to correctly import. To import a workbook(s), open esi.manage and select File>Import.

Then the EDL Import needs to be selected, followed by selecting the template to be used.

Then using the green plus icon, Excel workbooks can be added in bulk to be imported.

Successful imports will display success, while errors will display a warning sign. Edits will need to be made to the Excel Workbook or the EDL to allow the problem files to load.

If there were any User Parameters defined in the template, they will be prompted when Finish is pressed. Once they are filled out and OK is pressed, the Excel documents will be loaded into esi.manage.
