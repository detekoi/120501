Conditions

Conditions are used to include or exclude pieces of information that fit a specified criteria. For example, Capex and Fixed Opex can be loaded from the same table by utilizing conditions.

This example is a continuation of the Distinct and Merge example.

Both Capex and Fixed Opex are utilizing the column B to distinguish whether the value should be loaded or not. Using the formula builder, the nodes under the condition will only execute when the column reads its expected value.

Custom formulas can be used to define the logic in a condition. Most Excel formulas work in conjunction with the EDL special variables. See EDL Syntax.
