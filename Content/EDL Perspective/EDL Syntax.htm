EDL Syntax

EDL Specific Functions

=ESI.CELLVALUE(CELL): Used for cell references that are attributes utilizing a formula in a repetition.

=ESI.VAR("Variable Name"): Used to reference a Variable node in mappings.

=ESI.USERPARAM("User Parameter"): Used to reference a User Parameter in mappings.
