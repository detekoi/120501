.EDL Files

.EDL files house all of the mappings, translations, and parameters of a template. They can be exported to share with another user, then imported as a copy that user can utilize. To create an .EDL file, simply Right-Click on the desired template within the Navigator and select Export ‘Template’.

The export will prompt a window to specify which directory to place it in.

To import an .EDL file, right-click on the Private or Public folder within the EDL Templates node and select Import EDL Template.
