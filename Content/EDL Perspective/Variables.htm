Variables

Variable nodes can be used to store information to be used elsewhere in an EDL template. They can store any data type.

It is important to note that a variable must be defined higher in the tree than where it is being used.

Variables once defined in the tree can be renamed, then referenced using ESI.VAR("NAME") (quotations are needed around the variable name).

Loading Capex into a project scaled by a scale value.
