Mappings Tab

The mappings tab is where the data is defined and the data that will be mapped to Excel is identified. To properly populate the mappings tab, an understanding of what is in the Excel Workbook is required.

Mappings

There are three sections to this tab. The left most area is where attributes are added to be mapped. The attributes are arranged in a tree format where attributes lower down the tree are linked to the branches above it.

Attributes can be added using the green plus icon, or right-clicking and selecting the same icon.

Attributes can be moved around within the tree using the compass icon, or clicking and dragging attributes around.

Attributes can be sorted alphabetically using the sort icon.

Attributes can be removed using the red x icon.

 

There are different icons for different types of attributes and tree commands which are as follows:

: Top level object and sub-objects. Objects will contain nested attributes, sub-objects, and arrays. Nesting gives the input data-structure a tree shape. These attributes are not directly defined.

: Array of objects attribute. Arrays can define a repeat structure to load numerous values into a single attribute. Houses key and regular attributes.

: Command. Defines the type of template that is being created.

: Attribute. Where values can be assigned to.

: Condition. Attributes nested below a condition must meet the condition to be loaded.

: Repeat. Creates a repeating pattern for the attributes that are nested below.

: Distinct. Creates a relationship for nested attributes to roll up into a single attribute.

: Merge. Used with Distinct to merge data into the single attribute.

: Variable. Creates a variable that can be used and referenced by other attributes.

Properties

The properties section defines the rules in which to load data. Information can be loaded through a direct cell reference, a default value, or utilizing a formula. Different nodes in the mapping tree will have different properties available to them.

When using the Cell Default Value Scalar Type, the Copy cell address from excel icon can be used. There is a shortcut for this action by clicking CTRL+F1.

When using the Formula Scalar Type, the expression builder can be used by selecting the Edit Formula icon.

Preview

The preview will display the values that are being read from the properties. It is a good place to revise the template and ensure it is working as intended.

Each attribute has been limited to display only the first 10 iterations it finds. This was implemented to increase the performance of the preview. Therefore, only the first 10 projects are displayed, only the first 10 values of a time series are displayed, only the first 10 dates of a series are displayed, etc. This is annotated by a yellow warning sign over the truncated attribute's icon.

Errors will be displayed over an incident attribute with a red icon. Hovering over the attribute will display the error.
