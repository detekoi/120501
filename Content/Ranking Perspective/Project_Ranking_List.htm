Running Project Ranking

Once the a Ranking file has been selected or modified (or a new Ranking File configured), the calculations can be invoked by selecting the Rank button. The Projects being Ranked will be those that are in the Projects List.

  1. Each Project will be evaluated for each Metric. The table can be sorted by Metric or by Score (ascending / descending).
  2. The Summary flag determines if the Project will be included in the calculation of the values found in the Summary table. See Summary of Ranked Projects.
  3. The results can be colour coded - see the "Colour Scheme" topic.

The ability to copy the Project and Score column can be found in the right corner of the Ranking Results.

This will copy the scores to a clipboard. This function works directly with esi.schedule, by allowing the score from specific projects to be used while creating an optimal schedule.

When the selection of Projects has been determined which best meet the ranking objectives, a Project Set can be created that can be used for further work.
