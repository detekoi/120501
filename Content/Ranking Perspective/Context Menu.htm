Context Menu

 

The Context Menu on the Ranking Results table (shown above) provides access to a number of features:

  • Select, Select All and Select None - this option toggles individual Projects in or out of the Summary, or the list that will be used to create a New Project Set.
  • Export to Excel - sends the Table Results to an Excel worksheet.
  • Rank - easy button for ranking the project list without clicking on the 'Rank' button found in the top right corner of this view.
  • Create New Project Set - upon creating a project list that fits all criteria, user can choose to create a project set through this right click option.
  • Copy Score to Clipboard - this feature allows the user to copy the score to the specific projects within esi.manage and transfer them with a click of a button to esi.schedule for schedule optimization.
  • Colour Scheme - apply one of the available Colour Schemes to the Results - simplifying the Analysis of each Project's relative merits.

 

 

The colour scheme can be selected to help visualize the portfolio. To the right of the "Ranking Results" heading, there is a small downward triangle. Click this to open the context menu, then select Colour Scheme. There are two colour schemes to help organize the ranking output:

Interval (Average)

  1. Determine max and min of the data set.
  2. Calculate the size of each of the 4 intervals: (MAX-MIN)/4.
  3. First interval low range is min, upper range is Low Range + Interval Size
  4. Continue adding Interval Size until there are 4 ranges. The upper value of the last range should equal the MAX value.

 

Quartile (Median)

  1. Determine median (middle) value of the data set (the two middle values/2 if an even number of values), after ordering the data set.
  2. Determine lower quartile: middle value of the lower half of the data set (the two middle values/2 if an even number of values)
  3. Determine upper quartile: middle value of the upper half of the data set (the two middle values/2 if an even number of values)
  4. First colour code segment is min value to lower quartile; second is lower quartile to Median, etc.

To turn the Colour Scheme back off, select "No Colours".
