Copy Selected Projects to a new Budget based on Ranking Results

Once Projects have been ranked and selected in the Summary column, and the selection is ready for further analysis, the Projects can be copied to a new budget.

A prompt will appear to select the destination Budget, similar to Copying Selected Projects to a new Budget.

The Bulk Edit functionality can then be applied to the Projects in the new Budget, for example, updating all of the Projects to Status = Budgeted.
