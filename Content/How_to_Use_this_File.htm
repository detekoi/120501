Layout

This file is broken down into several sections to help explain the esi.manageTM solution.

Navigating esi.manage provides an overview of the program organization and general workflow.

The Project, Report, Ranking and Analysis Perspectives cover the main areas of Program Functionality. The Project Editor works through the various pages of a Project.

This is designed to cover all the user tasks that will occur on a day-to-day basis. This also features a breakdown of the pages in the Perspectives, what they do and how to use them.

The "Configuration Perspective" and "Importing" sections cover a brief walk through of the advanced setup required by the system administrator prior to roll-out in the company, to perform maintenance tasks, as well as has an in-depth explanation of each of the configuration pages.
