Evaluation Files

Evaluation Files are created through the Navigator and can be used in several areas of the application.

The files are based on the Analysis information found when creating a new Analysis case and can be set as the Global defaults within the Evaluation files criteria.

These files allow the user to group selected parameters together as global settings for ease of use.

To create a sensitivity file, right click in the Navigator and select New- Evaluation File.

 

 

Give your new File a proper name and select Next.

The Evaluation File consists of 4 separate parameters.

  1. Custom Economic Calculations- in this section you can select the parameters spreadsheet to be used with the spreadsheet economics engine. It may be left blank if spreadsheet economics are not being used.
  2. Data Truncation - defines the Span (number of years) of years to use for the generated Cash Flow, and the Start Date (explicit Point Forward Date or Full Cycle of the Projects). This section also allows the User to define the Economic Truncation method (either Calculated, Asset Economic Limit Date, or No Truncation), whether to use custom user-defined truncation formula (if Calculated), and whether or not there is any Income Loss Carry Forward for tax purposes.
  3. Discounting Options - User defines when the start date for economic discounting; the Discount Rate to apply; whether it is to be Mid Year (only valid with Yearly discounting), and whether to Retroactively apply discounting to cashflows that occur prior to the "Economic Discounting as of" date.
  4. Display Settings - User defines: date of which the real currency is being calculated on.

 

For more information on each type of file associated with the Evaluation Files please see their specific areas within Analysis Cases

Once the parameters have been filled the user will see their Sensitivity Files within the Navigator in the Private folder.

 

Global Parameters

The newly created Evaluation Files are applied within several areas of the application for economic evaluations.

The Global Parameter found below the windows tool bar allows the user to set sensitivity files and evaluation files that will be used for all economic runs within esi.manage.

Note:

By setting the Global Parameters with these files, the user will overwrite all uses of other Sensitivity or Evaluation files that exist previously in Analysis Cases or where these files may be in use.
