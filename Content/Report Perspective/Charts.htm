Charts

In addition to Reports, most reportable data can be displayed as a chart. There are toggles to control the display - when you select "View Report" you can view either the Chart & Report, just the Report, or just the Chart.

 

Copy Charts to Windows Explorer

You can drag and drop a chart on to your desktop or a Windows folder.

You will be prompted to move/copy the file. Click Yes to copy the file.

The file is saved as custom#.png.

This image file can be used like any other image and inserted into another document or emailed to management for review.

 

Copy Charts into Microsoft Word

You can drag and drop the chart directly into a word document and it will appear as an image.
