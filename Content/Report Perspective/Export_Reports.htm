Export Reports

Exporting is the means of getting data from esi.manageTM to an external format. Currently, reports can be exported into HTML format, Adobe .pdf, Comma Separated Value format (.csv) and Excel spreadsheets (.xls or .xlsx).  The report itself does not need to be viewed in order for the export to work, however the necessary parameters do need to be entered beforehand.

Note: When exporting to excel, the image associated will be embedded within the spreadsheet

To Export a Report:

  1. Open a Report and define the parameters so that it can be viewed.
  2. Click the "Export Report" button.
  3. You will now be prompted to enter a file name and select a file type.
  4. Html is the default type, and the name of the report in the program is the default name, rename the report accordingly and select the desired output format.
  5. Press "OK".
  6. Navigate to the folder the file was saved to and double-click it to open it and view the exported report.

 

Special Consideration when Exporting a Report:

Each type of Report Export file has a purpose that it is suited to.  HTML Exports are portable for systems such as dashboards, or web-pages, PDF files can be safely shared with external partners and CSV or Excel files can be used for further analysis or reporting.  Using the appropriate export file type will save time and effort.
