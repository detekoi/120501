View Reports

 

To View a Report:

  1. Select a report off the Report Selector
  2. Define the parameters in the "Report Parameters" window
  3. Press the "View" button on the toolbar.
  4. The report will generate in the window below. It may take a while depending on computer speed and the amount of data contained - the "View" button will remain greyed out to indicate the report is being generated. The first run of a report always takes the longest. The system defaults to generating a report followed by its associated graph (if it has one).
