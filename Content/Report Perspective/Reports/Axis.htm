Axis

 

The Axis screen lets you configure the basic options for each axis. If the "Visible" box is un-checked, the axis line will not appear.

The "Type" drop-down determines how the axis is labelled.

The "Colour" column lets you select what colour the axis text will be displayed in.

The X and Y axis sub-headings allow you to configure these options, as well as define if a grid will be generated, and adjust the axis labels.
