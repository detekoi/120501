AFE Accruals Report Standard Parameters

These are suggested values for the required Report Parameters to run a basic Sales Report.

Any of these values can be customized to generate a more specific, personalized report. The values listed here were used to generate the "Example" report.

If a parameter is not covered (ex: Project & Report filters), it is assumed you will use the default values.

Parameters marked with ** ** will vary based on the data set available.

 

Budget(s)

Budget: **2013 Initial Plan**

Currency Conversions: Default

Escalation File: (no escalation)

Measure Scale File: (no scaling)

 

Dates

To Date: **Dec 31, 2014**

 

Units of Measure:

Currency: Canadian Dollars (Thousands)
