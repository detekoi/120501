Type Options

The following are options available for the various charts:

  • Dimension: Choose between 2D, 2D with Depth, or 3D for enhanced visual effects.
  • Multiple Y Axis: Select if you wish to plot a second value opposite the primary Y axis.
  • Orientation: Reverse the X/Y axis. For example - Bar Charts will appear horizontal opposed to vertical.
  • Output Format: Select the picture format to display the chart in. Available options: SVG, BMP, JPG, PNG.
  • Series Type:Currently locked.
