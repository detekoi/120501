Report Filters

The Report Filters section of the Report Parameters allows you to control the type and amount of data available in the Report.  Report Filters are additive so as the number of Filters selected increases the resultant set of data will be more focused and generally smaller.

Also, each set of Filters is customized to a specific Report thus the Filters available to be selected changes depending on the Report being generated. Some examples are shown here.
