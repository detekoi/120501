Dashboard Viewing

Custom charts automatically size themselves to the size of the viewing window they are contained in.  4 charts can be shown on a single screen  by arranging 4 charts in a 2X2 matrix, and then running each chart. The result will be 4 charts sized to the windows. You can also perform a similar function to view 2 charts per page, or a single chart on different paper size.
