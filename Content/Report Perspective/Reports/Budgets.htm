General

The general section of the Report Parameters lets you select which Budget (Evergreen Forecast, Archive, Scenario) contains the Projects that you wish to report on. The Budget(s) which is/are selected define the Source for the Projects which are to be reported on. The specific Projects for a Report are identified through the Project Selection parameter, where individual Project Sets are selected.

The additional fields available on the Budget parameter screen will change depending on which Report you are viewing however, some common combinations are shown here.
