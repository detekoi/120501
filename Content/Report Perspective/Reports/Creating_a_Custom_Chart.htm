Creating a Custom Chart

The first step to creating a custom chart is to create a new Report in the Configuration perspective. The existing (canned) reports that ship with the program have pre-defined settings with them, so a new report is necessary to create a blank slate.

 

Create a Chart

  1. Follow the New Report workflow and select Custom Chart as the Report Type.
  2. The Budget parameter will only include Last Actuals and Last Field Estimates date cells, and only needs to be filled in if Blended numbers are desired.

  1. The Dates screen requires a Reporting Period, From and To date.

  1. Please refer to "Data Sets" for information on the Data Set section.
  2. Once all the Report Parameters are showing green , click the "Edit Chart" button.
  3. Refer to the "Edit Chart" section of the Help, and the "Assigning Series" page for details on additional configuration.
  4. After completing the editing phase, click the View Chart button.
