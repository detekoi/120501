OPEX Detail Variance Report Overview

This report shows a breakdown of all Operating Expenses for Projects in one budget and compares them to another budget . It drills down into detail right down to the individual OPEX Type. Individual line items belonging to the same Type are rolled up into a single aggregate line for each Asset. Each item is broken out into three lines:

First Line is the value for the first budget you selected

Second Line is the value for the second budget you selected

ThirdLine (with gray shading) is thevariance between the two.

You can also select the same source budget but change the Production Definition or Production Type to see the difference between Risked and Unrisked, or Forecast and Actual numbers.

This report provides values at the Project, Asset, and Type levels. It gives you a value for each period, as well as an Average over the periods selected and a Total all the periods combined. A Grand Total of each field is also available at the bottom of the report.
