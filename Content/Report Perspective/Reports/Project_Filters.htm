Project Selection

The Project Selection page allows you to identify which Projects are to be Reported on. There are two primary choices:

  • Project Browser - the Projects which are the result of the current Project Browser filters.
  • Project Set(s) - one or more Project Sets can be selected - the Report will be generated against the aggregate list of Projects from those Project Sets (i.e., a Project which is in the result set from two Project Sets will only be reported on once).

 

Note: Project Sets only become available once the Data Set(s) have been configured.  The Project Sets are filtered based on the budget that is selected in the Data Set, and will only show Project Sets which are found in the specified Budget.

 

How Project Sets are Used

Project Sets can be either Static (individually selected Projects) or Dynamic (based on Filtering of Hierarchy or Project Data). In order to report on a Static Project Set, ensure that the Budget for the Project Set (as shown in the screen shot below) is the same as the Budget that has been selected on the Budget(s) parameter page for the Report.

Dynamic Project Sets created against a Budget can be applied against other Budgets in a Report. For example, the Project Set selected below (Canada Projects) is built against the Training Forecast. However, it can be used against any related Budget (i.e., an Archive from the Training Forecast). The Filtering Criteria that is contained in the Project Set is applied against the Budget(s) selected for the Report.

 

 

The Toolbar

The Project Selector has a toolbar is designed to help the user find the Project Set(s) they need.  The description for each button is as follows.

- Create a New Project Set opens the Project Set window where users can create customized lists of Projects.

- Show my private Project Sets hides all public Project Sets so users can focus on the ones they are created for themselves.

 - Show public Project Sets hides the user’s personal files making it easier to browse the corporate templates.

- Expand All Nodes shows all the Project Sets available for use.

- Collapse All Nodes hides all the Project Sets and sub-folders.

- Refresh will load add any new or remove any deleted project sets from your list.

- Populate Project Browser Selection activates list of Projects in the selected Project Sets in the Project Browser.
