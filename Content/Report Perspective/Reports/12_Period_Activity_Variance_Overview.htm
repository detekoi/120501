Activity Variance Report Overview

 

This report shows the variance between what was expected to happen in Budget 1 and what actually happened in Budget 2, based on Expenditures. It breaks out the Capital Expenditures of each project and aggregates them by CAPEX type. Then takes the start/end date of each expenditure and allots the capital amount over the appropriate time period. Each entry has 3 lines:

 

First Line is the value for the first budget you selected

Second Line is the value for the second budget you selected

ThirdLine (with gray shading) is the variance between the two.

 

There is also a total of all the periods combined, and an average. The report includes a grand total that rolls up all the values for each period at the bottom of the report.
