Multiple Series

To add multiple series begin by adding multiple data sets. This is accomplished by filling in more than one row in the Data Set table. Next, open the Edit Chart window.

 

Assign the X-axis series, and the first Y-axis series by dragging and dropping them to the row fields, identified by the symbol. (As shown below).

Next, above the Y Series row line, select the "Value (Y) Series:" drop-down list.

Select "New Series" and a second series will be created. Next, select this series from the drop-down and drag the next data set to this series. (See below, make note of the colour coding).

 

This can be continued for as many series as necessary. Just remember that the series will all be plotted on the same scale.
