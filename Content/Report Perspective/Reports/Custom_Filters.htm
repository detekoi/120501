Custom Filters

Custom Filters can be set in the Edit Chart window. Click the "Filters..." button to open the Filters dialogue.

 

The Filter dialogue uses boolean operators and advanced variants. Double-click each column to enter values.

  1. Expression - select the data set the filter applies to.
  2. Operator - mathematical operator, such as greater than, less than, equal too, percent of... etc.
  3. Value 1 - number that the operator is defined by.
  4. Value 2 - for "between" or "like" operators, the second value that defines the between range.

Additional filters can be entered in rows below the first filter.These are applied in conjunction with the other filters, not independently.
