Data Sets

The Data Sets screen drives the custom charting engine. Here is where you define which criteria you are charting against.

Multiple Data Sets can be defined, and later applied as multiple series on the chart.

The required fields will change by Measure, but "Label", "Measure", and "Source" are required regardless.

  1. Label: Name of the data set.
  2. Measure: Type of data set. This will drive the rest of the drop-downs.
  3. Source: Source budget.
  4. Basis: Which set of numbers, for the selected Measure, are going to be used. (Planned, Scheduled, Actual...ex.)
  5. Product: For Production/Reserve-based Measures, which product is to be used? (Blank means all).
  6. Type: Used to specify whether risk and/or Working Interest are incorporated in the measure.
  7. Operator: Let's you select to calculate totals cumulatively if desired.
  8. Numerator: Unit for the selected Measure.
  9. Denominator: Breakdown by time series. Blank will default to the "Dates" selection.
  10. Price Scenario: Which of the Price Scenarios defined in the Configuration to use? (Default Price Scenario is automatically selected when certain types of Measures are chosen)
  11. Currency Conversions: Which of the Currency Files defined in the Configuration to use?
  12. Exchange File: Which of the Exchange File defined in the Configuration to use?
  13. Escalation File: Which of the Escalation File defined in the Configuration to use?
  14. Scale File: Which of the Measure Scale Files defined in the Configuration to use?
  15. Reserve Variant: For Reserve-based Measures, specify either Economic or Calculated as the Reserve variance type.
