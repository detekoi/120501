Project Variance Summary Report Overview

The Project Variance Summary report compares the key project values tracked in the Project Summary report between two budgets, or between two different capital definitions (such as Forecast and Actual numbers) within the same budget.

Key Fields compared in this report:

  • Unrisked Gross CAPEX
  • Reserves Added
  • F&D Rate
  • Annual Prod
  • Exit Rate
  • Annual Cost
  • Exit Cost
