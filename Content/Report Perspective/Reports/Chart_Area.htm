Chart Area

This subsection of the Format Chart tab allows you to modify the physical appearance of the charting area.

 

Background allows you to select a colour for the background space.

Style applies a preset configuration for the labels and headings.

Enable Chart Interactivity enables the Interactivity button which allows you to define specific actions that can be taken on the chart (such as right-clicking) and what happens when that occurs (link to an external dataset for example).

Outline opens the dialogue below, which lets you set a border for the chart, and the width/style of that border.

 

General Properties (shown below) define a description and the unit spacing.

 

Custom Properties let you script extra options if you are familiar with the report controls. This is for advanced users only.
