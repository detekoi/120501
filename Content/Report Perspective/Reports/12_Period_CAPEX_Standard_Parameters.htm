CAPEX Standard Parameters

These are suggested values for the required Report Parameters to run a basic CAPEX Report.

Any of these values can be customized to generate a more specific, personalized report. The values listed here were used to generate the "Example" report.

If a parameter is not covered (ex: Project & Report filters), it is assumed you will use the default values.

Parameters marked with ** ** will vary based on the data set available.

 

Budget(s)

Budget: **Evergreen Forecast**

Capital Definition: Unrisked Gross

Capital Type: Forecast

Currency Exchange File: Default

Currency Escalation File: (no escalation)

Measure Scale File: (no scaling)

Capital Last Actuals Date: Greyed out

 

Dates

Reporting Period: Monthly

From Date: **Jan 1, 2014**

To Date: **Dec 31, 2014**

 

Units of Measure

Currency: Canadian Dollars (Thousands)

 

Project Filter(s)

Project: **Wiliston**

 

 

Disclaimer: All reports are run with the "base" data configuration. Depending on your business-specific configuration, you may or may not have the same units or options available from all drop-downs.
