Report Appearance Preferences

There are default Configuration Options that control how the Reports render on the Screen, as well as in printed output.

 

  • Logo - upload your company's logo into your esi.manage database and have it printed in the Reports.
  • Page Size - defines how the Report should be rendered on the screen (this does not influence printed output). Selecting a larger default page size will place more white space between columns on the screen, however, it will likely result in a scroll bar showing up at the bottom of the Report when it is rendered.

 

There are also options to control how the various data types (Money, Percent, Volume) are displayed (on screen and in printed output) as well as the Pattern that should be used to represent a Project Description within a Report (for example, you could select Project Code only, and have this printed instead of the Project Name in the reports).

Each of these Formatting options has a local help screen (accessed via the ? button).
