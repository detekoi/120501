Saving and Publishing Reports

Report templates can be saved at any time, even if the Report Parameters have not been completely filled out.  This allows for the creation of temporary reporting templates that can be completed at a later date.

 

To Save a Report:

  1. Fill out the Report Parameters and click ‘Save’ .
  2. Name the report.
  3. Fill in the Comments field if desired.
  4. Press ‘Next’
  5. Choose the folder where the report is to be saved. If this is the first time you have saved this Report, then it will be saved in your Private Reports Folder.  
  6. Click ‘Finish’.

 

Using ‘Save As’

‘Save As’    follows the same procedure as the ‘Save’ function above.  However, the ‘Save As’ function requires a New report name.  To save a report under the current name use the ‘Save’ function.

 

Renaming a Report

To Rename a Report, right-click on the file that you want to rename and choose ‘Rename File’ from the drop-down, type the new name in the form and press ‘OK’.

Change the name of the File as desired.

In this case, '2013' was changed to '2015'.

 

Special considerations when saving a Report:

Report names are unique and cannot be duplicated in esi.manage.  If a Report name already exists, esi.manage will prompt you to modify the name to one that is not currently in use.

Comments are optional but can provide extra details on the report that are not provided in the name.

 

Publish a Report

When Reports are created they are saved in the Private folder by default.  They can be made available to all esi.manage users by Publishing the report.

 

To Publish a Report:

Right click on the Report name and choose ‘Publish File’.

 

Choose the Folder that will hold the Report and press ‘OK’.  The Report file will be moved from the original Folder to the Public Folder that was selected.

 

Special Consideration when Publishing a Report

Once a Report has been Published and made public it cannot be moved back to a Private Folder instead use the Copy / Paste functionality to create a copy of the Report in your Private Reports.

 

Changing Owners

Changing a Report Owner will move the Report out of its current folder and into the Private Folder of the specified User.  This feature is useful to administrators want to create templates for specific users.
