Baseline Fixed OPEX

If Production Baseline has been enabled, two additional Data elements are added to the Fixed OPEX table.

 

  • The Asset Baseline Type is a read-only field and indicates what Type of Baseline modelling has been specified for this Asset.
  • The Baseline element is a Boolean and indicates the Fixed OPEX Segment that will represent the Base (existing) Operating Expense profile.

 

Note: the Fixed OPEX Table is used for OPEX Actuals and Field Estimates - the Baseline Flag has no impact to OPEX calculations that are using Field Estimates or Actuals (including Blended). This is described below.

As with Production, when Baseline is enabled for an Asset, it is only the calculated Uplift OPEX that is Reported or used in Metric calculations.

Production Calculations

 

For Total Uplift Modelling

  1. The Fixed OPEX entry that is marked as Baseline represents an Operating Expense for the Asset in the Baseline state.
  2. A corresponding Fixed OPEX Entry (of the same OPEX Type and Asset, and Product if it has been defined for the Baseline entry) that includes the Base amount, plus the expected Uplift amount.

 

For a report on a Forecast Basis, the Uplift OPEX is the delta between the two.

Note that:

  1. The Start Date for the Uplift (non Baseline) OPEX item indicates when Uplift OPEX will be reported - it is not based on the Onstream Date of an associated Uplift production segment.
  2. It is necessary to define the Baseline Operating Expense as this value is used in Blended OPEX reporting as the Value that is reported is the Uplift amount (described below).

 

Calculations with Actuals or Field Estimates (including Blended)

When Actuals or Field Estimates have been entered against the Asset, the Baseline Flag on these entries is not used.

What is reported is the entered Actuals less the calculated Baseline amount for the corresponding period. For example:

  1. Baseline OPEX = 100 / month
  2. Uplift OPEX = 120 / month (Total Uplift).
  3. Actuals = 135
  4. An OPEX report on a Forecast Basis would report 20
  5. An OPEX report on an Actual Basis (including Blended) would report 35 (135 - 100).
