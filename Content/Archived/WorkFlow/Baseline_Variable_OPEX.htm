Baseline Variable OPEX

If Production Baseline has been enabled, an additional Data element is added to the Variable OPEX table.  

  • The Baseline element is a Boolean and indicates the Variable OPEX segment  that will represent the Base (existing) production profile.

 

Variable OPEX calculations (particularly where Actuals or Field Estimates are involved) is complex.

First, note that the Fixed OPEX Table is used for OPEX Actuals and Field Estimates (including entries that corresponding to Forecast Variable OPEX entries). The Baseline Flag on Actuals entries has no impact on the resultant calculations.

As with Production, when Baseline is enabled for an Asset, it is only the calculated Uplift OPEX that is Reported or used in Metric calculations.

Production Calculations

 

For Total Uplift Modelling

  1. The Variable OPEX entry that is marked as Baseline represents a Variable Operating Expense for the Asset in the Baseline state (Baseline Variable Rate).
  2. A corresponding Variable OPEX Entry (of the same OPEX Type, Asset and Product as the Baseline entry) that includes the Base amount, plus any anticipated incremental variable amount (Non-Baseline Variable Rate).

For a report on a Forecast Basis, the calculation is:

Uplift OPEX = (Uplift Production x Non Baseline Variable Rate) - (Baseline Production x Baseline Variable Rate)

 

Note that:

  1. The Start Date for the Uplift (non Baseline) Variable OPEX item indicates when Uplift OPEX is valid - it is not based on the Onstream Date of an associated Uplift production segment.
  2. For Total Uplift, if the Variable OPEX Start Date is prior to the Onstream Date of the earliest Uplift segment, No OPEX amount will be reported as it relies on the Uplift Production (which in the Total case is contained within the production Uplift segment).

Calculations with Actuals or Field Estimates (including Blended)

When Actuals or Field Estimates have been entered against the Asset, the Baseline Flag on these entries is not used.

What is reported is the entered Actuals less the calculated Baseline amount for the corresponding period. The calculated Baseline amount is: Baseline Variable Rate x Baseline Production.
