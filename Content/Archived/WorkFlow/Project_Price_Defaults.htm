Price Defaults

If desired, Default Price Forecast can be set at the Project Level.

 

A precedence is then used to determine which Price Forecast will be the default for each Production Segment in a Project:

  1. Selected Project Level Defaults
  2. Then Jurisdiction Level Defaults
  3. Global Defaults.

In addition, a specific Price Forecast can be selected for any Production segment.  Any Price Profiles that are populated by default will be highlighted in yellow (eg. Saskatchewan Gas_T), and any manually selected Price Profiles for the individual line items will be in white.

 

For more information, please see Configuration>Prices>Price Defaults
