Production Forecast Import Overview

esi.manage provides the ability to import the information required to define a Production Segment (either formulaic or Tabular) into a specific Project. The import routine is accessed via the Import button (shown above) on the Production Volume page.

The import dialogue (shown at right) allows you to specify some of the information required for a new Production Segment. Some of the information shown in this dialogue (such as On Stream Date, WI%) is also contained within the import file. The rule is that whatever is contained in the Import file will take precedence over what is specified on the dialogue.

 

  1. Production Import can either be Tabular or Formulaic.
  2. Multiple Segments can be identified within a single import file, however, when this import is applied all existing segments are deleted, except those which have Field Estimates,  Actuals or Financial Actuals.
    • Existing segments which have 'Actuals' will be converted to Tabular, and the new Forecast segments added in. Future Actuals imports will be applied against the Tabular segments.
  3. Dependencies can be established by selecting the Dependant On Expenditure, which can result in a shift of the On-Stream date in the Import file if it is prior to the End Date of the selected expenditure.
