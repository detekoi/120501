Copy and Paste Tabular Production

Within the Tabular Production View, there is the Option to Copy the table data to the Clipboard. This will take the entire Table (including the Header), and it can subsequently be pasted into another application such as Excel.

There are two 'modes' of Paste Function.

 

Full Table Paste

  1. Typically, this involves copying an existing table from esi.manage, then pasting it into Excel.
  2. Additional rows are added for future production, and historical rows may or may not be deleted.
  3. Once the 'replacement' data set has been established, the entire Table (including Header) would be copied from Excel and then Pasted into esi.manage.
  4. With this approach, the existing table is replaced - if you want to retain historical data it must be part of the data set that is being pasted.

 

Select Data Paste

Data can be selectively cut and paste into the tabular production table by column or row.

 

Row based pasting:

When pasting a single/group of rows, the data must contain a column for each mandatory column in the table, typically start/end dates and a production value.  New entries can be created in the table this way, or existing values can be updated.

  1. Select rows of data that you wish to paste into tabular production table.
  2. Right-click in the tabular production table, select paste.
  3. When prompted, chose between Replace or Merge.  
    • Replace: delete all existing data in the table and replace with data being pasted
    • Merge: data will automatically find the row in the tabular production table with the same start and end dates regardless of current cell selection in the tabular production table, and begin pasting.  New entries can be created in the tabular production table, and/or existing entries can be updated this way.

 

Column based pasting:

  1. Column based pasting can only be used to update numerical production values, and cannot be used to enter new dates if no production values are associated with them.
  2.  If you wish to have new periods - first have them added into the table by selecting .
  3. Select data to be pasted into tabular production table.
  4. Right click on cell in the tabular production table that you wish to begin pasting at.
  5. Select Paste from menu, and data will be pasted down from selected cell.
