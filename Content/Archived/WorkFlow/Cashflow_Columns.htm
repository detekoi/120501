Cashflow Columns

Cash Flow columns are pre-defined collections of the various elements that are part of the Cash Flow Calculations. You can select one of the configured Groups from the Cash Flow Analysis view, and have the data displayed as configured here.

  • Existing Groups can be modified, or new Groups created.
  • Display indicates what will appear as the Column Header in the Analysis view.
  • Element is a drop down of available Economic Elements
  • Sort Order indicates how the elements will be displayed (left to right).
  • Discount indicates whether the apply Discounting to the Element.
  • Aggregate is a toggle:
    1. If it is unchecked each CAPEX Type that is involved in the Analysis will be displayed in an individual Column.
    2. If it is checked, all CAPEX will be aggregated into a Single Column.
