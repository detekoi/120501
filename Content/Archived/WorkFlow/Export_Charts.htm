Export Charts

Exporting is the means of getting data from esi.manageTM to an external format. Currently, charts can be exported into HTML format, Adobe .pdf, and Excel spreadsheets (.xls). The chart itself does not need to be viewed in order for the export to work, however the necessary parameters do need to be entered beforehand.

 

Export a Chart:

  1. Open a Chart and define the parameters so that it can be viewed.
  2. Click the "Export" button.
  3. You will now be prompted to enter a file name and select a file type.
  4. Html is the default type, and the name of the chart in the program is the default name, rename the chart accordingly and select the desired output format.
  5. Press "Ok".
  6. Navigate to the folder the file was saved to and double-click it to open it and view the exported chart.

 

Copy Charts to Windows Explorer

You can drag and drop a chart on to your desktop or a Windows folder.

You will be prompted to move/copy the file. Click Yes to copy the file.

The file is saved as custom#.png.

This image file can be used like any other image and inserted into another document or emailed to management for review.

 

Copy Charts into Microsoft Word

You can drag and drop the chart directly into a word document and it will appear as an image.
