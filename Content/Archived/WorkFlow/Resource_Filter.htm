Project Filter

The left hand pane of the Resource Gantt view contains a similar hierarchy to the Project Filter of the Project Gantt. Resources do not "roll-up" the way expenditures do. The resources are instead broken down by resource type in the Gantt. This can be further sub-divided by Project.

 

The Resource hierarchy goes from Budget to Capacity File to Resource Category then Resource Type, and finally Project at the lowest level of granularity.

If resources have been allocated to project where there is no Capacity File the resources will still be displayed in the resource Gantt but will be displayed under a "No Capacity File" heading (shown below).
