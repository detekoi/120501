Edit Ranking Units

All of the Measures (CAPEX, Production, OPEX) and corresponding Metrics need to be evaluated on a consistent basis. This is established through the Ranking Units dialogue, which is set for each user.

 

If Economic Metrics are being used for Ranking, define the appropriate Economic Discount Date.

Select the desired blending mode:

  • Match Forecast CAPEX Total: Adjusts the first month of forecast data to align the projected Blended Total with the new Forecast Total
  • Use Original CAPEX Forecast: Adjusts the remaining months of forecast data to align with the original Forecast Total
