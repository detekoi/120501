Open Data Matrix

The Data Matrix can be Opened against a Project Set(s) (via the Navigator Right Click menu) or against the Projects currently displayed in the Project List (the Active Selection).

 

To Open the Matrix against a Project Set:

  1. Expand the Project Sets portion of the Navigator.
  2. Select one or more Project Sets (Ctrl + Left Click).
  3. Right Click in the Project Sets area of the Navigator.
  4. Select Open Data Matrix for 'Project Set'.

 

You can open multiple Data Matrixes; select a Project Set and use the process shown above, then select a different Project set and Repeat the process.

 

To Open the Data Matrix against the Projects List

  1. Using the process described in Active Selection, get the Projects of interest into the Projects List.
  2. Choose the Open Data Matrix button from the Projects List button ribbon.
