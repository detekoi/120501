Follow Up Report Overview

To enable this report you must first turn it on in the Configuration > System > Miscellaneous > Site-specific Features.

The Follow Up Report compares the Current Forecast for CAPEX and OPEX to the original Budget and up to three Archives to track progress against plan and Quarterly Archives during the year.

Note on this report:

Printing

This report is suitable for printing on legal size paper (8 1/2" x 14"). If other sizes of paper are used when printing, use the scaling option on the Print Dialogue to scale to the correct size of paper.
