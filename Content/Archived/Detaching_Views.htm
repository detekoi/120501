Detaching Views

This feature is suited to users who have multiple monitors. Certain views within esi.manage (the Analysis Views are a common choice), can be "Detached" from the esi.manage desktop and moved to a separate monitor.

 

Select the Page Tab (OPEX Analysis as shown above) and drag it - as you pass the edge of the esi.manage border, a dashed outline will be shown. Releasing it will move the view outside of esi.manage.

The view will still retain connection to esi.manage (you can still change settings on the Analysis View and have it recalculated).

Closing the detached view will return it to it's default behaviour.
