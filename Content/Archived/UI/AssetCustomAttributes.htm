Asset - Custom Attributes

Custom Attributes that have been configured to an asset appear on the bottom on the asset screen once an asset has been selected.

Values may be entered into these attributes to further define the asset.

To add comments, click on the comment bubble.

If a comment exists the comment bubble will have a check mark on it. If you hover over the comment bubble the comment will be visible.

You can collapse or expand the Custom Attributes by either clicking on the arrow at the attribute grouping or by using the expand all/collapse all buttons.

 

Custom Attributes are defined in the Configuration Perspective.
