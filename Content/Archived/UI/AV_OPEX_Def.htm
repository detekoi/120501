OPEX Analysis View

The OPEX Analysis View has been designed to streamline Project Tracking. From this one view you can:

  1. Examine all Data Types (Forecast, Field Estimates and Actuals), as well as the calculated Blended values.
  2. Filter the data by Asset or OPEX Type.
  3. View the data using any of the Production Definitions: Gross Unrisked Raw, Net Risked Sales, Net Risked Raw, Net Unrisked Sales, and Net Risked after Royalty.
  4. Select an Escalation File to apply to the calculations.
  5. Choose to display the data Daily, Weekly, Monthly, Quarterly or on an Annual Basis, for up to 100 Periods.
  6. Choose the Dates to use for Start Date, and Last Actuals and Field Estimates for Blending calculations.
  7. Modify the Chart view to display select data (Forecast and Blended for example), use different displays (Bar, Line or Area), or display Cumulative or Periodic values.

The Asset and OPEX Type drop downs utilize a multi-select Tree View control. The 'contents' of these list will be dependent on where the Analysis view was launched from:

  • From a Project - the Asset list will be based on the data contained within the Project.
  • Global - Assets will be the union of the Project's content.

 

Note: these tree controls do not interact, i.e.; Filtering the OPEX Type list does not reduce the Asset list to those which have a corresponding OPEX entry.

When the data is generated, it will be based on those OPEX Entries which satisfy ALL of the filtering.

 

Note: the Analysis View can be Detached and moved to a second monitor.
