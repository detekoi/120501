Linkage

The relationship dependency can then be toggled within the ‘Linkage’ column by double clicking the cell in the measure row you wish to change based on your project management preference.  Measure dependencies can be linked in two different methods.

 

  1. End -> Start - This linkage relationship means that the previous measure item must be completed before this measure can start.  (This is the equivalent as an FS Critical Path project management relationship)
  2. Start -> Start - This linkage relationship means that the previous measure item must be started before this measure can start.  (This is the equivalent as an SS Critical Path project management relationship)
