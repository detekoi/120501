Calculations

Within the dependencies tab, there are several calculations provided within the data view.  Calculations will behave different based on dependencies that have specified.  

NPD% = Net Probability of Doing %

NPS% = Net Probability of Success %

*the two values above are calculated automatically based on the values entered in the following 4 fields:

Activity Factor (AF%) – Based on the value entered here, the NPD% and NPS% will rise and fall.  

For example, in the image below, you can see that the AF% is 75% for the Completion expenditure, therefore the highest that NPD% and NPS% can be for that measure (and all subsequent measures in the dependency chain) is 75%.

 

 

Outcome Probability (OP%) – Based on the value entered here, the NPD% and NPS% will rise and fall.  

For example, in the image below, you can see that the OP% is 50% for the Equip & Tie-in expenditure, therefore the NPD% and NPS% are again effected.  Where previously in the image above these values were reduced to 75% because of the AF% impact, they are now further lowered to 37.5% (A result of the initial NPD% for this expenditure being 75% and subsequently multiplied by the 50% OP value for that line item).

 

 

Failure (Checkbox) – If this value is checked, the correlated expenditure will only occur when its parent node in the dependency chain fails. Thus, any AF% of OP% values entered for this activity will only effect calculations if the parent entity fails.  

For Example, the ‘Failure’ value has been specified against the two previously affected measures for the two images above.  Note that the NPD% and NPS% values for those measures are now blanked, as well as any subsequent measures.

 

Reset NPD% (Checkbox) – If this value is checked, the NPD% for that dependency chain is reset to 100% at that measure level.  Previously entered values for AF%/OP% will not affect NPD% or NPS%.  

 

 

Compounded Calculations – As different values are populated for AF% & OP% on a line by line basis, their impact will be compounded to determine the result of NPD% & NPS%.  

In the example below, you’ll see different AF%/OP% values applied to 3 different expenditures.  

Completion AF% = 75%

Equip & Tie-in (Well A 0001) OP% = 50%

Equip & Tie-in (Well A 0002) AF% = 50%  

Total NPD% & NPS% = 18.75%

{100% * 75% * 50% * 50% = 18.75%}

 

 

These compounded calculations provide the accurate overall impact on NPD%& NPS% based on the line-by-line variances of AF% & OP%.
