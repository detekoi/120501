Creating Measures

Within the ‘Dependencies’ tab, the user has the ability to add any of the 4 primary measures directly to the project by clicking one of the respective buttons for Expenditures, Production, Fixed OPEX or Resource Allocation.

The user will be prompted for the appropriate information required for each measure upon clicking the corresponding ‘Create’ button. The following is a list of the required information in order to create any type of measure in the Dependency screen.

 

Create Expenditures

  • Depends On
  • Asset
  • Expenditure Category
  • Expenditure Type
  • Start Date
  • End Date or Duration
  • CAPEX Amount
  • Currency Unit

 

Create Production

  • Depends On
  • Asset
  • Product
  • Production Rate
  • Onstream Date or Offset

 

Create Fixed OPEX

  • Depends On
  • Asset
  • Product
  • OPEX Type
  • Start Date
  • End Date or Duration
  • OPEX Amount
  • Currency Unit
  • Time Period
    • Apply While Producing

 

Create Resource Allocation

  • Depends On
  • Resource Type
  • Start Date or Offset
  • End Date or Duration
  • Allocated
