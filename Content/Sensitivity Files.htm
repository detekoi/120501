Sensitivity Files

Sensitivity Files are created through the Navigator and can be used in several areas of the application.

The files uses source data from several separate areas that the user can apply to the global settings within esi.manage. The sensitivity file can be selected for use within Analysis Cases as well.

These files allow the user to group selected parameters together as global settings for ease of use.

To create a sensitivity file, right click in the Navigator and select New- Sensitivity File.

 

 

Give your new File a proper name and select Next.

The Sensitivity File consists of 4 separate parameters.

1. Price Scenario - the user will select from a list of created price scenarios that are populated within configuration to use. This could be Current Price - High Price type of scenarios for the file.

2. Escalation File - the escalation files are set up within Configuration and will allow the user to select a specific file that will model inflation within the system as a percentage.

3. Exchange File - set up to model currency exchange within the system in Configuration, the user can select a specific file to model this within their sensitivity files.

4. Scale Files - the scale file applies scaling of a selected measure or measures. The user will first create a scale file in the configuration and apply within the sensitivity file.

 

For more information on each type of file associated with the Sensitivity Files please see their specific areas within Configuration

 

Once the parameters have been filled the user will see their Sensitivity Files within the Navigator in the Private folder.

 

Global Parameters

The newly created Sensitivity Files are applied within several areas of the application for economic evaluations.

The Global Parameter found below the windows tool bar allows the user to set sensitivity files and evaluation files that will be used for all economic runs within esi.manage.

Note:

By setting the Global Parameters with these files, the user will overwrite all uses of other Sensitivity or Evaluation files that exist previously in Analysis Cases or where these files may be in use.
