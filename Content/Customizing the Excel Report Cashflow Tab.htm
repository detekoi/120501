Customizing the Excel Report Cashflow Tab

Introduction

The target worksheet tab of the Excel Economic Report Template file designated to hold the Cashflow data from the Analysis Case customized by the user to bring in data from any of the columns available to display in the esi.manage Analysis Cashflow tab. In the past, it is necessary to include/exclude these columns within esi.manage first, prior to exporting the Excel report that will contain exactly the columns included. With the Excel Report Template, the column visibility settings in esi.manage do not impact the availability of data in the Excel export.

Available Data for Export

Each of the Cashflow data columns listed below are available to be exported to the Excel Economic Report file:

- Dates (and separately, corresponding duration of each period)

- Gross production (by product)

- Net production (by product)

- Net production after royalties (by product)

- Gross capex (by expenditure type)

- Net capex (by expenditure type)

- Gross opex (by opex type)

- Net opex (by opex type)

- Quality (by product)

- Effective price (by product)

- Burdens (by name)

- Royalty (by product)

- Gross revenue (by product)

- Net revenue (by product)

- Net revenue after royalties (by product)

- Current tax (by asset or jurisdiction)

- Deferred tax (by asset or jurisdiction)

- Tax depreciation (by asset or jurisdiction)

- Taxable income (by asset or jurisdiction)

- Income before tax

- Income after tax

- Operating income

- Net margin

- DD&A

- BTAX cashflow (present value with no discounting)

- ATAX cashflow (present value with no discounting)

- BTAX present value

- ATAX present value

- Custom (defined in Fiscal Model > Custom Economic Calculations spreadsheets)

- Custom Cashflow (defined in Fiscal Model > Custom Economic Calculations spreadsheets)

Each individual data column in esi.manage Analysis Cashflow can be mapped to columns in the Cashflow worksheet of the Excel Economic Report Template, by following the Formatting Requirements below.

Formatting Requirements

The worksheet tab where the Cashflow data are imported into must meet the following requirements:

- Worksheet name must be “ESI Cashflow (nominal)” if not accounting for future currency escalations, or “ESI Cashflow (real)” if accounting for future currency escalations

- For a given column (Column N) to be populated with data, the following requirements must be met:

o Row 1 (Cell N1):

 For the mapping to import from esi.manage: set Cell N1 value to IN

 To not import any values from esi.manage: leave Cell N1 blank

o Row 2 and 3 (Cells N2 and N3) – must meet the following syntax rules:
