Navigator Folders & WorkFlow

 

As shown above, each of the Navigator items (Project Sets, Reports, Analysis Cases, Exports, Import Templates and PEEP Imports) has a folder structure under it. There are two 'classes' of Folders:

  • Public - folders that contain Public instances (visible to everyone - depending on Roles and Permissions) of Reports, etc.
  • Private - folders that contains Reports, Analysis Cases, etc, that have been created by you, but have not been made Publicly available yet.

 

You interact with the Navigator through the Context (Right Mouse Click) menu. The options that are accessible are:

  • New, Edit or Delete - create a New Instance of the Navigator node (e.g.; Report), or open the existing item to modify it, or Delete the item.
  • File Manipulation - Copy, Rename, the selected item (i.e.; Optimize Forecast 2010 CAPEX)
  • Publish Options - the process of taking a Private item and making it available to the rest of your organization.
  • Set as Active Selection, Copy to Project Browser - the Workflow for Populating the Projects List.
  • Act on the Project Set - Open Data Matrix, Bulk Edit, Analyze, Aggregate, or Report On. Apply the selected Tool against the Project set.
