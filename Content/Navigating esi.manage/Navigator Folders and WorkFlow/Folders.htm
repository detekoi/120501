Folders

The main Navigator entities - Project Sets, Export Jobs, Analysis Cases and Reports can be organized into folders. Folders can be either Public (visible to all users) or Private (only visible to the creator). New items (a Report for example) are always Private when first created. You must Publish the file to make it visible to other users.

Private folders can only be created by users with the Public - Write role, and folders can only be edited or deleted by users with the Public - Admin role.

Folders are created by expanding the element in the Navigator you want to generate a folder for (such as Reports) and right clicking on the type of folder you want to create - Public or Private. From the context menu, select "New Folder". The top Folder (Public or Private) above where you Right Click controls whether the new Folder will be Public or Private.

 

You will be prompted to name the folder. Enter a name and click "OK".

 

Folders can be Moved within their 'Node' by dragging and dropping (this means that a Private Folder cannot be Dragged under  the Public Node - nor can a Report Folder be dragged into the Analysis Case tree). In addition, individual items can be moved by drag and drop (standard windows multi-select options apply).
