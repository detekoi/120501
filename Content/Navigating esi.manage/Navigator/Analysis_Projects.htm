Projects

To run an Analysis Case, you will need a list of Projects.

  • Projects are 'populated' into the Analysis Case by selecting Project Sets, within the Project Sets tab, as shown below.
  • More than one Project Set can be selected for a particular Analysis Case.
  • If a Project 'belongs' to two Project Sets, its results will be included at the Roll Up for each Project Set. It will effectively be "doubled-counted" in the results.
