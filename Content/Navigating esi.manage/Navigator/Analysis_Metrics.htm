Analysis Case (Portfolio) Metrics

As described in the Analysis Case Overview, companies can define Default Metrics within the esi.manage configuration.

Default Metrics will be calculated for every Analysis Case and cannot be modified.

However, there is no restriction on how many instances of a given metric can exist within an Analysis Case, i.e., there may be a Corporate Metric of NPV @ 10%, users could also add NPV @ 15% and NPV @ 7% into their particular analysis case.

  • When an Analysis Case is opened/created, any Corporate Metrics will be present, indicated by an Italic Font. These metrics cannot be modified.
  • Additional metrics can be added via the buttons at the upper right side of the page.
  • Properties vary for each selected metric, and you can modify the properties (i.e., Discount Rate) by selecting the individual metric in the left pane.
  • The Arrows can be used to order the Metrics, which will be reflected in the display of the Metric Analysis.

 

See Economic Calculations for a description of the various Metrics
