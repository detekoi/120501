Export Jobs

Export Jobs allow you to export one or more projects from esi.manageTM  into XML that can be consumed by Excel or translated into external systems such as reporting cubes or execution tools.

Export jobs can be created once and used over and over.

Export jobs can be run manually from individual projects, the export job itself or initiated Manually or Automated via a call from a Web Service.  

Enabling the Incremental functionality, users can Export the Projects that have Changed Since a specific date which can be used to efficiently run daily batch updates of external systems.
