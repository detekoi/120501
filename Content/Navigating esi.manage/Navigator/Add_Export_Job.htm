Add an Export Job

Right-click the Export Job folder and select new to create a new export job.

 

  1. Enter a name to describe your Export Job.
  2. Enter a file location including the entire path. Parameters can be used to assist in the naming of the file:
  3. %B = Budget Name
  4. %P = Project Name
  5. %J = Job Name
  6. %D = Current Date in YYYYMMDD format
  7. %T = Current Time in HHMMSS format

Budget and Project naming parameters only function when exporting one project.  If exporting more than one project, they will not appear in the file name.

The example in the screen print would create the following file C:\Export Files\Montney G&G-20090914.xml

  1. Select Elements from Options Tab.
  2. If you wish to run the export job against specific project sets, select the project sets from the projects tab.
  3. Save the Export Job by clicking save button on the esi.manageTM  toolbar.

 

To run the export job, click on the Run This Export Job button on the Project Export toolbar.

 

Delete an Export Job Set

Right-click the Export Job you wish to delete and select Delete "Export Job".

A warning message will appear on the screen notifying you of the delete. If you are sure you want to delete the Export Job, click OK.

The Export Job will be deleted from esi.manageTM .
