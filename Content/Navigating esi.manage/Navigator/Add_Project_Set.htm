Add a Project Set

Using the Navigator

Project sets are created by right clicking the Navigator and selecting "New Project Set". This will launch the Project Set creation wizard (shown below).

  1. Select the Budget and define the hierarchy filters needed to refine the project list as desired.
  2. The Projects in Set Preview column will update to show the projects defined by the selection.
  3. Note that the Quick Find and Clear workflows described with the Project Browser are valid for the Project Set Filter pane.
    • Project Sets can also be created out of the filtered projects list (Project Browser) through the right click menu.

 

Using the Project Browser

The alternative method to create a Project Set is by using the Project Browser. As you can tell from the previous method, it is simply a window where you set the filtering criteria that you want to save as your Project Set. If you set the same filtering criteria in the Project Browser, you can do the same thing. This may be easier at times, because you have already set your filtering criteria that you want to save in your Project Browser as part of another workflow.

  1. Set the Budget you wish to create the Project Set for, in the Project Browser
  2. Set the filtering criteria you want to save as a Project Set
  3. Click the "New Project Set from Existing Selection" button at the top of the Project Browser
  4. Name Project Set
  5. If you want to save in the Private folder, then click Finish
  6. If you want to save in a subfolder, click Next
  7. Select desired save folder and click Finish

 

Delete a Project Set

Right-click the Project Set you wish to delete and select Delete "Project Set".

 

A warning message will appear on the screen notifying you of the delete. If you are sure you want to delete the project set, click OK.

The project set will be deleted from esi.manageTM .
