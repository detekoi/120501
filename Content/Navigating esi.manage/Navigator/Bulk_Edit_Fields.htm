Bulk Edit Fields

By hitting the button you can add multiple levels within each option (except tabular scaling) to make multiple changes at once.

  • Project Attributes
    • Fixed Attributes  - Change the Project Status, Project Manager, Project Location or Project Comment across all projects.
    • Hierarchy Attributes - Change the Asset Team, Economic Group, Project Type, Play Type, Play Time, Op/ Non-Op, Drill/ Non-Drill, CAPEX Budget Year, Program or Reservoir across all projects.
  • Date Shifting - Change the project date by shifting Start Date or Shift by Days.
  • Production
    • Volume (Derived Yield) - Changes the Derived Products and Primary Products fields across all projects.
    • Downtime - Changes the downtime across all projects.
    • Shrinkage - Allows you to Add or Update Annual Shrinkage (%) by Product across all projects.
    • Tabular Scaling - Allows you to change the Scale Factor of different products across all projects.
