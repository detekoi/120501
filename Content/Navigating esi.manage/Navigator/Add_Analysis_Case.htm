Creating a New Analysis Case

From the Navigator Tab, right-click the "Analysis Cases" folder and select "New Analysis Case".

 

You are required to enter a Name for the Analysis Case. By default, a New Analysis Case will be Private to the user who created it.

 

If you click "Finish", the Analysis Case will be saved and opened. If you have multiple Private folders under Analysis Cases, you have the option to select "Next" and choose the location you wish to save the Analysis Case in.

 

When a new Analysis Case opens, it will be populated with the Default Economic Properties and Default Economic Metrics that have been established in the Configuration.

The General Sequence for Configuring an Analysis Case is:

  1. Configure  the Properties
  2. Add any additional Metrics needed
  3. Select the Project Set(s) that are to be analyzed by default if the Analysis Case is run from the Navigator (optional).
  4. Run the Analysis Case

 

Delete Analysis Case

From the Navigation Tab you can delete an existing Analysis Case.

  1. Open the Analysis Cases folder.
  2. Right-click the Analysis case you wish to delete and select Delete"Analysis Case".
  3. A warning message appears confirming you want to delete the selected Analysis Case. If you are sure, click "OK".
