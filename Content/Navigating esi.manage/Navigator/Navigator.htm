The Navigator

The navigator allows the configuration and storage of a number elements including: Project SetsA Project Set is a collection of Filtering Criteria applied against a Budget. It will return a list of Projects which satisfy the criteria, Reports, Analysis CasesDiscounted Cash Flow and Economic metric calculations, Analysis Results,Export JobsConfigurable Export of user supplied and calculated results in an XML schema., Import Templates, Enersight Import Jobs, Peep Import Jobs, Sensitivity Files, and Evaluation Files.

Items which have been saved are subsequently accessed through the Navigator.
