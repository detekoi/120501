Project Set Attributes

Project set attributes are a type of custom attribute used to append additional data to the project set.

Once configured, they appear in the custom attribute tab of the project set.

 

Double click on a project set to open it and then click on the custom attributes at the bottom of the screen.

 

The custom attributes for project sets are shared with those for Budgets and so any attributes configured for budgets will show up in the project sets and vice versa.
