Run an Export Job

If the Export job was created against a project set, it is possible to Run the export directly (once the Export job is open).

Click on the Run This Export Job button on the Export Job toolbar.

 

The Run Export Job dialogue will open on the screen, giving you the opportunity to decide whether you want to run an Incremental Export since a specific date, and/or change your Destination folder.

  • Click Run Export to execute the job.
  • If the Destination directory does not exist you will be prompted to create it.
  • Click OK to create the directory.

The other 2 methods to run an Export Job are from either the Project Browser, or from within a Project

  1. An Export Job can be run from the Project Browser, either against the highlighted Project or All visible Projects.

 

2. Similarly, you can invoke the Export This Project from within a Project by clicking the "Run this Export job" button . Regardless of the route (from the Project List or from with a Project), the following dialogue is opened.

 

Select the Export Job you wish to use. The Projects list will be populated based on the route (if the Export All Visible Projects button from the Project Browser was selected, the Projects List in the dialogue above will show a larger set). You have the option to use the Incremental Export options if desired. Browse to the desired Export folder and either select an existing File, or enter a new File name.

Click Run Export to execute the job.
