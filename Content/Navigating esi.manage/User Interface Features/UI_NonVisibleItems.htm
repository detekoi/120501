Displaying Non Visible, or Hidden views

At certain times (generally when there are a large number of items that have been opened) there can be Views (Projects, Reports) that don't fit within the screen. This will appear like:

  1. The number of non-visible items is listed at the right end of the Page Tabs, with the >> arrow.
  2. Selecting this arrow will open a menu of opened items, those that are currently non-visible will be bolded.
  3. Selecting the item will then make it visible.

 

Similarly, it may be that one or more of the Perspective buttons are not visible. If a Perspective button has been hidden due to the size of the Button Ribbon, this will be indicated by the ">>" arrow on the right hand side.

You can select the ">>" icon and access the other Perspectives (as described above). Alternatively, select the faint line to the left side of the perspective buttons and drag it left to expose the hidden items.

 

If a Perspective has been removed from the Button Ribbon, it can be accessed by selecting the grid button at the left end of the ribbon.

 

  1. Select the grid button, which will expose the drop down.
  2. Double click on the Perspective that you want access to.
  3. You will be transferred into that Perspective, and the Perspective button added to the Ribbon.
