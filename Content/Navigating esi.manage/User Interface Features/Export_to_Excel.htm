Export to Excel

Each editor in the Project Content view has an option to export the table to Microsoft Excel. Pressing this button will open a browser and allow you to select where to save and what to call the export file.

The export will contain all the data in the selected table, with each record as an individual row and all the headings in the same order as they appear in the table.

To export an entire project, use the Export to Excel option in the Project Details tab.

Note: some screens, such as Expenditures, contain several tables on the same screen (Expenditures, Forecast, Field Estimates, etc.). For views with multiple tables, you will need to export each table individually.
