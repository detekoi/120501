Split Screen

The esi.manage user interface allows 'split screen' functionality. The default presentation is tiled; for example, multiple opened projects are presented as page tabs.

At times, it is desirable to view the project data from multiple projects simultaneously, or arrange multiple reports in a dashboard manner.

The split screen functionality is accessed by selecting an 'item' (a Project page Tab as shown below), and dragging it to the border.

 

The page tab could also be dragged to the bottom border, in which case the split would be horizontal.

When you release the page tab, the screen will be split. Closing the last Project within a Split will close the split.

You can also Reset the Perspective, which will return the perspective to it's default presentation.
