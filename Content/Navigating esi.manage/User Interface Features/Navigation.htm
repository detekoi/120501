Customizing your esi.manage

While there is a default layout for each Perspective in esi.manage, there are a number of customization features that allow you to organize views in a manner that is most suited to your current task. Note that any 'customization' to the user interface is specific to each user, any changes will be stored and remembered for your next session.

  • Columns in Tables - can be Re-ordered or Hidden, Sized or Sorted.
  • Page Tabs - pages can be hidden from the Project View.
  • Maximize/Minimize - opened views (Projects, Reports) can be maximized by double clicking on the Page Tab. Double clicking again will return it to the previous size.
  • Split Screen - an area of the user interface (Opened Projects or Reports for example) can be split (once more than one item is opened).
  • Minimize/Restore - the Project Browser/Navigator views can be minimized to provide additional space, and then Restored when needed.
  • Detach Views - if you have a second monitor, user interface features such as the Analysis Views or the Gantt can be dragged to the second monitor. They will continue to update as appropriate.
