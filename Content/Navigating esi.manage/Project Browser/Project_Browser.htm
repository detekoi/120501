Project Browser

The Project Browser is one approach to populating the Projects List. This is suited to users who want ad-hoc access to Projects - to View, Edit, Report or Analyze.

The Project List provides access to Projects for editing, or have Tools applied against the projects:

Ranking, Global Analysis view…, Export Jobs, Analysis Cases, Data Matrix, Gantt Views, and Reports

The first step is to make the Project Browser the Active SelectionThe Active Selection identifies the Source for the Projects List, either the Project Browser or a Project Set(s). by selecting this button.  .

 

When the Project Browser is the Active Selection - whatever changes are made to the Filtering CriteriaFiltering Criteria are identified by a Green check mark - 2010 for the Expenditure Forecast Year are reflected in the Projects List.
