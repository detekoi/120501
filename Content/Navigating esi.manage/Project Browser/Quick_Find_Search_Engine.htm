Quick Find Search Engine

At the top of the Project Browser there is a text box that is used to automatically filter the Project Browser based on a text string supplied.

  1. The searching is applied to any node that has been "activated", such as Hierarchy shown below.
  2. All 'elements' under every active node are examined for matching against the supplied string.
  3. Since there is a Corporate Region (under Hierarchy), as well as a Project Manager Corporate Management, both are found.
  4. The Browser will remain in this state as long as the Text String is still in the cell.
  5. Any of the nodes can be selected to make them an Active Filter.


 

Click the clear button to remove the search criteria and reset the hierarchy.
