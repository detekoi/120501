Clear Filters

When the Project Browser has been filtered, the Context Menu (right click against a specific node) provides some options to modify the selections.

  • Clear Subitems - clears any selection below the node in the tree.
  • Toggle Subitems - toggles the state for all items below the node in the tree. This is shown in the images below, when the US subitems are toggled.
  • Clear All - this will clear all filter selections.
