Adding Projects to a Group

Adding projects to a Group is done from the Group Details view, which is opened by double clicking on the Group you wish to add Projects to.

 

To add Projects to a Group:

  • From the Group Details view, press the “Configure Projects” button.
  • The Project Filter acts just like the Project Browser for filtering to a specific set of Projects based on user criteria.
  • Filter to the set of Projects you wish to add to the Group, a preview of the Projects you are filtering to will appear in the pane on the right.

 

  • Once you have filtered to the Projects you wish to add to the Group, press the “OK” button.
  • You will now see the list of Projects you selected shown on the Group Details view.

 

Removing Projects from a Group

  • To remove a Project from a Group, you can select it in the list under Projects in the Group Details view and press the “Remove Projects” button.

 

  • Projects can be multi-selected in this pane by holding either the shift or the ctrl button.
