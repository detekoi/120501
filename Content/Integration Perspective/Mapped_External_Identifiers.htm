Mapped External Identifiers

  1. To map an external identifier to an asset, find the asset in the list and select it from the list.
  2. A list of already mapped EIDs will appear in the "Mapped External Identifiers" table to the right, and the % currently allocated to each.
  3. Search the external identifier section of the screen and select an EID out of the list to map.
  4. Click the Map External Identifier button. found on the bottom right of the Assets tab.
  5. The EID will now appear in the "Mapped External Identifier" table.
  6. Click the "Unmap External Identifier" button to unmap External Identifiers that have been mapped in error.
