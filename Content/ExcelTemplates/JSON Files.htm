.JSON Files

***THIS FEATURE IS BEING DEPRECATED. PLEASE MIGRATE ALL EXCEL DATA LOAD TEMPLATES TO THE NEW FORMAT***

As briefly mentioned earlier, .JSON files can be created and loaded from templates. A .JSON file will house all of the attribute definitions, mappings, translations, and parameters. A copy of the Excel document attached to the template will also be created when creating a .JSON file. To create a .JSON file, simply Right-Click on the desired template within the Navigator and select Export ‘Template’.

 

The export will prompt a window to specify which directory to place it in. To load a .JSON file, the Import Template button within the template can be selected and the template will be built, with the Excel document attached. If in possession of an Excel document that has the same format as the original, it can be loaded into the template to replace the existing one.

If a copy of the Excel document is needed, the export feature is a useful way to obtain said file.
