Entering Formulas

***THIS FEATURE IS BEING DEPRECATED. PLEASE MIGRATE ALL EXCEL DATA LOAD TEMPLATES TO THE NEW FORMAT***

  • A value that constitutes a valid Excel address (e.g. A9) will be treated as such
  • A value that starts with the equal sign (=) is treated as a formula. Such formulas may contain Excel addresses, variable references, functions, operators and constants.
  • Example of a numeric formula: =A7 + (MYVAR + 8.0).
  • Example of text formula: =SPLIT(A7, “_”).GET(2). Note that string literals in formulas must be surrounded by quotes
  • Any other value is treated as a literal (e.g. Blurg just is the constant “Blurg”)
  • Any value surrounded by quotation marks is also a literal. In particular “A9” means a literal constant “A9” and NOT an Excel address
  • Function names and Excel addresses will be automatically capitalized on data entry. This is a useful way to verify that the formula is being interpreted as you intended (e.g. no spelling mistakes in function names)
  • Two special strings ‘true’ and ‘false’ are interpreted as the Boolean values for true and false, respectively
