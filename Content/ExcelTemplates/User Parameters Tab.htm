User Parameters Tab

***THIS FEATURE IS BEING DEPRECATED. PLEASE MIGRATE ALL EXCEL DATA LOAD TEMPLATES TO THE NEW FORMAT***

 

The User Parameters Tab allows users to define special variables that are unknown at template creation or are user specific in such a way that the parameters are prompted on import to be entered.  A parameter can be created by clicking the Green Plus icon. The Name is the variable named that is defined, and is forced into all caps. The Description defines what the parameter is.  The Type can be selected as a String, Numeric, Date, or a Boolean value. The EDL will ensure that the value entered obeys this condition. A Default can be defined to populate the text box automatically when prompted. It can be overwritten at import. If Mandatory is selected, an error will prompt if the parameter is left blank. The prompt can be viewed and values entered within the template by pressing the human icon in the top right of the EDL template.
