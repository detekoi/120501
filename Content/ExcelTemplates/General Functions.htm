Special Variables

There are 6 special variables defined in the EDL.

  • CDL_IMPORT_TIME: The date/time when the import job was run
  • CDL_IMPORT_USER: The user logged in when import job was run
  • CDL_TEMPLATE_OWNER: Owner of the template
  • CDL_TEMPLATE_NAME: Name of template
  • CDL_TEMPLATE_ID: Database ID of template
  • CDL_SHEET_NAME: The name of the Excel Worksheet for the current Attribute, Time-Series, or Data-Series mapping.
  • This variable takes on different values based on the context of where it is used (i.e. the ‘sheet’ of the mapping where it is used)
  • For repeat-worksheet templates, this variable takes on the value of the appropriate worksheet, as the EDL iterates over the worksheet groups
