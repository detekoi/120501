Translations Tab

***THIS FEATURE IS BEING DEPRECATED. PLEASE MIGRATE ALL EXCEL DATA LOAD TEMPLATES TO THE NEW FORMAT***

 

The Translations tab allows the user to define Excel synonyms for esi.manage string attributes. It is generally used in more complex loads. For example, if a spreadsheet defines currency thousands of US dollars with the acronym MUSD, the EDL can translate MUSD to the esi.manage default kUSD, which describes thousands of US dollars as well. The attributes that are available in the Translations tab are the attributes that are mapped or set as tabular in the Attributes tab. Initially there is only one option per attribute, but as an attribute is filled out, an additional row will appear for that attribute.
