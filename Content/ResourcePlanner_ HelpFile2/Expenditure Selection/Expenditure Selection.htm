Expenditure Selection

The Asset Selection view also requires the user to select one or more expenditures (or activity type) to schedule against. The user can select multiple activities or expenditures. The expenditure list is populated from the list of expenditure types setup in the configuration perspective.


Once the asset and an associated activity or expense relating to the selected asset has been selected, the different combinations of projects, assets and expense lines appear in the Schedule Builder and in the ‘Unallocated’ line within the Resource Line, ready to be scheduled against a resource. Please note if the selected expenditure items do not appear in at least one asset, a yellow warning icon will appear with an associated warning message.
