Duration Modes


There are three types of Durations modes, Manual, Expenditure and Asset Attribute, these will affect the duration of the expenditures placed upon the Resource Line.

 


The first of these is Manual Duration timing, this will mean that we are able to manually choose the duration of the expenditure through the use of the “Duration (days)” text box. To manually select our duration we will first select the “Manual” node within our Resource Selection View, we must then select our “Duration”. This can be done by inserting a number into the “Duration (days)” text box. Doing this will set all of the activities durations to the selected number.

 

 


As we can see here the manual duration of the activities is set at 10 days and will be shown through the Resource Line.

 

 

 

 

 

 

 


The second duration mode is Expenditure Duration timing, this will mean when this is selected the activities durations will be based off of their own current expenditure duration/timing.



As it can be seen here the duration of the BC Dry Hole Scenario – Drilling is 13 days, this will be reflected within the Resource Line as seen below.

 


The Asset Attribute duration mode is the final of the three, when this is selected the activities durations will be based off of an Asset Attribute that will be pre-configured or will be set up by a super user. This asset attribute will apply a defaulted number to the duration of specified capital/expenditure types, but the start date of the activities will be populated by the expenditure start dates.

 

Shown above is an Asset Attribute created for the Resource Planner, this is done through the Configuration Perspective and will be done by a super user. This attribute sets the Cycle Time for Completions to be 25 days.
