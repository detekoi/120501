Perspectives

The Main areas of functionality in esi.manage are organized via Perspectives. Generally when you switch Perspectives, opened Content is hidden, and a new set of Views are opened that are appropriate to the new Perspective's function.

 

  • Projects - access to individual Projects, to create or edit.
  • Reports - create and generate Reports against Projects.
  • Ranking - apply Project Ranking calculations against Projects.
  • Integration - configure External Identifiers against Assets for data import.
  • Configuration - create and modify the Base configuration of your esi.manage instance.
