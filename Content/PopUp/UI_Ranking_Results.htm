Ranking Results

Once all of the Metrics have been defined, and Ranking Units specified, select the Rank button in the Ranking Results pane. Each of the Projects will be evaluated for each of the Metrics in the Ranking file. The Ranking Results table is a grid of Projects and Metrics:

  • Each Metric is shown as a column with the calculated value for each Project.
  • Each Project is shown as a line, with the value for each Metric, and a total Score based on all Metrics that are to be scored.

To determine the Score:

  1. Each Metric is first evaluated across all projects to determine the Range (if a Ceiling or Floor has been specified for a Metric, values outside of that Range are scored based on either that Range).
  2. The 'Best' metric value is assigned a score of 100, the poorest a score of 0. Weighting is simply a multiplier of the 0-100 score.
  3. Metric values which fall within the range are interpolated.
  4. For each Project the score for each metric is summed to provide a Total Score.

The Ranking Results table can be sorted on any column, as well as having a Colour scheme applied (colour coding on a metric by metric basis)
