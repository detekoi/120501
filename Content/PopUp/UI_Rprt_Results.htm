Report Results

The key aspects of the Results pane are:

  1. Header - the Report Type, Units, Source Budget and Reporting Dates.
  2. Definition and Type - what has been configured on the Budgets page, including Last Actuals date if appropriate.
  3. Filtering Criteria - any report specific filtering (Expenditure Type, etc.) that has been applied.
  4. Toggles that are applied - Exclude Zero, Daily Rate, etc.
  5. Sub-Total lines will be written out for each level of Grouping.
  6. Chart - if enabled (Toggles) the Grand Total line will be displayed as a Chart.
  7. Blending - if a Blended Report has been selected, the pertinent Types (Actuals, Field Estimates, Forecast) will be shaded differently within a row.
  8. Colour Shading - individual lines will be coloured differently to aid in visualization of the results.
