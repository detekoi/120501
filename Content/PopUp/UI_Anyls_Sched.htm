Analysis Schedule

  • Truncate to Economic Limit  - this will result in Cash Flow (Revenue, OPEX) being truncated once the Economic Limit has been reached for a particular Asset.
  • Analysis Start - Either Point Forward (a specific date) or Full Cycle (includes all Project Expenditures, Revenue and Costs regardless of when they occur).
  • Span - the number of years to generate the Cash Flow tables for, from the appropriate Start Date.
  • Point Forward Date - if Point Forward is selected for Analysis Start, this is the start date.
