Shift Date Function

There are many places in esi.manage that provide the ability to Move the dates of a Project or Asset. All of these tools utilize the same routine:

  1. The Earliest Date (Project or Asset, depending on what is being shifted) is first determined from all appropriate Measures (Expenditures, Production, Reserves and OPEX). This date is displayed as the Start Date within the Project Details.
  2. The Shift Dates feature allows you to shift by a Number of Days, or select a new Earliest Start Date.
  3. If Earliest State is supplied, it will be used with the existing Start Date from 1 to determine a corresponding Number of Days.
  4. This value is applied to all Forecast Data dates within the Project (or Asset).

Note that:

  1. Actuals data is not changed.
  2. Any entry whose 'Started' flag is set to True (Expenditures, Onstream) will not be shifted.
