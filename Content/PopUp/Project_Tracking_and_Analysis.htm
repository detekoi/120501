Project Tracking and Analysis

There are 3 main Analytical Tools:

  1. Variance Reports - comparing data types (Field Estimates vs Forecast, Actuals vs Field Estimates), from either the same source (Evergreen Forecast) or between sources (Evergreen Forecast vs Initial Plan, Evergreen Forecast against previous Archive).
  2. Analysis Screens - presents all data types (Forecast, Field Estimates, Actuals, Blended) for a particular Measure (CAPEX, OPEX, Production) in both a Tabular and Graphical manner.
  3. Data Matrix - presents Capital and Operating Expenses in a spreadsheet like view, with the ability to copy and paste data, or Update the Forecast from Actuals.
