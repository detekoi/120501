Report Parameters

All Reports have the same set of Parameters; the options that are available are appropriate to the Report Template.

  • Budgets - identify the Budget that is the Source for the Report, as well as Definitions and Types to be used.
  • Dates - specify the granularity (Monthly, Quarterly, Annual, etc.) and Date range for the report.
  • Grouping - specify any Grouping. The options will be a mixture of Hierarchy elements and Report Appropriate options.
  • Units of Measure - Currency Units, Production Volumes, etc.
  • Toggles - Suppress Details, Exclude Zero, Daily Rate, $ / volume.
  • Report Filters - select specific Types to be reported on.
  • Project Selection - choose Project Set(s) (or use the Project Browser for ad-hoc) to run the report against
  • Data Set - configure the Chart (for Custom Charts only).
