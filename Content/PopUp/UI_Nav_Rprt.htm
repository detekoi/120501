The Navigator in Reports

The Navigator can be used within the Reports Perspective to:

  • Open an Existing Report - expand the Private/Public folder structure under the Reports Node.
  • Create a New Report - right click within the Navigator and select New > Report.
  • Report on a Project Set - select a Project Set(s), right click on it and select Report On.

The Navigator provides access to stored instances of Reports, Project Sets, Export Jobs and Analysis cases. Each of these Nodes has a Private and Public Folder, additional folders can be created to aid in organization.
