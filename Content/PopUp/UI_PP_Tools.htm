Project Perspective Tools

The Project Perspective button ribbon provides access to a set of capabilities that can be applied to the List of Projects:

  • Tasks - Add, Delete, Clone Projects
  • Functions - Shift Project Dates, Show all Versions, Promote Virtual Projects
  • Tools - apply an Analysis Case or Export Job to a Project, or All Projects, or access the Data Matrix and Project Template Schedule editor.

in esi.manage, hovering the mouse cursor over a button will provide a Description of the button's function.
