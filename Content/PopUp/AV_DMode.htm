Data Display Mode

The Analysis View can chart data on either a Periodic or Cumulative basis.

  • Period Total - will correspond to the data in the Table (periodic)
  • Cumulative from Inception - will chart data on a cumulative basis, using the Start Date from the Project.
  • Cumulative form first visible period - will chart data on a cumulative basis, based on the Starting date specified in the Analysis view.

For the Production Analysis view, and additional option is provided to display the data on a Daily Rate basis.
