Production Type

There is the ability to associate four data 'sources' against a Production segment. By default they are Named:

  • Forecast - the Expected amount
  • Field Estimates - measured in the Field
  • Actuals - from the Accounting System
  • Financial Actuals - post reconciliation

These four can be combined to present two additional Types:

  • Blended - which combines Actuals, Field Estimates and Forecast depending on data availability
  • Financial Blended - which combines Financial Actuals, Actuals, Field Estimates and Forecast depending on data availability

If Backout calculations have been enabled: it is applied to Forecast calculations, and allows two additional Types:

  • Blend + BO - which subtracts the Backout component from the Blended value
  • Financial Blended - which subtracts the Backout component from the Financial Blended value

The Names shown above are modifiable in the Configuration; a company's particular esi.manage installation may show different names in locations where Production Type can be selected.
