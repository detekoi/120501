Table Options

These can be accessed through the button ribbon, or through the Context Menu (right click within the table).

- Add a new entry into the table, moving focus to the last empty row.

- Delete the highlighted row from the database.

- View the current row in a Form View.

- Reorder the Columns in the table.

- Export the table contents into an Excel file.

- Refresh the Table (re-query from the database)
