Asset Buttons

- Clone an Asset. Create a copy of an Asset that exists in a different Project. Only Forecast data from the Source Asset will be copied.

- Import an Asset defined with the Asset XML schema.

- Duplicate Asset - create one or more Copies of the Asset (Forecast data only), with staggered start dates.

- Shift Asset Dates - move all Dates for the Asset a specified number of days. Only entries whose 'Started' or 'Ended' flags have not been set will be shifted.

- View EIDsExternal Identifiers mapped to the Asset.
