The Project List

The Project List should be the set of Projects that you wish to Edit, or whose data you need to look at in detail. Reports, Analysis Cases and Export Jobs do not require a Project to be in the Project List to be part of the calculations.

  1. The Project List can be based on making a Project Set(s) the Active Selection, using the Project Browser Filters (if it is the Active Selection), or both.
  2. Projects can be opened from the Project List by double clicking on it. The Project View will open in the pane at the bottom of the screen.
  3. Tools and Tasks (accessed from the Button Ribbon) can be applied against the Project List, or a selected Project.
