Chart Options

You can control what data is displayed on the Chart, and on what basis it should be displayed via the Right Click menu.

  • Scale - Linear or Logarithmic
  • Series Type - Line, Bar or Area
  • Data Mode - Period Total (periodic), Daily Rate, Cumulative from Inception (from Start Date within the Project), Cumulative from first Visible Period (based on Start Date of Analysis view)
  • Data Series - select which of the Data Types (Forecast, Actuals, Blended, etc) are to be displayed.

An option is provided to save a bitmap of the Chart.
