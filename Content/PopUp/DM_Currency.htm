Currency

Individual Expenditures and OPEX Lines have a Currency Unit which relates to the supplied Gross Amount.

  • The Data Matrix can display the Source Currency unit for each entry as a Column.
  • All values displayed in the Data Matrix are converted to the selected Display Currency Unit.
  • You can select any of the available Currency Units to display the data in.
  • However, you can only update a particular line if the Display Currency is the same as the line's Source Currency.
