Analysis Settings

Analysis case Cash Flow values are always be generated with a monthly granularity from the start date:

  • Point Forward Date or earliest Project date for Full Cycle for the period represented by the Span.

Beyond this, the Display/Discounting settings serves two purposes:

  • Display  - any values in the Cash Flow results which occur prior to the specified Start Date will be lumped (in the Results) into a single line.
    • Any values which occur after the aggregate Display Period (but prior to the end date represented by the Span) will be lumped into a single line.
  • Discounting Scheme - Periods can either be Monthly or Yearly.
    • For monthly periods, Discounting can either be applied in an Annual, or Monthly manner.
    • Mid Year discounting is only applied when Discounting is Yearly.
