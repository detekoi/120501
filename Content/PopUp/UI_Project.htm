The Project

The Project encapsulates an ActivityAn Activity is represented by Capital Expenditure(s) and an OutcomeThe result of the Expenditure(s): Production, Reserve Adds, Operating Expenses.

Within a Project there must be one or more AssetsAn Asset can be a Well, Facility or Cost Center.

Expenditures, Production, Reserves, and Operating Expenses are associated against an Asset.

The 'plan' for the Project is termed the Forecast.

Progress can be tracked by entering (importing) Actuals against the Project data.
