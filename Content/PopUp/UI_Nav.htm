The Navigator

Each of the following items represent Features in esi.manage that can be Configured and Saved.

  • Project Set - a collection of Projects, either selected individually, or based on filtering.
  • Report - a pre-defined Report Type (CAPEX, OPEX) which can be configured using a variety of options.
  • Analysis Case - Discounted Cash Flow calculations, both Before and After Tax.
  • Export Job - Project Data (user Supplied and Calculated) for a variable time period, using an XML schema.

The Folder Structure under each Navigator node can be configured to suit your company's needs. Each node has two default folders, Private (user specific) and Public (available to everyone with sufficient Permissions).

The Project Set is a fundamental building block. Reports, Analysis Cases and Export Jobs have two aspects:

  1. The Configuration of their calculations, and
  2. the Project Set(s) they are to apply the calculations against.

Project Sets can be used to Populate the Project List, and you can also Apply the features listed above against a Project Set.
