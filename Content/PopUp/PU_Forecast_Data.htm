Copying Forecast Data

The various Copy functions (Clone or Duplicate Asset, Clone Project, etc.) will only create copies of Forecast Data. The following will not be part of the Copy process:

  • Spudded or Rig Released Flags
  • Started or Ended Flags on Expenditures
  • Onstream or Offstream Flags for Production
  • AFEs
  • Any Field Estimates or Actuals
