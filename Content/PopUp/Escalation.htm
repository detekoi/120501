Currency Escalation

A Currency Escalation file defines Annualized Cost Escalation Factors for a Time Series (vector of dates).

  • Individual Expenditure Types can have Escalation Factors defined by Jurisdiction.
  • OPEX has a blanket Escalation Factor, by Jurisdiction.
  • User supplied Escalation Percentages will be used for all periods into the future, until another user supplied value is encountered.

When an Escalation File is selected for calculations, the Date of the particular expense (Capital or OPEX) is adjusted based on the corresponding entry in the Escalation File.
