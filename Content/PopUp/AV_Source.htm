Source Data

For each of the Analysis Views, there are several selections that will filter the Source Data. There is no interaction between the 'Source Lists', i.e.; the selections in one list do not re-filter the other lists.

  • Expenditure - for the Expenditure Analysis View, this is the aggregate list of Expenditures from the Project scope.
  • Asset - for the OPEX and Production Analysis view, this is the aggregate list of Assets from the Project scope.
  • Type - the entire list of Expenditure or OPEX Types, from the esi.manage Configuration.
  • AFEs, Products - for Expenditure and Production analysis views.

If no filtering is selected, the display will be the aggregate of the Projects in the Projects List.
