Analysis Results

Once the Analysis case has been run, the results are displayed in three manners:

  • Cashflow Analysis - the resulting Discounted Cash Flow calculations.
  • Results will be aggregated at an Asset, Project, Project Set and Jurisdiction level.
  • Metric Analysis - the Economic Metrics that are defined for the Analysis Case.
  • Analysis Chart - a collection of Analytical tools (NPV sensitivity, Pareto Chart, etc.) applied against the results.
