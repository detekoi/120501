Analysis Case Pages

There are 3 parts to defining an Analysis Case:

  • Properties - for a new Analysis, these will be defaulted to whatever has been set in the Configuration.
    • Depending on the Lock status on these properties, some or all can be modified for a new Analysis Case.
  • Metrics - for a new Analysis, these will be defaulted to whatever has been set in the Configuration.
    • Global metrics cannot be modified.
    • New Metrics (different Discount Rate, etc.) can be added to an Analysis Case beyond what has been Configured.
  • Project Sets - Project Sets can be associated against, and saved with, an Analysis.
    • Once an Analysis Case has been defined, it can be used with other Project Sets, or Projects as needed - applying the Analysis case to Project Sets or individual Projects.
