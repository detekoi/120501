Ranking Units

Ranking Units are defined on a user by user basis, independently of the Ranking Files.

  • Product Type - all Production and Reserves in all Projects will be converted to this basis.
  • Production, Reserve, F&D and Capital Efficiency Units - on what basis to report these metrics.
  • Currency Unit
  • Economic Discounting As Of - used for Economic Discounting calculations.
  • Blending Mode - Capital Metrics are always Blended (based on the selected Milestone). This choice allows the user to determine which Blending mode is to be used for Capital metrics.
