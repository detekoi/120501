Working with the Cash Flow Results

An Analysis case can contain multiple Project Sets, Jurisdictions, Projects and Assets.

  • Calculations are performed at the Asset Level, and rolled up accordingly.
  • The Cash Flow table indicates this using a Tree Structure.

Since there are many potential levels of nesting, there are options in the Cash Flow table to simplify this, which are accessed by Right Clicking on a node.

  • Edit will open the 'Source' for the particular node (Project Set or Project)
  • Add, Remove and Clear Focus apply to which Assets will be used as the Data Source for the Analysis Charts.
  • Go Into moves one level in within the Tree Structure (and hides the outer nodes)
  • Go Back move one level out
  • Go Home returns to the outermost level.
