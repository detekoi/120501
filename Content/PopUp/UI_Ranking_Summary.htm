Ranking Summary

The Ranking summary displays the Results on a Metric by Metric basis. Only Projects whose 'Include in Summary' check box are included in the Results.

  • Low - the smallest calculated value across all Projects.
  • High - the largest calculated value across all Projects.
  • Total - the sum of the metric across all Projects .
  • Target and Gap - the user supplied Target value for the Metric, and the delta between the Total and that Target Value.

The Update Summary button is a toggle - if you wish to toggle a number of Projects into/out of the Summary, turn off the Update summary button and use the Refresh option once you have finished making your changes.
